    Funktioner¶

    Fuld integration med Gramps Desktop¶

    Gramps Web bruger den samme Model / Database struktur, som Gramps Desktop bruger til at gemme genealogiske data. Du kan gennemse alle de samme Record Types, som du gør i Gramps Desktop: personer, familier, begivenheder, steder, arkiver, kilder, citater, medieobjekter og noter.

    Ved at bruge Gramps Web Sync Add-on til Gramps Desktop kan data synkroniseres bi-direktionelt mellem Gramps Web og Gramps Desktop! Gå videre og rediger dine data med Gramps Web eller Gramps Desktop App, som du kender og elsker - de arbejder sammen uden problemer!

    Privatlivsniveauer / Brugeradgang¶

    Mange ønsker at holde nogle detaljer private, og det respekterer vi! Du kan markere poster som private og kontrollere, hvilke brugere der har lov til at se private poster. Private poster filtreres ud på databaselaget for maksimal sikkerhed. Derudover kan du kontrollere, hvad brugere kan tilføje og redigere.

    Webbrowser Bekvemmelighed¶

    Adgang til Gramps Web fra enhver web-aktiveret enhed. Du kan uploade fotos, oprette eller redigere poster, vise dit slægtstræ for andre, eller finde de familiemedlemmer, hvis navne du ikke kan huske til din næste familiegensyn!

    Interaktive slægtstræ grafer¶

    Naviger dit slægtstræ som et forfædre-diagram, efterkommer-diagram, timeglas-diagram eller som fan-diagram, med høj-kvalitets interaktive grafik og konfigurerbart antal generationer.

    Kraftfuldt kort¶

    Vis alle steder på et interaktivt, søgbart kort.

    Forvandl et historisk kort gemt som medieobjekt i Gramps til et tilpasset kortoverlæg.

    Genealogiblog inkluderet¶

    Sammenfat din forskning i form af bloghistorier med billeder. Alle data gemmes i Gramps-databasen.

    Fuldstændig internationaliseret¶

    Skift sproget i grænsefladen mellem nogen af de 40 sprog, der er oversat af Gramps-fællesskabet.

    Ingen låsning – dataimport og eksport¶

    Udover at kunne importere data i forskellige formater, herunder Gramps XML og GEDCOM, gør Gramps Web det nemt for brugerne at downloade alle deres data (slægtstrædata, mediefiler, brugerkonti) når som helst, til backupformål eller for at flytte til en anden server. Dine data tilhører kun dig!

    Generer udskrivbare rapporter¶

    Da det er bygget direkte på kernen, der driver Gramps Desktop, kan du generere næsten alle de rapporter, som desktopappen understøtter direkte fra browseren, herunder slægtsdiagrammer eller bograpporter som PDF.

    Tag personer i fotos med automatiseret ansigtsgenkendelse¶

    Samarbejd med dine slægtninge for at identificere forfædre i gamle familiefotos. Takket være automatiseret ansigtsgenkendelse er det kun to klik væk at tagge personer.

    Kraftfuld fuldtekstsøgning¶

    Søgemaskinen understøtter søgning på alle Gramps-objekttyper, herunder indholdet af tekstnoter, og understøtter wildcard-syntaks og logiske operatorer.

    DNA matches & kromosombrowser¶

    Hvis du har DNA-matchdata fra en af DNA-genealogileverandørerne, kan du uploade det og gemme det på en fremtidssikret måde og se dine matches i en interaktiv kromosombrowser.

    Integreret opgavestyringsapp¶

    Gramps Web leveres med en integreret opgavestyringsapp til at organisere og planlægge din genealogiske forskning. Opgaverne gemmes som kilder i Gramps-databasen, så de udgør en del af dine genealogiske data og kan tilgås og redigeres i Gramps Desktop også.

    Integration af OpenHistoricalMap¶

    De historiske vektorkort, der er oprettet af OpenHistoricalMap projektet, er den perfekte komplement til genealogisk kortlægning. Brug tidsglideren til at rulle gennem udviklingen af de steder i din families historie og vis de steder, hvor forfædre boede eller begivenheder fandt sted.

    Integreret AI chat-assistent¶

    Drevet af AI, giver Gramps Web dig mulighed for at chatte med dit slægtstræ – på dit modersmål!

     

    Demo¶

    For at logge ind på demoen, brug en af de følgende BRUGER / KODE loginoplysninger. Hver repræsenterer en brugertype, som en Gramps Web-bruger kan tildeles.

    owner / owner
    editor / editor
    contributor / contributor
    member / member
