    Как внести вклад в Gramps Web¶

    Спасибо, что рассматриваете возможность внести вклад в Gramps Web! В зависимости от ваших навыков и интересов, есть множество способов, как вы можете помочь.

    Помощь в переводе Gramps Web на ваш язык¶

    Gramps и Gramps Web поддерживают несколько десятков различных языков. Помощь в переводе приложения не требует никаких навыков программирования. Мы используем мощный онлайн-инструмент Weblate для переводов. Пожалуйста, смотрите здесь инструкции о том, как начать.

    Помощь в ответах на вопросы других пользователей¶

    Если вы уже используете Gramps Web, помогите новым пользователям найти ответы на их вопросы, участвуя в обсуждениях в категории Gramps Web официального форума Gramps Discourse.

    Помощь в улучшении документации¶

    Сайт документации, который вы просматриваете, размещен на Github. Вклады для его улучшения очень приветствуются. Просто отправьте запрос на извлечение в этот репозиторий.

    Вклад в разработку бэкенда (Python)¶

    Если вы программист на Python, внесите вклад в разработку Gramps Web API, который является бэкендом для Gramps Web. Смотрите Разработка бэкенда для получения инструкций о том, как начать.

    Вклад в разработку фронтенда (Javascript)¶

    Если вы веб-программист, внесите вклад в разработку отзывчивого фронтенда Gramps Web. Смотрите Разработка фронтенда для получения инструкций о том, как начать.

    Вклад в дополнение Sync (Python)¶

    Если у вас есть опыт разработки на Python и GTK, внесите вклад в улучшение дополнения Gramps Web Sync. Вы можете использовать репозиторий для разработки для предложений и отправки улучшений.
