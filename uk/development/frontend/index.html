    Розробка веб-інтерфейсу Gramps¶

    Веб-інтерфейс Gramps Web написаний на Javascript і створений з використанням веб-компонентів з Lit. Це дозволяє інкапсулювати функціональність у повторно використовувані компоненти.

    Веб-інтерфейс спроектований як прогресивний односторінковий веб-додаток з орієнтацією на мобільні пристрої. Він активно використовує Material Web Components для вигляду та відчуття Material Design.
