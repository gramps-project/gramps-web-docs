# Як внести свій внесок у Gramps Web

Дякуємо, що розглядаєте можливість внести свій внесок у Gramps Web! Залежно від ваших навичок та інтересів, є багато способів, як ви можете допомогти.

## Допоможіть перекласти Gramps Web на вашу мову

Gramps і Gramps Web підтримують кілька десятків різних мов. Допомога в перекладі додатку не вимагає жодних програмістських навичок. Ми використовуємо потужний онлайн-інструмент Weblate для перекладів. Будь ласка, дивіться [тут](../development/frontend/translation.md) для інструкцій про те, як почати.

## Допоможіть відповідати на запитання інших користувачів

Якщо ви вже використовуєте Gramps Web, допоможіть новим користувачам знайти відповіді на їхні запитання, беручи участь в обговореннях у [категорії Gramps Web](https://gramps.discourse.group/c/gramps-web/) на офіційному форумі Gramps Discourse.

## Допоможіть покращити документацію

Сайт документації, який ви переглядаєте, розміщено [на Github](https://github.com/gramps-project/gramps-web-docs). Внески для його покращення дуже вітаються. Просто надішліть запит на злиття до цього репозиторію.

## Внесіть свій внесок у розробку бекенду (Python)

Якщо ви програміст на Python, внесіть свій внесок у розробку [Gramps Web API](https://github.com/gramps-project/gramps-web-api/), який є бекендом для Gramps Web. Дивіться [Розробка бекенду](../development/backend/index.md) для інструкцій про те, як почати.

## Внесіть свій внесок у розробку фронтенду (Javascript)

Якщо ви веб-програміст, внесіть свій внесок у розробку [відповідного фронтенду Gramps Web](https://github.com/gramps-project/gramps-web/). Дивіться [Розробка фронтенду](../development/frontend/index.md) для інструкцій про те, як почати.

## Внесіть свій внесок у додаток Sync (Python)

Якщо у вас є досвід розробки на Python і GTK, внесіть свій внесок у покращення додатку Gramps Web Sync. Ви можете використовувати [репозиторій для розробки](https://github.com/DavidMStraub/gramps-web-sync/) для пропозицій та подання покращень.
