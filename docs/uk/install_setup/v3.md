# Посібник з оновлення Gramps 6.0

Gramps 6.0 був випущений 19 березня 2025 року. Оскільки він використовує нову схему бази даних родинного дерева в порівнянні з Gramps 5.2, при оновленні Gramps Web для використання з Gramps 6.0 необхідні додаткові кроки. Ця сторінка описує необхідні кроки.

!!! info
    Останнє оновлення: 4 травня 2025 року

## Детальні кроки

Цей розділ надає більше деталей про деякі з окремих кроків вище.

### Резервне копіювання вашого родинного дерева

У Gramps Web перейдіть на сторінку "Експорт" і виберіть "Gramps XML (родинне дерево)" (за замовчуванням). Натисніть "Генерувати" і чекайте, поки не почнеться завантаження файлу .gramps.

Ви також можете створити резервну копію в Gramps Desktop [звичним способом](https://www.gramps-project.org/wiki/index.php/How_to_make_a_backup).

### Оновити схему бази даних дерева

Коли ви вперше отримуєте доступ до Gramps Web після його оновлення, вам буде представлено екран з повідомленням: 

> Родинне дерево, яке ви намагаєтеся завантажити, має версію схеми, яка не підтримується цією версією Gramps Web. Тому ви не можете завантажити це родинне дерево без оновлення його схеми. Цю дію не можна скасувати.

Тепер ви можете натиснути кнопку "Оновити родинне дерево". Вона покаже індикатор прогресу і не повинна зайняти більше хвилини для завершення. Не хвилюйтеся, якщо ви випадково закриєте вікно браузера, конверсія продовжиться у фоновому режимі.

!!! warning
    Якщо ви використовуєте нестандартну конфігурацію *без* черги фонових завдань, оновлення бази даних може завершитися невдало, і ваше родинне дерево може бути пошкоджене. Ви можете перевірити (перед оновленням Gramps Web), чи використовуєте ви чергу фонових завдань, перейшовши в розділ "Системна інформація" в налаштуваннях і переконавшись, що там є "черга завдань: true".


### Оновити Gramps Desktop

Слідуйте інструкціям з установки Gramps Desktop для вашої операційної системи.

### Встановити оновлений додаток Sync

Ви можете використовувати Менеджер плагінів у Gramps 6.0 для установки додатку синхронізації Gramps Web. Знайдіть Gramps Web Sync, натисніть на встановлення і перезапустіть Gramps.
