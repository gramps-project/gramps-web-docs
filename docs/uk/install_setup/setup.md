# Gramps Web Встановлення / Налаштування

Цей розділ стосується встановлення та налаштування Gramps Web, а також інших варіантів для початку роботи.

## Початок роботи з Gramps Web

Gramps Web — це веб-додаток, який працює на сервері і доступний через веб-браузер. Він призначений для надання доступу автентифікованим користувачам через інтернет.

Якщо ви хочете використовувати Gramps Web для ваших генеалогічних досліджень, вам потрібно вибрати один з наступних варіантів:

1. Самостійне хостинг на власному обладнанні
2. Самостійне хостинг у хмарі
3. Зареєструватися на хостингову інстанцію

Хоча перший варіант надає максимальну гнучкість і контроль, він також може бути технічно складним.

!!! tip
    Один з основних принципів Gramps Web — дати користувачам контроль над своїми даними в будь-який час, тому міграція даних з однієї інстанції на іншу є простою. Не турбуйтеся про те, що ви будете заблоковані після вибору одного з варіантів!

## Самостійне хостинг на власному обладнанні

Найзручніший спосіб самостійного хостингу Gramps Web — це використання Docker Compose. Ми також надаємо Docker-образи для архітектури ARM, тому ви можете запустити Gramps Web на Raspberry Pi у вашому підвалі.

Дивіться [Розгортання з Docker](deployment.md) для інструкцій з налаштування.

## Самостійне хостинг у хмарі

Встановлення Gramps Web може бути складнішим, ніж для інших простих веб-додатків, і не сумісне з звичайними провайдерами "спільного хостингу". Ви можете зареєструватися на віртуальному сервері та встановити Gramps Web [вручну](deployment.md).

Простішим варіантом є використання попередньо встановленого образу в хмарі. Один з прикладів — наш [DigitalOcean 1-клік додаток](digital_ocean.md).

## Зареєструватися на хостингову інстанцію

Хостинг Gramps Web — це найпростіший спосіб почати працювати з Gramps Web, оскільки не потрібно встановлення або налаштування.

Ось список спеціалізованих провайдерів хостингу для Gramps Web (громадськість Gramps з відкритим кодом не несе відповідальності за їхні послуги):

- Grampshub ([www.grampshub.com](https://www.grampshub.com)), запропонований одним з основних contributors Gramps Web

Якщо ви використовуєте хостинговий варіант для Gramps Web, ви можете пропустити решту цього розділу і перейти безпосередньо до розділу [Адміністрування](../administration/admin.md).
