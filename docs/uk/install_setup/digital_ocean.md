# Gramps Web DigitalOcean 1-Click App

Замість того, щоб [налаштовувати Gramps Web самостійно](deployment.md), ви також можете скористатися [Gramps Web DigitalOcean 1-Click App](https://marketplace.digitalocean.com/apps/gramps-web?refcode=b1d13ebe86ac&action=deploy). Digital Ocean хостить демонстраційну версію Gramps Web.

У рамках процедури налаштування вам потрібно буде зареєструвати обліковий запис у DigitalOcean та вибрати платний план для "дроплета" (віртуальної машини), яку ви будете використовувати.

Можна стверджувати, що це наразі найпростіший спосіб розгорнути власну, самостійно хостовану інстанцію Gramps Web, захищену SSL, без використання власного обладнання.

!!! info
    Зверніть увагу, що ви будете платити DigitalOcean за хостингові послуги. Проект Gramps з відкритим кодом не надає платну підтримку.

## Крок 1: Створіть обліковий запис DigitalOcean

Створіть обліковий запис на [DigitalOcean](https://www.digitalocean.com/), якщо у вас його ще немає.

## Крок 2: Створіть дроплет

- Перейдіть до [Gramps Web 1-Click App](https://marketplace.digitalocean.com/apps/gramps-web?refcode=b1d13ebe86ac&action=deploy) і натисніть "Створити дроплет Gramps Web".
- Виберіть план з принаймні 2 ГБ оперативної пам'яті.
- Налаштуйте автентифікацію для вашого дроплета.
- Натисніть "Створити дроплет".

!!! info
    Вам може знадобитися почекати до десяти хвилин, поки 1-Click App встановить останню версію `docker-compose`.
    Використання останньої версії `docker-compose` може зменшити кількість помилок, пов'язаних з `firstlogin.sh`.

## Крок 3: Налаштуйте доменне ім'я

Вам потрібно доменне ім'я (або піддомен). Якщо ви володієте доменом, вкажіть його на IP-адресу вашого дроплета. В іншому випадку ви можете скористатися безкоштовною послугою, такою як [DuckDNS](https://www.duckdns.org/).

## Крок 4: Увійдіть до вашого дроплета

SSH до вашого дроплета. Ви повинні побачити повідомлення "Ласкаво просимо до налаштування Gramps Web DigitalOcean 1-click app!". Якщо це не так, почекайте кілька хвилин і спробуйте ще раз (встановлення ще не завершено).

Скрипт налаштування запитає у вас доменне ім'я (наприклад, `mygrampswebinstance.duckdns.org`) та електронну адресу (необхідну для сертифіката Let's Encrypt).

Коли це буде зроблено, чекайте, поки налаштування завершиться у фоновому режимі.

## Крок 5: Запустіть Gramps Web

Ваша інстанція Gramps Web тепер повинна бути доступною за коренем вашого домену, з дійсним SSL сертифікатом, і вона повинна показувати помічника першого запуску.
