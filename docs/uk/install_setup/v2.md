## Детальні кроки

Цей розділ надає більше деталей про деякі з окремих кроків, наведених вище.

### Резервне копіювання вашого родинного дерева

У Gramps Web перейдіть на сторінку "Експорт" і виберіть "Gramps XML (родинне дерево)" (за замовчуванням). Натисніть "Генерувати" і чекайте, поки не почнеться завантаження файлу .gramps.

Ви також можете створити резервну копію в Gramps Desktop [звичним способом](https://www.gramps-project.org/wiki/index.php/How_to_make_a_backup).

### Оновлення схеми бази даних дерева

Коли ви вперше отримуєте доступ до Gramps Web після його оновлення, вам буде представлено екран з повідомленням:

> Родинне дерево, яке ви намагаєтеся завантажити, знаходиться в схемі версії, яка не підтримується цією версією Gramps Web. Тому ви не можете завантажити це родинне дерево без оновлення його схеми. Цю дію не можна скасувати.

Тепер ви можете натиснути кнопку "Оновити родинне дерево". Вона відобразить індикатор прогресу і не повинна зайняти більше хвилини для завершення. Не хвилюйтеся, якщо ви випадково закриєте вікно браузера, конвертація продовжиться у фоновому режимі.

!!! warning
    Якщо ви використовуєте нестандартну конфігурацію *без* черги фонових завдань, оновлення бази даних може не вдатися, і ваше родинне дерево може бути пошкоджено. Ви можете перевірити (перед оновленням Gramps Web), чи використовуєте ви чергу фонових завдань, перейшовши в розділ "Системна інформація" в налаштуваннях і переконавшись, що там є "task queue: true".

### Оновлення Gramps Desktop

Слідуйте інструкціям з установки Gramps Desktop для вашої операційної системи.

### Встановлення оновленого додатку Sync

Ви можете використовувати [новий Менеджер плагінів](https://gramps-project.org/wiki/index.php/Addon:Plugin_Manager), включений у Gramps 5.2 - просто знайдіть Gramps Web Sync, натисніть на встановлення і перезапустіть Gramps.
