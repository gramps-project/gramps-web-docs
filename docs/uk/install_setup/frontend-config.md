# Налаштування фронтенду

Фронтенд Gramps Web є JavaScript-додатком, який розгортається як набір статичних HTML, CSS та JavaScript файлів. Зазвичай, спеціальна конфігурація для фронтенду не потрібна. Однак деяку поведінку можна змінити, встановивши відповідні параметри у файлі `config.js` в корені дистрибутиву.

Файл повинен мати таку структуру:

```javascript
window.grampsjsConfig = {
    option: value
}
```

Існують такі ключі параметрів.

Ключ | Тип | Опис 
----|-----|-----------
`hideDNALink` | boolean | Якщо true, приховати посилання на ДНК на панелі навігації.
`hideRegisterLink` | boolean | Якщо true, приховати посилання на реєстрацію на сторінці входу. Це слід використовувати для розгортань з кількома деревами.
`loginRedirect` | string | URL для перенаправлення, коли не увійшли в систему та переходять на будь-яку сторінку, окрім "вхід" або "реєстрація"
`leafletTileUrl` | string | Користувацький URL тайлів для карт Leaflet
`leafletTileSize` | number | Користувацький розмір тайлів для карт Leaflet
`leafletZoomOffset` | number | Користувацький зсув масштабу для карт Leaflet
`leafletTileAttribution` | string | Користувацька атрибуція для карт Leaflet
