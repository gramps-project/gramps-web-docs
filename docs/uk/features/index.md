# Особливості


![Image title](screenshots/sync.png){ align=left width="300"}

## Повна інтеграція з Gramps Desktop

Gramps Web використовує ту ж **Модель / Базу даних**, що й [Gramps Desktop](https://gramps-project.org/) для зберігання генеалогічних даних. Ви можете переглядати всі ті ж [Типи записів](https://gramps-project.org/wiki/index.php/Gramps_Data_Model), що й у Gramps Desktop: ***люди, родини, події, місця, архіви, джерела, цитати, медіа-об'єкти та нотатки.***

Використовуючи [Gramps Web Sync Add-on](../administration/sync.md) для Gramps Desktop, дані можуть бути синхронізовані в обох напрямках між Gramps Web та Gramps Desktop! Продовжуйте редагувати свої дані за допомогою Gramps Web або програми Gramps Desktop, яку ви знаєте та любите - вони працюють разом безшовно!

## Рівні конфіденційності / Доступ користувачів

Багато людей хочуть зберегти деякі деталі приватними, і ми це поважаємо! Ви можете позначити записи як Приватні та контролювати, які користувачі можуть переглядати приватні записи. Приватні записи фільтруються на рівні бази даних для максимальної безпеки. Крім того, ви можете контролювати, що користувачі можуть додавати та редагувати.

## Немає прив'язки – імпорт та експорт даних

Окрім можливості імпортувати дані в різних форматах, включаючи Gramps XML та GEDCOM, Gramps Web спрощує користувачам завантаження всіх своїх даних (дані родинного дерева, медіа-файли, облікові записи користувачів) у будь-який час, для резервного копіювання або для переходу на інший сервер. Ваші дані – це тільки ваші!

## Інтегрований додаток для управління завданнями

Gramps Web постачається з інтегрованим додатком для управління завданнями, щоб організувати та спланувати ваше генеалогічне дослідження. Завдання зберігаються як джерела в базі даних Gramps, тому вони є частиною ваших генеалогічних даних і можуть бути доступні та редаговані в Gramps Desktop також.

## Інтеграція OpenHistoricalMap

Історичні векторні карти, створені проектом [OpenHistoricalMap](https://www.openhistoricalmap.org/), є ідеальним доповненням до генеалогічного картографування. Використовуйте повзунок часу, щоб переглядати еволюцію місць у вашій родинній історії та відображати місця, де жили предки або відбувалися події.

`editor / editor` <br>
`contributor / contributor` <br>
`member / member`
