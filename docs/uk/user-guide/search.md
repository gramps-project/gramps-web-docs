# Пошук

Сторінка пошуку доступна, якщо натиснути на іконку лупи в верхній панелі додатку або натиснути `s` [гарячу клавішу](shortcuts.md).

## Повнотекстовий пошук

Введіть будь-який запит у поле пошуку та натисніть Enter (або натисніть кнопку пошуку). Gramps Web виконує пошук серед усіх типів об'єктів – людей, сімей, подій, місць, джерел, цитат, репозиторіїв, нотаток та медіа – і повертає результати, відсортовані за релевантністю.

Результати показують тип об'єкта, ім'я та короткий опис. Натисніть на будь-який результат, щоб відкрити відповідну сторінку деталей.

Знак `*` може бути використаний як підстановочний символ, наприклад, `Mey*` відповідає "Meyer", "Meyers", "Meyerhofer" тощо.

## Фільтрація за типом об'єкта

Під полем пошуку кнопки-перемикачі для кожного типу об'єкта (Люди, Сім'ї, Події, Місця, …) дозволяють звузити результати до одного або кількох конкретних типів. За замовчуванням шукаються всі типи. Активуючи один або кілька перемикачів, ви обмежуєте результати лише цими типами.

## Семантичний пошук

Якщо адміністратор сервера увімкнув [семантичний (на базі ШІ) пошук](../install_setup/configuration.md), у верхньому правому куті сторінки пошуку з'являється перемикач режиму з двома варіантами:

- **Пошук** – стандартний повнотекстовий пошук (за замовчуванням)
- **Семантичний** – пошук на базі ШІ, який розуміє значення вашого запиту, а не просто збігається з точними словами

Семантичний пошук корисний для запитів природною мовою, таких як "фермер у Баварії в 19 столітті". Він вимагає, щоб семантичний індекс пошуку був заповнений; див. [Налаштування адміністрації](../administration/settings.md) для того, як створити або оновити індекс.
