У встановленні з одною деревом будь-хто може зареєструвати новий обліковий запис на екземплярі Gramps Web, але його потрібно буде активувати вручну власником сайту, перш ніж ним можна буде скористатися для входу.

У встановленні з кількома деревами користувачам додатково потрібен ідентифікатор дерева для реєстрації; на практиці це означає, що вони можуть зареєструватися, використовуючи посилання на реєстрацію, яке поділився власник дерева.

## 1. Знайти форму реєстрації

На формі входу на домашній сторінці Gramps Web зазвичай є посилання "Зареєструвати новий обліковий запис".

Якщо це посилання відсутнє, ймовірно, це через те, що це встановлення Gramps Web з кількома деревами. У такому випадку зв'яжіться з власником дерева для отримання посилання на реєстрацію.

Власник дерева може знайти та скопіювати посилання на реєстрацію в розділі Налаштування/Адміністрування.

## 2. Надіслати форму реєстрації

Форма реєстрації має чотири обов'язкові поля: ім'я користувача, пароль, електронну адресу та повне ім'я. Коли всі чотири поля будуть заповнені, натисніть "надіслати", щоб розпочати процес реєстрації.

## 3. Підтвердити електронну адресу

Після того, як форма буде надіслана, на вказану електронну адресу буде надіслано автоматизований електронний лист, що містить посилання для підтвердження електронної адреси. Натискання на нього відкриє сторінку підтвердження.

Якщо підтвердження електронного листа не надійшло (і ви перевірили папку Спам), можливо, електронна пошта не була налаштована правильно адміністратором сервера Gramps Web.

## 4. Активувати обліковий запис

Після підтвердження електронної адреси власник дерева отримає сповіщення електронною поштою і зможе активувати новий обліковий запис користувача, перейшовши до розділу "Адміністрування користувачів" на сторінці налаштувань (доступно через іконку користувача на верхній панелі додатку) і змінивши роль користувача з "вимкнено" на будь-яку з [інших ролей](../install_setup/users.md).

Зверніть увагу, що новий користувач *не* отримає автоматизоване сповіщення про те, що його обліковий запис було активовано, тому рекомендується адміністратору повідомити їх особисто.
