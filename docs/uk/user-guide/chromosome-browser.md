# Хромосомний браузер

Хромосомний браузер — це інструмент для візуалізації ДНК-матчів особи. Він показує сегменти ДНК, які спільні з іншими людьми в дереві, як кольорові смуги на хромосомах.

Ви можете знайти його на вкладці &ldquo;Хромосомний браузер&rdquo; у перегляді ДНК.

Для кожної хромосоми верхня (світло-блакитна) смуга представляє копію, успадковану від батька, нижня (світло-червона) смуга — ту, що успадкована від матері. Спільні сегменти показані як кольорові смуги поверх цих. Якщо сторона (батьківська або материнська) не відома, сегмент охоплює обидві смуги і показується з рисками.

Клацнувши на ім'я матчу в легенді, показуються лише спільні сегменти цієї особи.
