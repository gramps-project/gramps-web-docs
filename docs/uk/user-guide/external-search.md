# Зовнішній пошук

Gramps Web дозволяє шукати людей на зовнішніх генеалогічних веб-сайтах безпосередньо з детальної сторінки особи.


## Виконання зовнішнього пошуку

Щоб знайти людину на зовнішніх веб-сайтах:

1. Відкрийте детальний перегляд особи.
2. Натисніть кнопку зовнішнього пошуку.
3. З'явиться спливаюче вікно з посиланнями на різні генеалогічні веб-сайти, такі як CompGen, FamilySearch та Ancestry.
4. Натискаючи на посилання, ви відкриєте сторінку результатів пошуку на цьому веб-сайті, попередньо заповнену ім'ям особи, місцями народження/смерті та іншою відповідною інформацією.

## Налаштування параметрів зовнішнього пошуку

Ви можете налаштувати, які зовнішні веб-сайти для пошуку відображаються у спливаючому вікні.

1. У спливаючому вікні зовнішнього пошуку натисніть кнопку шестерні вгорі, щоб увійти в режим редагування.
2. Виберіть або зніміть вибір з веб-сайтів, які ви хочете показати або приховати.
3. Збережіть свої зміни.

## Додавання власного пошукового двигуна

Ви можете додати нові зовнішні веб-сайти для пошуку з власними параметрами.

1. У спливаючому вікні зовнішнього пошуку натисніть кнопку шестерні вгорі, щоб увійти в режим редагування.
2. Натисніть кнопку "+ Додати власний пошуковий двигун".
3. З'явиться секція з полями для "Назва" та "URL пошуку". Введіть назву веб-сайту та URL з параметрами для пошуку, такими як `https://example.com/search?name={{name_given}}_{{name_surname}}&place={{birth_place}}`.
4. Натисніть кнопку "Додати", щоб включити новий пошуковий двигун до списку.
5. Новий пошуковий двигун буде збережено і може бути налаштовано пізніше.
