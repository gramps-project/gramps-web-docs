# Використання комбінацій клавіш

Комбінації клавіш дозволяють вам швидко переміщатися по Gramps Web без використання миші, економлячи ваші кліки та час.

З будь-якого виду ви можете безпосередньо перейти до загальних дій та сторінок всього за кілька натискань клавіш. Наприклад, натисніть **`s`** на клавіатурі, щоб відкрити пошук, **`g`** + **`c`** для переходу до Дерева родини, або **`n`** + **`p`** для створення нової особи.

Щоб побачити повний список усіх доступних комбінацій клавіш, просто натисніть **`?`** у будь-який час.
