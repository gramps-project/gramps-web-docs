# Налаштування користувача

Налаштування користувача доступні через іконку користувача у верхній панелі додатку, потім **Налаштування користувача**. Сторінка організована в згортані секції. Зміни набирають чинності негайно, якщо не зазначено інше.

!!! note
    Зміни в Налаштуваннях користувача впливають лише на ваш обліковий запис. Налаштування, які впливають на всіх користувачів дерева, керуються в [Налаштуваннях адміністрації](../administration/settings.md).

## Обліковий запис

Охоплює інформацію про ваш профіль, облікові дані та безпеку облікового запису.

### Інформація про користувача

Показує ваше **ім'я користувача** та поточну **роль користувача** (наприклад, Гість, Член, Редактор). Ці дані доступні лише для читання.

### Змінити електронну пошту

Введіть нову електронну адресу та натисніть **Надіслати**, щоб оновити адресу, пов'язану з вашим обліковим записом. Електронна адреса використовується для скидання пароля та (якщо налаштовано) сповіщень.

### Змінити пароль

Введіть ваш поточний пароль та новий пароль, потім натисніть **Надіслати**. Якщо ви забули свій поточний пароль, скористайтеся посиланням **Забули пароль** на сторінці входу.

## Зовнішній вигляд

Керує налаштуваннями відображення, збереженими на вашому пристрої.

### Мова

Виберіть мову для веб-інтерфейсу Gramps. Налаштування мови зберігається в локальному сховищі браузера і застосовується лише до поточного пристрою.

### Тема

Виберіть між:

- **Системна** – слідує перевагам світлого/темного режиму операційної системи (за замовчуванням)
- **Світла** – завжди використовувати світлу тему
- **Темна** – завжди використовувати темну тему

Налаштування теми зберігається в локальному сховищі браузера.

### Налаштування родинного дерева

#### Вигляд родинного дерева за замовчуванням

Встановлює, який тип діаграми відкривається за замовчуванням, коли ви переходите на сторінку [Родинне дерево](tree.md). Варіанти: Дерево предків, Дерево нащадків, Граф піскового годинника, Граф відносин та Фан-діаграма.

Це налаштування зберігається в локальному сховищі браузера.

## Інструменти розробника

### API токен

Копіює ваш поточний токен сесії в буфер обміну. Токен можна використовувати для прямої аутентифікації через REST API, наприклад, в інтерактивному Swagger UI, який надається вашим екземпляром Gramps Web за адресою `/api/swagger-ui`.

Натисніть **Запустити Swagger**, щоб відкрити Swagger UI в новій вкладці з уже доступною вашою сесією.

!!! note
    Токен сесії має короткий термін дії. Скопіюйте його відразу перед використанням у Swagger, оскільки він може закінчитися.
