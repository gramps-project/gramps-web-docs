# Налаштування користувача

Налаштування користувача доступні через іконку користувача у верхній панелі програми, потім **Налаштування користувача**. Зміни набирають чинності негайно, якщо не зазначено інше.

## Інформація про користувача

Показує ваше **ім'я користувача** та поточну **роль користувача** (наприклад, Гість, Член, Редактор). Ці дані доступні лише для читання.

## Мова

Виберіть мову для інтерфейсу Gramps Web. Налаштування мови зберігається у локальному сховищі браузера і застосовується лише до поточного пристрою.

## Тема

Виберіть між:

- **Системна** – слідує перевагам світлої/темної теми операційної системи (за замовчуванням)
- **Світла** – завжди використовувати світлу тему
- **Темна** – завжди використовувати темну тему

Налаштування теми зберігається у локальному сховищі браузера.

## Змінити електронну пошту

Введіть нову електронну адресу та натисніть **Надіслати**, щоб оновити адресу, пов'язану з вашим обліковим записом. Електронна адреса використовується для скидання пароля та (якщо налаштовано) для сповіщень.

## Змінити пароль

Введіть ваш поточний пароль та новий пароль, потім натисніть **Надіслати**. Якщо ви забули свій поточний пароль, скористайтеся посиланням **Забули пароль** на сторінці входу.

## Налаштування родинного дерева

### За замовчуванням перегляд родинного дерева

Встановлює, який тип діаграми відкривається за замовчуванням, коли ви переходите на сторінку [Родинне дерево](tree.md). Варіанти: Дерево предків, Дерево нащадків, Граф піскового годинника, Граф зв'язків та Фан-діаграма.

Це налаштування зберігається у локальному сховищі браузера.

## Інструменти розробника

### API токен

Копіює ваш поточний токен сесії в буфер обміну. Токен можна використовувати для безпосередньої аутентифікації через REST API, наприклад, в інтерактивному Swagger UI, який надається вашим екземпляром Gramps Web за адресою `/api/swagger-ui`.

Натисніть **Запустити Swagger**, щоб відкрити Swagger UI в новій вкладці з вашою сесією, вже доступною.

!!! note
    Токен сесії має короткий термін дії. Скопіюйте його негайно перед використанням у Swagger, оскільки він може закінчитися.
