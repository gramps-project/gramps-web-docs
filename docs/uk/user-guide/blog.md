# Використання вбудованого блогу

Блог призначений для представлення історій про дослідження вашої родинної історії.

У базі даних Gramps публікації блогу представлені як джерела з прикріпленою приміткою, що містить текст блогу, а за бажанням, медіафайли для зображень публікації блогу. Gramps Web розглядає кожне джерело з тегом `Blog` як статтю блогу.

## Додати публікацію блогу

Щоб додати публікацію блогу, ви можете використовувати Gramps Web або Gramps Desktop ([синхронізовано](../administration/sync.md) з Gramps Web), кроки однакові в обох випадках:

- Додайте нове джерело. Заголовок джерела буде заголовком вашої публікації блогу, автором джерела буде автор публікації.
- За бажанням, асоціюйте джерело з репозиторієм, що відповідає вашому блогу Gramps Web.
- Додайте нову примітку до джерела. Напишіть вашу публікацію блогу та скопіюйте текст у примітку.
- За бажанням, додайте один або кілька медіафайлів до вашого джерела. Перший медіафайл буде взято як зображення попереднього перегляду публікації, яке відображається над текстом. Усі медіафайли будуть показані під текстом у вигляді галереї.
- Додайте мітку `Blog` до джерела (створіть її, якщо вона не існує).

## Взаємозв'язок між блогом і джерелами

Оскільки публікації блогу є лише джерелами, усі статті блогу також з'являються у списку джерел і показуються як джерела в пошуках. У перегляді джерела є кнопка "показати в блозі", яка перенаправить вас до перегляду блогу для цієї публікації. URL публікації блогу також містить ID Gramps відповідного джерела, тому стаття за адресою `yourdomain.com/blog/S0123` відповідає джерелу за адресою `yourdomain.com/source/S0123`.

Внизу кожної публікації блогу є кнопка "деталі", яка перенаправить вас до перегляду джерела.
