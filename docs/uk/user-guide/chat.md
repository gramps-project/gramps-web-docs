# Використання AI чату

!!! info
    AI чат вимагає Gramps Web API версії 2.5.0 або вище та Gramps Web версії 24.10.0 або вище.


Вид чату в Gramps Web (якщо доступний у вашій установці) надає доступ до AI асистента, який може відповідати на запитання про ваше родинне дерево.

!!! warning
    Оскільки це все ще нова та еволюційна функція, деякі типи запитань працюють добре, а інші - ні. Також, як і з будь-яким AI асистентом, він може давати фактично неправильні відповіді, тому завжди перевіряйте інформацію.

## Як це працює

Щоб зрозуміти, які типи запитань асистент може відповісти, корисно зрозуміти, як це працює під капотом:

1. Користувач ставить запитання.
2. Gramps Web ідентифікує кілька (наприклад, десять) об'єктів Gramps, які, ймовірно, містять інформацію, що відповідає на запитання. Для цього використовується техніка, званою "семантичний пошук". Наприклад, якщо ви запитаєте "Які імена дітей Джона Доу?", якщо існує родина з Джоном Доу як батьком, ймовірно, вона буде серед найкращих результатів.
3. Gramps Web передає запитання користувача разом з отриманою контекстною інформацією великій мовній моделі ("чат-боту") і просить її витягти правильну відповідь.
4. Відповідь відображається користувачу.

## Як поставити запитання

Через те, як працює чат, наразі AI асистент не може відповідати на запитання про конкретні стосунки, окрім батьків або дітей, якщо ця інформація не міститься у вигляді тексту в нотатці.

Оскільки кожна відповідь базується на обмеженій кількості найкращих результатів семантичного пошуку, він також не може відповідати на запитання про статистику ("скільки людей у моїй базі даних ...").

Щоб уникнути неоднозначностей і непорозумінь, корисно формулювати запитання якомога детальніше.

Зверніть увагу, що великі мовні моделі є багатомовними, тому ви можете спілкуватися з ними своєю рідною мовою, і вони відповідатимуть тією ж мовою.

!!! tip
    Будь ласка, поділіться своїм досвідом щодо того, що працює, а що не працює, з громадою.
