# Використання AI чату

!!! info
    AI чат вимагає Gramps Web API версії 2.5.0 або вище та Gramps Web версії 24.10.0 або вище. Gramps Web API версії 3.6.0 впровадив можливості виклику інструментів для більш інтелектуальних взаємодій.

Чат-інтерфейс у Gramps Web (якщо доступний у вашій установці) надає доступ до AI асистента, який може відповідати на запитання про ваше родинне дерево.

!!! warning
    Оскільки це все ще нова та еволюційна функція, деякі типи запитань працюють добре, тоді як інші - ні. Також, як і з будь-яким AI асистентом, він може давати фактично неправильні відповіді, тому завжди перевіряйте інформацію.

## Як це працює

Щоб зрозуміти, які типи запитань асистент може відповідати, корисно зрозуміти, як це працює під капотом:

1. Користувач ставить запитання.
2. AI асистент може використовувати кілька підходів для знаходження відповідей:
   - **Семантичний пошук**: Gramps Web ідентифікує об'єкти у вашому родинному дереві, які, ймовірно, містять релевантну інформацію. Наприклад, якщо ви запитаєте "Які імена дітей Джона Доу?", родини з Джоном Доу як батьком будуть серед найкращих результатів.
   - **Виклик інструментів (Gramps Web API v3.6.0+)**: Асистент може безпосередньо запитувати вашу базу даних, використовуючи спеціалізовані інструменти для пошуку, фільтрації людей/подій/сімей/місць за конкретними критеріями, розрахунку відносин між особами та отримання детальної інформації.
3. Gramps Web передає запит разом з отриманою інформацією великій мовній моделі для формулювання відповіді.
4. Відповідь відображається вам.

## Що ви можете запитати

З можливостями виклику інструментів, впровадженими в Gramps Web API версії 3.6.0, AI асистент тепер може обробляти більш складні запитання:

- **Сімейні відносини**: "Хто є дідусями та бабусями Джейн Сміт?" або "Як Джон Доу пов'язаний з Мері Джонсон?"
- **Фільтровані пошуки**: "Покажіть мені всіх людей, народжених у Лондоні після 1850 року" або "Які події відбулися в Парижі?"
- **Запити на основі дати**: "Хто помер до 1900 року?" або "Перерахуйте шлюби, які відбулися між 1920 і 1950 роками"
- **Інформація про місця**: "Які місця є у Франції?" або "Розкажіть мені про церкву Святого Марії"
- **Загальні запитання**: "Які імена дітей Джона Доу?" або "Коли народилася Мері Сміт?"

## Поради для запитів

Щоб отримати найкращі результати від AI асистента:

- **Будьте конкретними**: Формулюйте ваше запитання з максимальною деталізацією, щоб уникнути неоднозначностей. Наприклад, "Коли Джон Сміт, народжений у 1850 році в Бостоні, одружився?" краще, ніж "Коли Джон Сміт одружився?"
- **Використовуйте правильні імена**: Зазначайте конкретні імена, місця та дати, коли це доречно.
- **Запитуйте одне за раз**: Розбивайте складні запитання на простіші частини для кращих результатів.
- **Використовуйте свою мову**: Великі мовні моделі є багатомовними, тому ви можете ставити запитання своєю мовою та отримувати відповіді тією ж мовою.

!!! tip
    Будь ласка, поділіться своїм досвідом щодо того, що працює, а що ні, з громадою.
