# Нещодавно переглянуті об'єкти

**Історія** у бічній панелі відкриває сторінку Нещодавно переглянуті об'єкти, яка містить список останніх об'єктів, які ви відкривали в Gramps Web, з найостаннішими переглянутими зверху. Запам'ятовується до 20 об'єктів.

Це корисно для швидкої навігації назад до особи, родини або іншого об'єкта, який ви переглядали раніше під час вашої сесії, без необхідності знову його шукати.

Натисніть будь-який запис, щоб повернутися до цього об'єкта. Використовуйте кнопку **Очистити все**, щоб скинути список.

Історія зберігається у локальному сховищі браузера – вона специфічна для поточного пристрою та браузера і не ділиться з іншими користувачами.
