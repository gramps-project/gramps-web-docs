# Сповіщення

**Сповіщення** — це елемент бокової панелі з іконкою дзвоника. Коли виникають помилки, значок показує кількість непрочитаних сповіщень. Натисніть на нього, щоб відкрити журнал сповіщень.

Журнал сповіщень — це запис помилок, які виникли під час вашої сесії — невдалі запити до API, помилки фонових завдань, невдачі збереження або помилки на рівні браузера. Кожен запис показує коротке повідомлення, джерело (Мережа, Завдання, Збереження або Браузер) та часову мітку.

Деякі сповіщення містять структуровані деталі. Натискаючи на такий запис, відкривається діалогове вікно з розбивкою даних про помилку та кнопкою **Копіювати JSON**. Це корисно при повідомленні про помилку, оскільки JSON містить точну інформацію про помилку з сервера.

Використовуйте **Очистити все**, щоб відхилити всі сповіщення.

!!! note
    Сповіщення зберігаються в пам'яті лише і очищуються, коли ви перезавантажуєте сторінку.
