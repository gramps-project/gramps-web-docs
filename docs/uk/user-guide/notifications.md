# Сповіщення

**Сповіщення** — це елемент бокової панелі з іконкою дзвоника. Коли виникають помилки або виконуються фонові завдання, значок показує кількість непрочитаних сповіщень. Натисніть на нього, щоб відкрити журнал сповіщень.

Журнал сповіщень має дві мети:

- Це запис помилок, які сталися під час вашої сесії — невдалі API запити, помилки фонових завдань, невдачі збереження або помилки на рівні браузера.
- Він відстежує прогрес довготривалих фонових завдань — таких як імпорт та експорт, генерація звітів, розпізнавання тексту OCR, оновлення бази даних та перебудова пошукових/семантичних індексів — показуючи їхній стан (наприклад, очікування, розпочато, в процесі) та сповіщаючи вас, коли вони завершуються або зазнають невдачі.

Кожен запис показує коротке повідомлення, джерело (Мережа, Завдання, Збереження або Браузер) та часову мітку.

Деякі сповіщення містять структуровані деталі. Натискання на такий запис відкриває діалогове вікно з розподілом даних про помилку та кнопкою **Копіювати JSON**. Це корисно при повідомленні про помилку, оскільки JSON містить точну інформацію про помилку з сервера.

Використовуйте **Очистити все**, щоб закрити всі сповіщення.

!!! note
    Сповіщення зберігаються лише в пам'яті і очищуються, коли ви перезавантажуєте сторінку.
