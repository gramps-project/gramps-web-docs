## Резервне копіювання вашого родоводу

Щоб створити резервну копію вашого родоводу, відкрийте сторінку Експорт у Gramps Web і виберіть формат Gramps XML.

Натискання на "експорт" згенерує файл і розпочне завантаження, як тільки він буде готовий.

Зверніть увагу, що якщо ваш користувач Gramps Web не має дозволу на перегляд приватних записів, експорт не буде повною резервною копією, оскільки не міститиме жодних приватних записів.

## Поділіться своїм родоводом з користувачами інших генеалогічних програм

Коли обмін генеалогічними даними у форматі Gramps XML не є варіантом, ви також можете експортувати файл GEDCOM. Зверніть увагу, що це не підходить як резервна копія вашого родоводу в Gramps Web.

## Резервне копіювання ваших медіафайлів

Щоб зробити резервну копію ваших медіафайлів, ви можете створити та завантажити ZIP-архів усіх медіафайлів на сторінці Експорт.

Зверніть увагу, що, особливо для великих дерев, це може бути витратною операцією для сервера і має виконуватись лише у разі крайньої необхідності.

Кращим варіантом для регулярного резервного копіювання ваших медіафайлів є використання [додатку Gramps Web Sync](sync.md) (який сам по собі не є рішенням для резервного копіювання) і створення інкрементних резервних копій на вашому локальному комп'ютері.

В обох випадках, якщо ваш користувач Gramps Web не має дозволу на перегляд приватних записів, експорт не міститиме файлів приватних медіаоб'єктів.

## Переміщення до іншої інстанції Gramps Web

Gramps Web не прив'язує вас до конкретного постачальника, і ви завжди можете переміститися до іншої інстанції Gramps Web, не втрачаючи жодних даних і без прямого доступу до жодного з серверів.

Щоб здійснити повну міграцію, виконайте ці кроки (припускаючи, що у вас є права власника дерева):

1. Перейдіть на сторінку Експорт і експортуйте своє дерево як файл Gramps XML (`.gramps`). Якщо ви використовуєте [додаток Sync](sync.md), ви також можете згенерувати експорт у настільному Gramps.
2. На сторінці Експорт згенеруйте та завантажте архів медіа. Якщо ви використовуєте [додаток Sync](sync.md), ви також можете просто ZIP-увати вашу локальну папку медіа Gramps.
3. Перейдіть до Налаштування > Адміністрування > Керування користувачами і натисніть кнопку "Експортувати деталі користувача". Це завантажить файл JSON.
4. У новій інстанції Gramps Web відкрийте сторінку Імпорт. Імпортуйте файл `.gramps`, експортований на кроці 1.
5. На сторінці Імпорт нової інстанції Gramps Web завантажте архів медіа (ZIP).
6. Перейдіть до Налаштування > Адміністрування > Керування користувачами нової інстанції Gramps Web. Натисніть кнопку "Імпортувати облікові записи користувачів" і завантажте файл JSON, завантажений на кроці 3.

Зверніть увагу, що, хоча ваші облікові записи користувачів будуть мігруватися, усім вашим користувачам потрібно буде встановити нові паролі, використовуючи посилання "забули пароль", оскільки паролі зберігаються в зашифрованій формі і не можуть бути експортовані.
