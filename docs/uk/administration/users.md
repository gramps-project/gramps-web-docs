# Керування користувачами

Інтерфейс керування користувачами доступний через **Налаштування > Керувати користувачами** (іконка користувача в верхній панелі програми). Він доступний лише для користувачів з роллю Власника або Адміністратора.

## Ролі користувачів

Дивіться [Система користувачів](../install_setup/users.md) для повного опису доступних ролей користувачів та їхніх прав.

## Перегляд та фільтрація користувачів

Сторінка керування користувачами показує таблицю всіх зареєстрованих облікових записів користувачів з наступними стовпцями:

- **Ім'я користувача** — ім'я для входу
- **Повне ім'я** — відображуване ім'я
- **E-mail** — електронна адреса користувача
- **Роль** — призначена роль (Гість, Член, Учасник, Редактор, Власник або Адміністратор)
- **Джерело облікового запису** — або "Пароль" (локальний обліковий запис), або назва зовнішнього постачальника ідентичності (наприклад, при використанні OIDC)

Використовуйте поле пошуку та випадаючий список ролей у верхній частині таблиці для фільтрації списку. Натисніть кнопку скидання фільтрів, щоб скинути всі фільтри.

## Редагування користувача

Натисніть на іконку редагування (олівець) у будь-якому рядку, щоб відкрити діалог редагування. Ви можете змінити:

- Повне ім'я
- Електронну адресу
- Роль

Це основний спосіб **активувати нового самостійно зареєстрованого користувача**: змініть їхню роль з *вимкненої* на будь-яку активну роль (наприклад, Член або Редактор).

## Додавання користувача вручну

Натисніть на іконку **додати користувача** (додати особу) над таблицею, щоб створити новий обліковий запис користувача без необхідності самостійної реєстрації. Заповніть ім'я користувача, повне ім'я, електронну адресу, пароль та роль у діалозі та натисніть **Зберегти**.

## Видалення користувача

Натисніть на іконку видалення (смітник) у будь-якому рядку та підтвердіть у діалозі. Цю дію не можна скасувати.

## Експорт та імпорт облікових записів користувачів

Ці кнопки корисні при [міграції на іншу інстанцію Gramps Web](export.md).

- **Експорт деталей користувача** (іконка завантаження) — завантажує файл JSON, що містить всі облікові записи користувачів (без паролів, оскільки паролі зберігаються в зашифрованій формі).
- **Імпорт облікових записів користувачів** (іконка групи-додати) — завантажує раніше експортований файл JSON для масового створення облікових записів користувачів. Усі імпортовані користувачі повинні будуть встановити новий пароль через посилання "Забули пароль", оскільки паролі не можуть бути перенесені.

## Посилання для реєстрації (тільки для налаштування з кількома деревами)

У налаштуванні з кількома деревами посилання для реєстрації нових користувачів показується вгорі сторінки керування користувачами. Ви можете скопіювати це посилання та поділитися ним з людьми, яких хочете запросити зареєструвати обліковий запис у вашому дереві.

!!! note
    У налаштуванні з одним деревом є загальне посилання "Зареєструватися" на сторінці входу; посилання для реєстрації для кожного дерева потрібне лише в установках з кількома деревами.

## Дозволи на чат з AI

Якщо чат з AI був увімкнений на сервері, випадаючий список у верхній частині сторінки дозволяє вам контролювати, які ролі користувачів можуть використовувати функцію чату:

- Усі (включаючи гостей)
- Члени та вище
- Учасники та вище
- Редактори та вище
- Лише власники та адміністратори
- Ніхто (вимкнути чат для всіх користувачів)
