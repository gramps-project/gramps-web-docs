# Специфікація API

Інтерактивна специфікація API (Swagger UI) надається безпосередньо вашим екземпляром Gramps Web за адресою `/api/swagger-ui` – наприклад, `https://mygrampsweb.example.com/api/swagger-ui`.

Ви також можете отримати доступ до неї з [Налаштувань користувача](../../user-guide/settings.md) через кнопку **Запустити Swagger**, яка відкриває Swagger UI з попередньо заповненим токеном вашої сесії.
