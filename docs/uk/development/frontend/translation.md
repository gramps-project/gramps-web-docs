Gramps Web використовує переклади Gramps Desktop, надані спільнотою, для понад 40 мов. Додаткові рядки, які потрібні лише для веб-додатку, перекладаються через Weblate.

Щоб зробити внесок, дотримуйтесь [інструкцій у Вікі Gramps](https://gramps-project.org/wiki/index.php/Translating_Gramps_using_Weblate), щоб почати працювати з Weblate для Gramps. Після автентифікації ви можете почати перекладати рядки як для Gramps Desktop, так і для Gramps Web.
