Якщо у вас виникли проблеми або вам потрібна допомога з Gramps Web, будь ласка, виберіть один із наведених варіантів.

[Форум :material-forum:](https://gramps.discourse.group/c/gramps-web/){ .md-button }
[Проблеми з бекендом :material-github:](https://github.com/gramps-project/gramps-web-api/issues){ .md-button }
[Проблеми з фронтендом :material-github:](https://github.com/gramps-project/gramps-web/issues){ .md-button }

Дивіться нижче кілька порад, з чого почати.

## Запитання

Офіційний форум Gramps Discourse має [окрему категорію для Gramps Web](https://gramps.discourse.group/c/gramps-web/). Будь ласка, використовуйте її, щоб задати будь-які питання, які у вас можуть виникнути щодо Gramps Web, наприклад:

- Питання щодо використання Gramps Web
- Питання щодо налаштування Gramps Web
- Виправлення проблеми з розгортанням Gramps Web
- Ідеї щодо покращення Gramps Web
- ...

## Повідомлення про проблеми

Якщо ви зіткнулися з проблемою, яку вважаєте помилкою в Gramps Web, будь ласка, повідомте про це через GitHub.

Є два окремі репозиторії GitHub для коду, що використовується в Gramps Web: один для інтерфейсу користувача (“фронтенд”) і один для серверного коду (“бекенд”):

- [Проблеми з фронтендом](https://github.com/gramps-project/gramps-web/issues)
- [Проблеми з бекендом](https://github.com/gramps-project/gramps-web-api/issues)

Якщо ви не впевнені, куди подати проблему, не хвилюйтеся і просто виберіть будь-який з двох – утримувачі зможуть перенести проблему, якщо це буде необхідно.

У будь-якому випадку, будь ласка, завжди включайте наступну інформацію у ваш звіт:

- Деталі про вашу конфігурацію (наприклад, файл docker-compose з прихованими чутливими значеннями або чи використовуєте ви хостингову версію, таку як Grampshub, або попередньо налаштоване зображення, таке як DigitalOcean)
- Інформація про версію. Щоб отримати її, перейдіть на вкладку "Інформація про систему" на сторінці Налаштувань у Gramps Web і скопіюйте/вставте значення в полі, яке має виглядати приблизно так:

```
Gramps 5.1.6
Gramps Web API 1.5.1
Gramps.js 24.1.0
locale: en
multi-tree: false
task queue: true
```

## Пропозиції щодо покращень

Для загальних ідей та обговорень щодо майбутніх покращень, не соромтеся відкрити обговорення на [форумі](https://gramps.discourse.group/c/gramps-web/). Ви також можете перевірити сторінки проблем (див. посилання вище), чи вже заплановано або працюється над певною функцією.

Для конкретних покращень з обмеженим обсягом, не соромтеся безпосередньо відкрити проблему з запитом на функцію в відповідному репозиторії GitHub для фронтенду або бекенду.
