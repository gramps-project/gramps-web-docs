# Uso de Atajos de Teclado

Los atajos de teclado te permiten navegar rápidamente a través de Gramps Web sin usar el ratón, ahorrándote clics y tiempo.

Desde cualquier vista, puedes saltar directamente a acciones y páginas comunes con solo unos pocos pulsaciones de teclas. Por ejemplo, presiona **`s`** en tu teclado para abrir la búsqueda, **`g`** + **`c`** para navegar al Árbol Genealógico, o **`n`** + **`p`** para crear una nueva persona.

Para ver una lista completa de todos los atajos de teclado disponibles, simplemente presiona **`?`** en cualquier momento.
