# Objetos Navegados Recientemente

**Historial** en la barra lateral abre la página de Objetos Navegados Recientemente, que lista los últimos objetos que abriste en Gramps Web, con los más recientemente vistos en la parte superior. Se recuerdan hasta 20 objetos.

Esto es útil para navegar rápidamente de regreso a una persona, familia u otro objeto que estabas viendo anteriormente en tu sesión, sin tener que buscarlo nuevamente.

Haz clic en cualquier entrada para volver a ese objeto. Usa el botón **Borrar Todo** para restablecer la lista.

El historial se almacena en el almacenamiento local del navegador: es específico para el dispositivo y navegador actuales, y no se comparte con otros usuarios.
