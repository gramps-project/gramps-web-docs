# Búsqueda Externa

Gramps Web te permite buscar personas en sitios web de genealogía externos directamente desde la página de detalles de una persona.


## Realizando una búsqueda externa

Para buscar a una persona en sitios web externos:

1. Abre la vista de detalles de una persona.
2. Haz clic en el botón de búsqueda externa.
3. Aparecerá un popup con enlaces a varios sitios web de genealogía, como CompGen, FamilySearch y Ancestry.
4. Hacer clic en un enlace abrirá la página de resultados de búsqueda en ese sitio web, prellenada con el nombre de la persona, lugares de nacimiento/fallecimiento y otra información relevante.

## Personalizando las opciones de búsqueda externa

Puedes personalizar qué sitios web de búsqueda externa se muestran en el popup.

1. En el popup de búsqueda externa, haz clic en el botón de engranaje en la parte superior para entrar en modo de edición.
2. Selecciona o deselecciona los sitios web que deseas mostrar u ocultar.
3. Guarda tus cambios.

## Agregando un motor de búsqueda personalizado

Puedes agregar nuevos sitios web de búsqueda externa con parámetros personalizados.

1. En el popup de búsqueda externa, haz clic en el botón de engranaje en la parte superior para entrar en modo de edición.
2. Haz clic en el botón "+ Agregar motor de búsqueda personalizado".
3. Aparecerá una sección con campos para "Nombre" y "URL de búsqueda". Ingresa el nombre del sitio web y la URL con parámetros de búsqueda, como `https://example.com/search?name={{name_given}}_{{name_surname}}&place={{birth_place}}`.
4. Haz clic en el botón "Agregar" para incluir el nuevo motor de búsqueda en la lista.
5. El nuevo motor de búsqueda se guardará y podrá ser personalizado más tarde.
