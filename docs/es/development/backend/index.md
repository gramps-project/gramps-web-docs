# Desarrollo del backend de Gramps Web

El backend de Gramps Web se llama [Gramps Web API](https://github.com/gramps-project/gramps-web-api/). De hecho, es más que solo un backend de aplicación: proporciona una API RESTful para leer y modificar una base de datos de Gramps.

Gramps Web API es una aplicación de Python y está impulsada por la biblioteca `gramps` (que también impulsa la aplicación de escritorio Gramps).

En esta sección de la documentación, puedes aprender cómo contribuir al desarrollo de Gramps Web API.
