# Especificación de la API

La especificación de la API interactiva (Swagger UI) se sirve directamente desde tu instancia de Gramps Web en `/api/swagger-ui` – por ejemplo `https://mygrampsweb.example.com/api/swagger-ui`.

También puedes acceder a ella desde [Configuraciones del Usuario](../../user-guide/settings.md) a través del botón **Iniciar Swagger**, que abre la Swagger UI con tu token de sesión pre-rellenado.
