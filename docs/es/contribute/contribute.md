# Cómo contribuir a Gramps Web

¡Gracias por considerar contribuir a Gramps Web! Dependiendo de tus habilidades e intereses, hay muchas maneras en las que puedes ayudar.

## Ayuda a traducir Gramps Web a tu idioma

Gramps y Gramps Web soportan varias docenas de idiomas diferentes. Ayudar a traducir la aplicación no requiere habilidades de programación. Utilizamos la poderosa herramienta en línea Weblate para las traducciones. Por favor, consulta [aquí](../development/frontend/translation.md) para obtener instrucciones sobre cómo comenzar.


## Ayuda a responder preguntas de otros usuarios

Si ya estás utilizando Gramps Web, ayuda a los nuevos usuarios a encontrar las respuestas a sus preguntas contribuyendo a las discusiones en la [categoría de Gramps Web](https://gramps.discourse.group/c/gramps-web/) del foro oficial de Gramps Discourse.

## Ayuda a mejorar la documentación

El sitio de documentación que estás viendo está alojado [en Github](https://github.com/gramps-project/gramps-web-docs). Las contribuciones para mejorarlo son muy bienvenidas. Simplemente envía una solicitud de extracción a ese repositorio.

## Contribuye al desarrollo del backend (Python)

Si eres programador de Python, contribuye al desarrollo de la [API de Gramps Web](https://github.com/gramps-project/gramps-web-api/), que es el backend que impulsa Gramps Web. Consulta [Desarrollo del backend](../development/backend/index.md) para obtener instrucciones sobre cómo comenzar.


## Contribuye al desarrollo del frontend (Javascript)

Si eres programador web, contribuye al desarrollo del [frontend responsivo de Gramps Web](https://github.com/gramps-project/gramps-web/). Consulta [Desarrollo del frontend](../development/frontend/index.md) para obtener instrucciones sobre cómo comenzar.

## Contribuye al complemento de sincronización (Python)

Si tienes experiencia desarrollando con Python y GTK, contribuye a la mejora del complemento de sincronización de Gramps Web. Puedes utilizar el [repositorio de desarrollo](https://github.com/DavidMStraub/gramps-web-sync/) para sugerir y enviar mejoras.
