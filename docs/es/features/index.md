# Características


![Título de la imagen](screenshots/sync.png){ align=left width="300"}

## Integración completa con Gramps Desktop

Gramps Web utiliza la misma estructura de **Modelo / Base de Datos** que [Gramps Desktop](https://gramps-project.org/) utiliza para almacenar datos genealógicos. Puedes explorar todos los mismos [Tipos de Registros](https://gramps-project.org/wiki/index.php/Gramps_Data_Model) que en Gramps Desktop: ***personas, familias, eventos, lugares, repositorios, fuentes, citas, objetos multimedia y notas.***

¡Usando el [Complemento de Sincronización de Gramps Web](../administration/sync.md) para Gramps Desktop, los datos pueden sincronizarse en ambas direcciones entre Gramps Web y Gramps Desktop! ¡Adelante, edita tus datos con Gramps Web o la aplicación de Gramps Desktop que conoces y amas - trabajan juntos sin problemas!

## Niveles de Privacidad / Acceso de Usuarios

¡Muchas personas quieren mantener algunos detalles privados y respetamos eso! Puedes marcar Registros como Privados y controlar qué usuarios pueden ver registros privados. Los registros privados se filtran en la capa de base de datos para máxima seguridad. Además, puedes controlar qué usuarios pueden agregar y editar.

## Sin bloqueos &ndash; importación y exportación de datos

Aparte de poder importar datos en varios formatos, incluyendo Gramps XML y GEDCOM, Gramps Web facilita a los usuarios descargar todos sus datos (datos del árbol familiar, archivos multimedia, cuentas de usuario) en cualquier momento, para fines de respaldo o para trasladarse a otro servidor. ¡Tus datos son solo tuyos!

## Aplicación de gestión de tareas integrada

Gramps Web viene con una aplicación de gestión de tareas integrada para organizar y planificar tu investigación genealógica. Las tareas se almacenan como fuentes en la base de datos de Gramps, por lo que forman parte de tus datos genealógicos y pueden ser accedidas y editadas en Gramps Desktop también.

## Integración de OpenHistoricalMap

Los mapas vectoriales históricos creados por el proyecto [OpenHistoricalMap](https://www.openhistoricalmap.org/) son el complemento perfecto para la cartografía genealógica. Usa el control deslizante de tiempo para desplazarte a través de la evolución de los lugares en tu historia familiar y muestra los lugares donde vivieron los ancestros o donde ocurrieron eventos.

`owner / owner` <br>
`editor / editor` <br>
`contributor / contributor` <br>
`member / member`
