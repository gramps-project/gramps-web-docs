# Guía de actualización de Gramps 5.2

**Gramps 5.2** fue lanzado el 23 de febrero de 2024. Dado que utiliza un nuevo esquema de base de datos de árbol genealógico en comparación con Gramps 5.1, se requieren pasos adicionales al actualizar Gramps Web para ser utilizado con Gramps 5.2. Esta página describe los pasos necesarios.

## Resumen de los pasos

Este es un resumen de los pasos requeridos para la actualización.  
También muestra qué pasos son ejecutados por el administrador del servidor Gramps Web, el propietario del árbol genealógico y si es necesario solo para los usuarios del [complemento de sincronización](../administration/sync.md).

Administrador del servidor | Propietario del árbol  | Acción
----|----|----
&nbsp; | ✓ | Hacer una copia de seguridad de su árbol genealógico
&nbsp; | ✓ <sup>1</sup> |  [Sincronizar Gramps Web con Gramps Desktop](../administration/sync.md)
✓ |  | [Actualizar Gramps Web](update.md)
&nbsp; | ✓ | Actualizar el esquema de la base de datos del árbol
&nbsp; | ✓ <sup>1</sup> | Actualizar Gramps Desktop
&nbsp; | ✓ <sup>1</sup> | Instalar el complemento de sincronización actualizado
&nbsp; | ✓ | Hacer una copia de seguridad de su árbol genealógico
&nbsp; | ✓ <sup>1</sup> | [Sincronizar Gramps Web con Gramps Desktop](../administration/sync.md)

<sup>1</sup> Solo necesario si sincroniza Gramps Web con Gramps Desktop.

## Pasos detallados

Esta sección proporciona más detalles sobre algunos de los pasos individuales mencionados anteriormente.

### Hacer una copia de seguridad de su árbol genealógico

En Gramps Web, vaya a la página "Exportar" y seleccione "Gramps XML (árbol genealógico)" (la opción predeterminada). Haga clic en "Generar" y espere hasta que comience la descarga del archivo .gramps.

También puede crear una copia de seguridad en Gramps Desktop [de la manera habitual](https://www.gramps-project.org/wiki/index.php/How_to_make_a_backup).

### Actualizar el esquema de la base de datos del árbol

Cuando acceda a Gramps Web por primera vez después de haberlo actualizado, se le presentará una pantalla que muestra el mensaje:

> El árbol genealógico que está intentando cargar está en una versión de esquema no soportada por esta versión de Gramps Web. Por lo tanto, no puede cargar este árbol genealógico sin actualizar su esquema. Esta acción no se puede deshacer.

Ahora puede hacer clic en el botón "Actualizar árbol genealógico". Se mostrará una barra de progreso y no debería tardar más de un minuto en completarse. No se preocupe, si cierra accidentalmente la ventana de su navegador, la conversión continuará en segundo plano.

!!! advertencia
    Si utiliza una configuración no estándar *sin* cola de tareas en segundo plano, la actualización de la base de datos podría fallar y su árbol genealógico podría corromperse. Puede verificar (antes de actualizar Gramps Web) si está utilizando una cola de tareas en segundo plano yendo a la sección "Información del sistema" en la configuración y asegurándose de que tenga "cola de tareas: true".

### Actualizar Gramps Desktop

Siga las instrucciones de instalación de Gramps Desktop para su sistema operativo.

### Instalar el complemento de sincronización actualizado

Puede usar [el nuevo Administrador de complementos](https://gramps-project.org/wiki/index.php/Addon:Plugin_Manager) incluido en Gramps 5.2: solo busque Gramps Web Sync, haga clic en instalar y reinicie Gramps.
