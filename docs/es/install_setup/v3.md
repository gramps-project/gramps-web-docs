# Guía de actualización de Gramps 6.0

Gramps 6.0 fue lanzado el 19 de marzo de 2025. Dado que utiliza un nuevo esquema de base de datos de árbol genealógico en comparación con Gramps 5.2, se requieren pasos adicionales al actualizar Gramps Web para que se use con Gramps 6.0. Esta página describe los pasos necesarios.

!!! info
    Última actualización: 4 de mayo de 2025


## Resumen de los pasos


Este es un resumen de los pasos requeridos para la actualización. 
También muestra qué pasos son ejecutados por el administrador del servidor Gramps Web, el propietario del árbol genealógico, y si es necesario solo para los usuarios del [complemento de sincronización](../administration/sync.md)

Administrador del servidor | Propietario del árbol  | Acción
----|----|----
&nbsp; | ✓ | Realiza una copia de seguridad de tu árbol genealógico
&nbsp; | ✓ <sup>1</sup> |  [Sincroniza Gramps Web con Gramps Desktop](../administration/sync.md)
✓ |  | [Actualiza Gramps Web](update.md)
&nbsp; | ✓ | Actualiza el esquema de la base de datos del árbol
&nbsp; | ✓ <sup>1</sup> | Actualiza Gramps Desktop
&nbsp; | ✓ <sup>1</sup> | Instala el complemento de sincronización actualizado
&nbsp; | ✓ | Realiza una copia de seguridad de tu árbol genealógico
&nbsp; | ✓ <sup>1</sup> | [Sincroniza Gramps Web con Gramps Desktop](../administration/sync.md)


<sup>1</sup> Solo necesario si sincronizas Gramps Web con Gramps Desktop.

## Pasos detallados

Esta sección proporciona más detalles sobre algunos de los pasos individuales mencionados anteriormente.

### Realiza una copia de seguridad de tu árbol genealógico

En Gramps Web, ve a la página "Exportar" y selecciona "Gramps XML (árbol genealógico)" (la opción predeterminada). Haz clic en "Generar" y espera hasta que comience la descarga del archivo .gramps.

También puedes crear una copia de seguridad en Gramps Desktop [de la manera habitual](https://www.gramps-project.org/wiki/index.php/How_to_make_a_backup).

### Actualiza el esquema de la base de datos del árbol

Cuando accedas a Gramps Web por primera vez después de haberlo actualizado, se te presentará una pantalla que muestra el mensaje: 

> El árbol genealógico que estás intentando cargar está en una versión de esquema no soportada por esta versión de Gramps Web. Por lo tanto, no puedes cargar este árbol genealógico sin actualizar su esquema. Esta acción no se puede deshacer.

Ahora puedes hacer clic en el botón "Actualizar árbol genealógico". Se mostrará una barra de progreso y no debería tardar más de un minuto en completarse. No te preocupes, si cierras accidentalmente la ventana de tu navegador, la conversión continuará en segundo plano.

!!! warning
    Si utilizas una configuración no estándar *sin* cola de tareas en segundo plano, la actualización de la base de datos podría fallar y tu árbol genealógico podría corromperse. Puedes verificar (antes de actualizar Gramps Web) si estás utilizando una cola de tareas en segundo plano yendo a la sección "Información del sistema" en la configuración y asegurándote de que tenga "cola de tareas: true".


### Actualiza Gramps Desktop

Sigue las instrucciones de instalación de Gramps Desktop para tu sistema operativo.

### Instala el complemento de sincronización actualizado

Puedes usar el Administrador de complementos en Gramps 6.0 para instalar el complemento de sincronización de Gramps Web. Busca Gramps Web Sync, haz clic en instalar y reinicia Gramps.
