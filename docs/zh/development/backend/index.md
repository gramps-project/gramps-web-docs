# Gramps Web 后端开发

Gramps Web 的后端称为 [Gramps Web API](https://github.com/gramps-project/gramps-web-api/)。实际上，它不仅仅是一个应用后端：它提供了一个 RESTful API 来读取和修改 Gramps 数据库。

Gramps Web API 是一个 Python 应用程序，并由 `gramps` 库本身提供支持（该库也为 Gramps 桌面应用程序提供支持）。

在本节文档中，您可以了解如何为 Gramps Web API 开发做出贡献。
