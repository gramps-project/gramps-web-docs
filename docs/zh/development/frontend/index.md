# Gramps Web 前端开发

[Gramps Web 前端](https://github.com/gramps-project/gramps-web) 是用 Javascript 编写的，并使用 [Lit](https://lit.dev/) 构建了 Web 组件。这使得将功能封装成可重用的组件成为可能。

前端设计为渐进式单页面 Web 应用，具有移动优先的体验。它大量使用 [Material Web Components](https://github.com/material-components/material-web) 来实现 Material Design 的外观和感觉。
