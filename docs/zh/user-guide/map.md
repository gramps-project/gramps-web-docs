# 地图

地图页面以交互式标记的形式在地理地图上显示您家谱中的所有地点。您可以从侧边栏访问此页面。

## 地点标记

只有在 Gramps 数据库中存储了 GPS 坐标的地点才会显示在地图上。没有坐标的地点将被静默省略。GPS 坐标可以在地点详细信息页面上设置（编辑地点并填写纬度和经度字段）。

!!! tip
    如果您的许多地点在地图上缺失，请打开地点详细信息页面并检查纬度和经度是否已设置。您可以直接从地点的编辑视图中添加或更正坐标。

每个有坐标的地点都显示为一个标记。点击标记会打开一个摘要卡片，显示地点名称及其关联的事件和人物。点击卡片中的地点名称以打开完整的地点详细信息页面。

## 搜索

地图左上角的搜索框允许您通过名称跳转到世界上的任何位置。这不会过滤树中的地点——它只是将地图平移和缩放到搜索的位置。

## 时间滑块

页面底部的时间滑块根据与其关联事件的年份过滤显示的地点标记：

- 拖动滑块选择一个年份。
- 仅显示与所选时间窗口内的事件相关联的地点。
- 使用此功能追踪您的祖先在历史特定时刻的居住地。

## 地图图层

左下角的图层切换按钮（叠层图标）让您在两个基础地图之间进行选择：

### 基础地图

默认图层，由 [OpenFreeMap](https://openfreemap.org) 提供支持（轻模式为 Liberty 风格，暗模式为暗色风格）。这是一个现代通用地图，适合定位地点。

### 历史地图

将基础地图切换到 [OpenHistoricalMap](https://www.openhistoricalmap.org)（OHM），这是一个社区驱动的项目，绘制了世界在不同时间点的样貌——可以将其视为 OpenStreetMap 的历史对应物。

当历史地图图层处于活动状态时，时间滑块还会过滤地图瓦片本身：OHM 以所选年份的样貌渲染地图，因此显示历史边界、地点名称和特征，而不是现代的。这使得可以在单一视图中看到您祖先的位置以及当时的地理和政治背景。

!!! note
    OpenHistoricalMap 的覆盖范围因地区和时期而异。贡献稀少的地区或时代可能显示有限的历史细节。如果您注意到缺失或不准确的历史数据，请考虑 [为 OpenHistoricalMap 贡献](https://www.openhistoricalmap.org)——这是一个任何人都可以编辑的开放社区项目。
