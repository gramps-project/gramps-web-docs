# 时间线

!!! info
    时间线视图需要足够新的 Gramps Web API 版本。如果您的服务器较旧，**时间线** 项目将从侧边栏中隐藏。

**时间线** 是一个侧边栏项目，按时间顺序在一个水平可缩放的时间线上显示您家谱中的所有事件。

## 浏览时间线

- 滚动、拖动或捏合以穿越时间；使用缩放按钮（或您的鼠标滚轮/捏合手势）进行放大和缩小。
- 当您放大时，事件标签会出现；点击事件点以查看其详细信息（以及同一日期上其他事件的详细信息）。

## 过滤

使用过滤字段将时间线聚焦于单个 **人** 或 **地点**：

- **人** – 仅显示与该人相关联的事件。一个切换按钮让您在 **人**（仅该个体的事件）、**祖先** 和 **后代**（所有祖先/后代的事件）之间切换。
- **地点** – 仅显示发生在该地点的事件。

清除过滤芯片以返回完整时间线。

## 跳转到时间线

在事件或地点的详细信息页面上，**在时间线中显示** 按钮会打开时间线视图，并滚动和缩放到该事件的日期。
