# 标签

标签是可以应用于 Gramps 数据库中任何对象的标签——人、家庭、事件、地点、来源、引用、存储库、备注和媒体。它们对于分组和过滤对象非常有用。标签存储在 Gramps 家谱数据库中，并在所有用户之间共享；它们也与在 Gramps Desktop 中创建的标签完全兼容。

## 管理标签

**标签** 页面可以从侧边栏访问，仅对具有编辑者角色或更高角色的用户可见。它显示所有现有标签，并允许您：

- **创建** 新标签，使用 + 按钮（右下角）
- **重命名** 标签，使用编辑（铅笔）图标
- **更改标签颜色**，使用颜色选择器
- **删除** 标签，使用删除图标

!!! note
    删除标签会将其从所有应用该标签的对象中移除。

## 将标签应用于对象

在编辑模式下，可以在对象的详细信息页面上将标签应用于对象或从对象中移除标签。

## 按标签过滤

所有对象列表页面（人、家庭、事件、地点、来源、引用、存储库、备注、媒体）都包含标签过滤器。使用它仅显示应用了特定标签的对象。

## 特殊标签

在 Gramps Web 中，有两个标签具有特殊含义：

- **`Blog`** – 任何标记为 `Blog` 的来源被视为博客文章，并出现在 [博客](blog.md) 视图中
- **`ToDo`** – 任何标记为 `ToDo` 的备注被视为研究任务，并出现在 [任务](tasks.md) 视图中

这些标签在您首次使用博客或任务功能时会自动创建。重命名或删除它们将破坏相应的功能。
