# 书签

书签让您保存链接到常访问的对象（人、家庭、地点、来源等），以便快速访问。

## 添加和删除书签

每个对象详细信息页面的顶部工具栏都有一个书签图标（书签轮廓）。点击它以将对象添加为书签——图标变为填充的书签。再次点击以删除书签。

## 查看书签

**书签**页面可以从侧边栏访问，并列出您所有的书签对象。点击任何条目以导航到该对象的详细信息页面。

## 用例

一个典型的用例是书签您当前正在研究的人或家庭，以便您可以快速返回而无需搜索。您还可以书签您经常参考的关键来源或地点。

## 存储

书签存储在浏览器的本地存储中，范围限定于当前树。它们特定于您正在使用的设备和浏览器，并且不会在设备之间同步或与其他用户共享。

!!! note
    Gramps Web 书签独立于 Gramps Desktop 中的书签。它们不存储在 Gramps 数据库中，并且不会出现在 Gramps Desktop 中，反之亦然。
