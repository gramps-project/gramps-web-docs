# 使用 AI 聊天

!!! info
    AI 聊天需要 Gramps Web API 版本 2.5.0 或更高版本，以及 Gramps Web 版本 24.10.0 或更高版本。



Gramps Web 中的聊天视图（如果在您的安装中可用）提供了一个 AI 助手，可以回答有关您的家谱的问题。

!!! warning
    由于这仍然是一个新兴的功能，某些类型的问题效果很好，而其他类型则不然。此外，与任何 AI 助手一样，它可能会给出事实不准确的答案，因此请务必始终进行双重检查。

## 工作原理

要了解助手可以回答哪些类型的问题，了解其背后的工作原理是有帮助的：

1. 用户提出一个问题。
2. Gramps Web 确定一系列（例如，十个）最有可能包含回答该问题信息的 Gramps 对象。为此，它使用了一种称为“语义搜索”的技术。例如，如果您问“约翰·杜的孩子叫什么名字？”，如果存在一个以约翰·杜为父亲的家庭，它很可能会出现在前几条结果中。
3. Gramps Web 将用户问题与检索到的上下文信息一起提供给一个大型语言模型（“聊天机器人”），并要求其提取正确的答案。
4. 答案显示给用户。

## 如何提问

由于聊天的工作方式，AI 助手目前无法回答关于特定关系（除了父母或子女）的提问，除非这些信息以文本形式包含在备注中。

由于每个答案基于有限数量的顶级语义搜索结果，因此它也无法回答有关统计的问题（“我的数据库中有多少人...”）。

为了避免歧义和误解，尽可能详细地表述问题是有帮助的。

请注意，大型语言模型是多语言的，因此您可以用自己的语言与其交流，它将用相同的语言回答。

!!! tip
    请与社区分享您关于有效和无效的体验。
