# 使用 AI 聊天

!!! info
    AI 聊天需要 Gramps Web API 版本 2.5.0 或更高版本，以及 Gramps Web 版本 24.10.0 或更高版本。Gramps Web API 版本 3.6.0 引入了工具调用功能，以实现更智能的交互。

Gramps Web 中的聊天视图（如果在您的安装中可用）提供了一个 AI 助手，可以回答有关您的家谱的问题。

!!! warning
    由于这仍然是一个新功能并且在不断发展，一些类型的问题效果很好，而另一些则不然。此外，与任何 AI 助手一样，它可能会给出事实不正确的答案，因此请务必始终进行双重检查。

## 工作原理

要了解助手可以回答哪些类型的问题，了解其背后的工作原理是有帮助的：

1. 用户提出问题。
2. AI 助手可以使用多种方法来寻找答案：
   - **语义搜索**：Gramps Web 确定您家谱中最有可能包含相关信息的对象。例如，如果您问“约翰·多的孩子叫什么名字？”，那么以约翰·多为父亲的家庭将位于结果的前列。
   - **工具调用（Gramps Web API v3.6.0+）**：助手可以直接使用专门的工具查询您的数据库，以特定标准搜索、过滤人员/事件/家庭/地点，计算个体之间的关系，并检索详细信息。
3. Gramps Web 将问题和检索到的信息提供给大型语言模型，以制定答案。
4. 答案将显示给您。

## 您可以问什么

随着 Gramps Web API 版本 3.6.0 中引入的工具调用功能，AI 助手现在可以处理更复杂的问题：

- **家庭关系**： “简·史密斯的祖父母是谁？”或“约翰·多与玛丽·约翰逊有什么关系？”
- **过滤搜索**： “给我显示 1850 年后在伦敦出生的所有人”或“在巴黎发生了哪些事件？”
- **基于日期的查询**： “谁在 1900 年之前去世？”或“列出 1920 年到 1950 年之间发生的婚姻”
- **地点信息**： “法国有哪些地方？”或“告诉我关于圣玛丽教堂的事”
- **一般问题**： “约翰·多的孩子叫什么名字？”或“玛丽·史密斯是什么时候出生的？”

## 提问技巧

为了从 AI 助手那里获得最佳结果：

- **具体**：尽可能详细地表述您的问题，以避免歧义。例如，“1850 年在波士顿出生的约翰·史密斯是什么时候结婚的？”比“约翰·史密斯什么时候结婚？”更好。
- **使用专有名词**：在相关时提及具体的名字、地点和日期。
- **一次问一件事**：将复杂的问题分解为更简单的部分，以获得更好的结果。
- **使用您的语言**：大型语言模型支持多种语言，因此您可以用自己的语言提问，并以相同的语言获得答案。

!!! tip
    请与社区分享您关于有效和无效的体验。
