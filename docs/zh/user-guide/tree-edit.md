# 编辑家谱

祖先树视图提供了一种快速构建和扩展家谱结构的方法，可以直接从图表中进行，而无需离开树视图。

!!! note
    树编辑功能在祖先树、后代树、沙漏图和关系图选项卡中可用。在扇形图中不可用。只有具有编辑者角色或更高权限的用户才能使用此功能。

## 进入和退出编辑模式

具有编辑权限的用户在支持的图表页面上会看到一个浮动的铅笔按钮（右下角）。点击它以进入编辑模式，或按 `e` [键盘快捷键](shortcuts.md)。应用程序栏将显示编辑模式指示器。

每次更改（添加或链接一个人）在对话框中确认后会立即保存到数据库中。要退出编辑模式，请点击应用程序栏中的关闭按钮，或切换到其他图表选项卡。

!!! note
    切换图表选项卡会自动退出编辑模式。

## 添加家庭成员

在编辑模式下，图表中的每个人卡片上都会显示一个小 **+** 按钮。点击它将打开该人的 **添加家庭成员** 对话框，每种可用关系会显示一个或两个图标：

- **父亲** – 仅在该人在其主要父母家庭中尚未有父亲时显示
- **母亲** – 仅在该人在其主要父母家庭中尚未有母亲时显示
- **子女**
- **配偶**

每种关系提供两个并排的选项：

- **链接图标** – 连接数据库中已存在的人
- **+ 图标** – 创建一个全新的人

## 链接现有人员

选择链接图标并按姓名搜索该人。从结果中选择他们以创建家庭链接。

对于 **子女**，您还可以设置父亲和母亲的关系类型（出生、收养、继子女等）。

对于 **配偶**，您可以设置家庭关系类型（已婚、民事结合、未知等）。

## 创建新人员

选择 + 图标以打开一个最简创建表单。可用的字段包括：

- **姓名**（名字和姓氏）
- **性别**（未知、男性、女性、其他）
- **出生日期和地点**
- **死亡日期和地点**

新人员将被保存并立即与图表中选定的人链接。

## 限制

树编辑视图用于快速构建家谱的框架。对于超出基本信息的内容，请使用该人的详细页面：引用、备注、媒体、属性、别名、现有事件详情、多父母家庭（此处仅可访问主要家庭）以及管理多个配偶。

使用 **人员详情**（工具栏中的账户卡片图标）直接跳转到当前居中人员的完整编辑视图。
