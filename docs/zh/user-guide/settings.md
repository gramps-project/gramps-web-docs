# 用户设置

用户设置可以通过应用程序顶部工具栏中的用户图标访问，然后选择 **用户设置**。更改会立即生效，除非另有说明。

## 用户信息

显示您的 **用户名** 和当前 **用户角色**（例如：访客、成员、编辑）。这些信息为只读。

## 语言

选择 Gramps Web 界面的语言。语言设置存储在浏览器的本地存储中，仅适用于当前设备。

## 主题

选择以下选项之一：

- **系统** – 遵循操作系统的亮/暗模式偏好（默认）
- **浅色** – 始终使用浅色主题
- **深色** – 始终使用深色主题

主题设置存储在浏览器的本地存储中。

## 更改电子邮件

输入新的电子邮件地址，然后点击 **提交** 以更新与您的帐户关联的地址。电子邮件地址用于密码重置和（如果配置）通知。

## 更改密码

输入您当前的密码和新密码，然后点击 **提交**。如果您忘记了当前密码，请在登录页面使用 **忘记密码** 链接。

## 家谱偏好

### 默认家谱视图

设置在您导航到 [家谱](tree.md) 页面时默认打开的图表类型。选项包括祖先树、后代树、沙漏图、关系图和扇形图。

此偏好存储在浏览器的本地存储中。

## 开发者工具

### API 令牌

将您当前的会话令牌复制到剪贴板。该令牌可用于直接对 REST API 进行身份验证，例如在您 Gramps Web 实例提供的 `/api/swagger-ui` 中的交互式 Swagger UI。

点击 **启动 Swagger** 以在新标签页中打开 Swagger UI，您的会话将自动可用。

!!! note
    会话令牌是短暂的。请在使用它之前立即复制，因为它可能会过期。
