# 用户指南

本节文档记录了 Gramps Web 用户可用的功能。

!!! note "没有看到所有功能？"
    Gramps Web 使用基于角色的权限系统。一些功能——例如编辑数据、管理标签或查看私人记录——仅对具有足够权限的用户可用。您可以在 [用户设置](settings.md) 中检查您当前的角色。如果您需要更多访问权限，请联系您的树的所有者或管理员。有关所有角色的描述，请参见 [用户系统](../install_setup/users.md)。

## 导航界面

### 主要导航

侧边栏（或移动设备上的汉堡菜单）是在各个部分之间移动的主要方式：

- **首页** – 仪表板（见下文）
- **博客** – 以博客文章形式撰写的家族历史故事
- **家谱** – 互动树图
- **时间线** – 树中事件的时间顺序视图（需要足够新的 Gramps Web API 版本）
- **地图** – 树中地点的地理视图
- **DNA** – DNA 匹配分析工具
- **列表** – 浏览每种类型的所有对象：人物、家庭、事件、地点、来源、引用、资料库、笔记
- **媒体** – 浏览所有媒体文件（照片、文档等）
- **助手** – AI 聊天助手（如果管理员启用）
- **历史** – 最近更改的对象
- **书签** – 您保存的书签
- **任务** – 研究任务
- **报告** – 生成报告
- **导出** – 导出家谱
- **修订** – 完整的交易历史（可见于成员及以上）
- **通知** – 过去的通知

!!! note
    标签不再从侧边栏管理——标签管理已移至 [管理设置](../administration/settings.md#tags)（仅限所有者/管理员）。有关标签使用方式，请参见 [标签](tags.md)。

### 顶部应用栏

每个页面顶部的栏包含：

- **添加**（加号图标，仅对贡献者及以上可见） – 打开菜单以创建新对象：人物、家庭、事件、地点、来源、引用、资料库、笔记、媒体对象或任务
- **搜索**（放大镜） – 打开搜索页面
- **用户图标** – 打开设置菜单：用户设置、管理（仅限所有者）、管理用户（仅限所有者）、系统信息

## 首页（仪表板）

首次登录时显示仪表板。它有两列：

**左侧列：**

- **首页人物卡** – 显示您选择的首页人物的姓名、照片（如果可用）和关键信息，并提供链接到其完整个人资料和快速导航到家谱。单击卡片上的 **设置首页人物** 按钮以搜索并选择其他人物。
- **纪念日** – 根据今天的日期，显示来自树的即将到来的生日和纪念日。
- **最近更改** – 最近修改的对象的简短列表，有助于跟踪协作编辑。

**右侧列：**

- **最近的博客文章** – 来自 [博客](blog.md) 的最新条目（如果存在）。
- **统计信息** – 树中对象数量的摘要（人物、家庭、事件等的数量）。

如果树的管理员配置了 **首页备注** 和/或 **首页图像**，这些内容将显著显示在主列上方。当两者都设置时，图像出现在备注文本旁边。有关如何配置这些内容，请参见 [管理设置](../administration/settings.md#customization)。

!!! tip
    如果树是空的并且您具有编辑权限，仪表板将显示“开始使用”的提示，并提供添加您的第一个人物或导入家谱文件的按钮。
