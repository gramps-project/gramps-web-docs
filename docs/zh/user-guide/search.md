# 搜索

通过点击顶部应用栏中的放大镜图标，或按 `s` [键盘快捷键](shortcuts.md) 可以访问搜索页面。

## 全文搜索

在搜索字段中输入任何查询并按 Enter（或点击搜索按钮）。Gramps Web 会在所有对象类型中进行搜索——人员、家庭、事件、地点、来源、引用、库、笔记和媒体——并返回按相关性排名的匹配结果。

结果显示对象类型、名称和简要摘要。点击任何结果以打开相应的详细页面。

可以使用尾随的 `*` 作为通配符，例如 `Mey*` 匹配 "Meyer"、"Meyers"、"Meyerhofer" 等。

## 按对象类型过滤

在搜索字段下方，每个对象类型（人员、家庭、事件、地点等）的切换按钮让您可以将结果缩小到一个或多个特定类型。默认情况下会搜索所有类型。激活一个或多个切换按钮将结果限制为仅这些类型。

## 语义搜索

如果服务器管理员启用了 [语义（AI 驱动）搜索](../install_setup/configuration.md)，搜索页面右上角会出现一个模式切换，提供两个选项：

- **搜索** – 标准全文搜索（默认）
- **语义** – AI 驱动的搜索，理解您的查询含义而不是匹配确切的词语

语义搜索对于自然语言查询非常有用，例如 "19 世纪巴伐利亚的农民"。它需要填充语义搜索索引；请参见 [管理设置](../administration/settings.md) 以了解如何构建或更新索引。
