# 外部搜索

Gramps Web 允许您直接从个人详细信息页面搜索外部家谱网站上的人员。

## 执行外部搜索

要在外部网站上搜索某个人：

1. 打开该人的详细视图。
2. 点击外部搜索按钮。
3. 将弹出一个包含各种家谱网站链接的窗口，例如 CompGen、FamilySearch 和 Ancestry。
4. 点击链接将打开该网站的搜索结果页面，预填充该人的姓名、出生/死亡地点和其他相关信息。

## 自定义外部搜索选项

您可以自定义在弹出窗口中显示哪些外部搜索网站。

1. 在外部搜索弹出窗口中，点击顶部的齿轮按钮以进入编辑模式。
2. 选择或取消选择您想要显示或隐藏的网站。
3. 保存您的更改。

## 添加自定义搜索引擎

您可以添加带有自定义参数的新外部搜索网站。

1. 在外部搜索弹出窗口中，点击顶部的齿轮按钮以进入编辑模式。
2. 点击“+ 添加自定义搜索引擎”按钮。
3. 将出现一个包含“名称”和“搜索 URL”字段的部分。输入网站名称和带有可搜索参数的 URL，例如 `https://example.com/search?name={{name_given}}_{{name_surname}}&place={{birth_place}}`。
4. 点击“添加”按钮以将新搜索引擎包含在列表中。
5. 新搜索引擎将被保存，并可以在以后进行自定义。
