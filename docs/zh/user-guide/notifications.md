# 通知

**通知** 是一个带有铃铛图标的侧边栏项目。当发生错误时，徽章会显示未读通知的数量。点击它以打开通知日志。

通知日志是您会话期间发生的错误记录——失败的 API 请求、后台任务错误、保存失败或浏览器级错误。每个条目显示一条简短消息、来源（网络、任务、保存或浏览器）和时间戳。

一些通知包含结构化细节。点击此类条目会打开一个对话框，显示错误数据的详细信息和一个 **复制 JSON** 按钮。这在报告错误时非常有用，因为 JSON 包含来自服务器的确切错误信息。

使用 **清除所有** 来解除所有通知。

!!! note
    通知仅存储在内存中，并在您重新加载页面时清除。
