# 家谱

家谱页面可以从侧边栏访问，并显示以所选人物为中心的交互式图表。页面顶部的选项卡提供五种图表类型。

## 选择起始人物

所有图表均以当前在树中选择的人物为起点（在工具栏中显示）。当前居中的人物会用阴影高亮显示。使用 **家庭人物** 按钮跳回您的家庭人物，或使用 **返回** 按钮返回到之前查看的人物。点击任何图表中的人物卡片将图表重新居中到该人物。

如果尚未设置家庭人物，页面将提示您前往主页并设置一个。

## 图表类型

### 祖先树

显示所选人物祖先的谱系图。父母位于左侧（或右侧，具体取决于布局），祖父母更远，依此类推。

### 后代树

显示所选人物的后代——子女、孙子女等。

### 沙漏图

将所选人物的祖先放在上方，后代放在下方，合并为一个视图。

### 关系图

显示两个人之间的关系路径。所选人物是一个端点；点击图表中的任何其他人物以设置第二个端点，并显示它们之间的最短关系路径。

!!! note
    同一个人物可能在图表中出现多次——例如，当某人结过多次婚，并在路径上的多个家庭单元中出现时。这是故意的，而不是一个错误。

### 扇形图

一个圆形的谱系图。祖先从位于中心的所选人物向外辐射。

## 导航控件

所有图表类型共享一个工具栏，包含以下按钮：

- **家庭人物** – 跳回您的家庭人物
- **返回** – 返回到之前居中的人物
- **人物详情** – 打开当前居中人物的完整个人资料页面
- **偏好设置** – 打开对特定图表显示选项进行调整的对话框（见下文）

所有图表支持 **平移**（点击并拖动）和 **缩放**（滚轮或捏合）。

## 图表偏好设置

**偏好设置** 对话框（工具栏中的齿轮图标）允许您根据图表类型调整以下选项：

- **最大祖先世代** – 显示多少代祖先
- **最大后代世代** – 显示多少代后代
- **最大分离度** – 对于关系图，搜索的最大路径长度
- **最大显示图像数量** – 限制图表中显示的个人资料照片数量以提高性能
- **姓名显示格式** – 控制在人物卡片上显示姓名的方式

点击 **重置** 恢复默认设置，或点击 **关闭** 应用您的更改。

图表偏好设置存储在浏览器的本地存储中，因此在同一设备上的会话之间会持续存在。

## 默认图表类型

您首次打开家谱页面时显示的图表类型可以在 [用户设置](settings.md) 中进行配置。您选择的默认设置适用于您所有的设备。
