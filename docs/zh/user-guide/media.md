有两种方法可以添加新的媒体文件（图像、音频文件、视频文件或其他任何文件）：

## 添加新的独立媒体文件

要添加独立媒体文件，请点击应用程序顶部工具栏中的 + 图标，然后选择“媒体对象”。

点击“选择文件”以从您的计算机中选择一个文件。在移动设备上，点击此按钮将给您直接使用设备相机拍照的选项。

可选地，

- 在“标题”下输入媒体文件的描述
- 输入日期
- 将媒体文件设置为私有（这将使其仅对具有足够授权的用户可见）

点击“添加”以上传文件并创建媒体对象。

## 添加新的媒体文件并将其链接到另一个对象

在 Gramps 中，以下对象类型可以附加媒体对象：人物、家庭、事件、地点、来源和引用。

在任何对象的详细视图中，点击右下角的蓝色编辑按钮（如果您看不到它，说明您的用户没有编辑权限）。点击“图库”选项卡，然后点击蓝色 + 按钮。

将打开一个对话框，提供与上一节中描述的相同字段。点击“保存”以上传文件，添加新的媒体对象，并将其链接到所查看的对象。
