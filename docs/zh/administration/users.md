# 管理用户

用户管理界面可以通过 **设置 > 管理用户**（应用程序顶部工具栏中的用户图标）访问。仅对拥有所有者或管理员角色的用户可用。

## 用户角色

有关可用用户角色及其权限的完整描述，请参见 [用户系统](../install_setup/users.md)。

## 查看和过滤用户

管理用户页面显示所有注册用户帐户的表格，包含以下列：

- **用户名** – 登录名
- **全名** – 显示名称
- **电子邮件** – 用户的电子邮件地址
- **角色** – 分配的角色（访客、成员、贡献者、编辑、所有者或管理员）
- **帐户来源** – "密码"（本地帐户）或外部身份提供者的名称（例如，使用 OIDC 时）

使用表格顶部的搜索字段和角色下拉菜单来过滤列表。点击清除过滤器按钮以重置所有过滤器。

## 编辑用户

点击任意行上的编辑（铅笔）图标以打开编辑对话框。您可以更改用户的：

- 全名
- 电子邮件地址
- 角色

这是 **启用新自注册用户** 的主要方式：将他们的角色从 *禁用* 更改为任何活动角色（例如，成员或编辑）。

## 手动添加用户

点击表格上方的 **添加用户**（添加人员）图标，直接创建一个新用户帐户，而无需自注册。在对话框中填写用户名、全名、电子邮件地址、密码和角色，然后点击 **保存**。

## 删除用户

点击任意行上的删除（垃圾桶）图标并确认对话框。此操作无法撤销。

## 导出和导入用户帐户

这些按钮在 [迁移到不同的 Gramps Web 实例](export.md) 时非常有用。

- **导出用户详细信息**（下载图标） – 下载一个包含所有用户帐户的 JSON 文件（不包含密码，因为密码以加密形式存储）。
- **导入用户帐户**（添加组图标） – 上传先前导出的 JSON 文件以批量创建用户帐户。所有导入的用户需要通过“忘记密码”链接设置新密码，因为密码无法转移。

## 注册链接（仅限多树设置）

在多树设置中，新用户的注册链接显示在管理用户页面的顶部。您可以复制此链接并与希望邀请注册您树上帐户的人分享。

!!! 注意
    在单树设置中，登录页面上有一个通用的“注册”链接；每棵树的注册链接仅在多树安装中需要。

## AI 聊天权限

如果服务器上启用了 AI 聊天，页面顶部的下拉菜单让您控制哪些用户角色可以使用聊天功能：

- 所有人（包括访客）
- 成员及以上
- 贡献者及以上
- 编辑及以上
- 仅所有者和管理员
- 无人（禁用所有用户的聊天）
