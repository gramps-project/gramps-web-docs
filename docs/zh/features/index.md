# 功能


![图片标题](screenshots/sync.png){ align=left width="300"}

## 与 Gramps 桌面版的完全集成

Gramps Web 使用与 [Gramps 桌面版](https://gramps-project.org/) 相同的 **模型/数据库** 结构来存储家谱数据。您可以浏览与 Gramps 桌面版相同的 [记录类型](https://gramps-project.org/wiki/index.php/Gramps_Data_Model)：***个人、家庭、事件、地点、资料库、来源、引用、媒体对象和笔记。***

使用 [Gramps Web 同步插件](../administration/sync.md) 可以在 Gramps Web 和 Gramps 桌面版之间双向同步数据！尽管使用您熟悉和喜爱的 Gramps Web 或 Gramps 桌面应用程序来编辑数据 - 它们可以无缝协作！

<div style="clear:both;"></div>

![图片标题](screenshots/list.png){ align=right width="400"}

## 隐私级别/用户访问

许多人希望保留某些细节的私密性，我们对此表示尊重！您可以将记录标记为私密，并控制哪些用户可以查看私密记录。私密记录在数据库层被过滤，以确保最大安全性。此外，您可以控制用户能够添加和编辑的内容。

<div style="clear:both;"></div>

![图片标题](screenshots/mobile.png){ align=left width="400"}

## 网络浏览器便利性

可以从任何支持网络的设备访问 Gramps Web。您可以上传照片、创建或编辑记录、向他人展示您的家谱，或者在下次家庭聚会上查找您记不起来的家庭成员姓名！


<div style="clear:both;"></div>

## 互动家谱图

以祖先图、后代图、沙漏图或扇形图的形式浏览您的家谱，提供高质量的互动图形和可配置的世代数。

## 无锁定 - 数据导入和导出

除了能够以各种格式导入数据，包括 Gramps XML 和 GEDCOM，Gramps Web 还使用户能够随时下载他们的所有数据（家谱数据、媒体文件、用户帐户），以备份或迁移到其他服务器。您的数据仅属于您！

<div style="clear:both;"></div>

## 生成可打印报告

由于它直接建立在支持 Gramps 桌面版的核心上，您可以直接从浏览器生成几乎所有 [报告](https://gramps-project.org/wiki/index.php/Gramps_5.2_Wiki_Manual_-_Reports)，包括关系图或 PDF 格式的书籍报告。

## 强大的全文搜索

搜索引擎支持搜索所有 Gramps 对象类型，包括文本笔记的内容，并支持通配符语法和逻辑运算符。

## DNA 匹配和染色体浏览器

如果您有来自某个 DNA 家谱提供商的 DNA 匹配数据，可以上传并以未来证明的方式存储它，并在互动染色体浏览器中查看您的匹配。

![图片标题](screenshots/tasks.png){ align=left width="400"}

## 集成任务管理应用

Gramps Web 配备了集成的任务管理应用，以组织和规划您的家谱研究。这些任务作为来源存储在 Gramps 数据库中，因此它们是您家谱数据的一部分，并且可以在 Gramps 桌面版中访问和编辑。

<div style="clear:both;"></div>

---


![图片标题](screenshots/ohm.png){ align=right width="400"}

## OpenHistoricalMap 的集成

由 [OpenHistoricalMap](https://www.openhistoricalmap.org/) 项目创建的历史矢量地图是家谱映射的完美补充。使用时间滑块浏览您家族历史中地点的演变，并显示祖先居住或事件发生的地点。
