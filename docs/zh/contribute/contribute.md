# 如何为 Gramps Web 贡献

感谢您考虑为 Gramps Web 贡献！根据您的技能和兴趣，有很多方法可以帮助我们。

## 帮助将 Gramps Web 翻译成您的语言

Gramps 和 Gramps Web 支持数十种不同的语言。帮助翻译应用程序不需要任何编程技能。我们使用强大的 Weblate 在线工具进行翻译。请参见 [这里](../development/frontend/translation.md) 获取入门说明。


## 帮助回答其他用户的问题

如果您已经在使用 Gramps Web，请通过参与 [Gramps Web 类别](https://gramps.discourse.group/c/gramps-web/) 的官方 Gramps Discourse 论坛讨论，帮助新用户找到他们问题的答案。

## 帮助改善文档

您正在查看的文档网站托管在 [Github](https://github.com/gramps-project/gramps-web-docs) 上。欢迎对其进行改进的贡献。只需向该存储库提交拉取请求。

## 贡献后端开发 (Python)

如果您是 Python 程序员，请为 [Gramps Web API](https://github.com/gramps-project/gramps-web-api/) 的开发做出贡献，该 API 是支持 Gramps Web 的后端。请参见 [后端开发](../development/backend/index.md) 获取入门说明。


## 贡献前端开发 (Javascript)

如果您是网页程序员，请为 [Gramps Web 的响应式前端](https://github.com/gramps-project/gramps-web/) 的开发做出贡献。请参见 [前端开发](../development/frontend/index.md) 获取入门说明。

## 贡献 Sync 插件 (Python)

如果您有使用 Python 和 GTK 开发的经验，请为 Gramps Web Sync 插件的改进做出贡献。您可以使用 [开发存储库](https://github.com/DavidMStraub/gramps-web-sync/) 提出和提交改进建议。
