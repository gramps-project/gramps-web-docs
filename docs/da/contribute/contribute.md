# Sådan bidrager du til Gramps Web

Tak fordi du overvejer at bidrage til Gramps Web! Afhængigt af dine færdigheder og interesser er der mange måder, du kan hjælpe på.

## Hjælp med at oversætte Gramps Web til dit sprog

Gramps & Gramps Web understøtter flere dusin forskellige sprog. At hjælpe med at oversætte appen kræver ikke nogen programmeringsfærdigheder. Vi bruger det kraftfulde Weblate online værktøj til oversættelser. Se venligst [her](../development/frontend/translation.md) for instruktioner om, hvordan du kommer i gang.


## Hjælp med at besvare andre brugeres spørgsmål

Hvis du allerede bruger Gramps Web, kan du hjælpe nye brugere med at finde svar på deres spørgsmål ved at bidrage til diskussioner i [Gramps Web-kategorien](https://gramps.discourse.group/c/gramps-web/) på det officielle Gramps Discourse forum.

## Hjælp med at forbedre dokumentationen

Dokumentationssiden, du ser på, er hostet [på Github](https://github.com/gramps-project/gramps-web-docs). Bidrag til at forbedre den er meget velkomne. Indsend blot en pull request til det repository.

## Bidrag til backend-udvikling (Python)

Hvis du er Python-programmør, kan du bidrage til udviklingen af [Gramps Web API](https://github.com/gramps-project/gramps-web-api/), som er backend'en, der driver Gramps Web. Se [Backend-udvikling](../development/backend/index.md) for instruktioner om, hvordan du kommer i gang.


## Bidrag til frontend-udvikling (Javascript)

Hvis du er webprogrammerer, kan du bidrage til udviklingen af [Gramps Web's responsive frontend](https://github.com/gramps-project/gramps-web/). Se [Frontend-udvikling](../development/frontend/index.md) for instruktioner om, hvordan du kommer i gang.

## Bidrag til Sync-udvidelsen (Python)

Hvis du har erfaring med udvikling med Python og GTK, kan du bidrage til forbedringen af The Gramps Web Sync Addon. Du kan bruge [udviklingsrepositoryet](https://github.com/DavidMStraub/gramps-web-sync/) til at foreslå og indsende forbedringer.
