# Gramps 5.2 opgraderingsvejledning

**Gramps 5.2** blev udgivet den 23. februar 2024. Da det bruger et nyt database-schema for familietræet sammenlignet med Gramps 5.1, er der nødvendige ekstra trin ved opgradering af Gramps Web til brug med Gramps 5.2. Denne side beskriver de nødvendige trin.

## Oversigt over trinene

Dette er en oversigt over de trin, der kræves for at opgradere. 
Det viser også, hvilke trin der udføres af administratoren af Gramps Web-serveren, ejeren af familietræet, og om det kun er nødvendigt for brugere af [sync-tilføjelsen](../administration/sync.md)

Serveradministrator | Træ ejer  | Handling
----|----|----
&nbsp; | ✓ | Tag backup af dit familietræ
&nbsp; | ✓ <sup>1</sup> |  [Synkroniser Gramps Web med Gramps Desktop](../administration/sync.md)
✓ |  | [Opdater Gramps Web](update.md)
&nbsp; | ✓ | Opdater træets databaseschema
&nbsp; | ✓ <sup>1</sup> | Opdater Gramps Desktop
&nbsp; | ✓ <sup>1</sup> | Installer den opdaterede Sync-tilføjelse
&nbsp; | ✓ | Tag backup af dit familietræ
&nbsp; | ✓ <sup>1</sup> | [Synkroniser Gramps Web med Gramps Desktop](../administration/sync.md)

<sup>1</sup> Kun nødvendigt, hvis du synkroniserer Gramps Web med Gramps Desktop.

## Detaljerede trin

Dette afsnit giver flere detaljer om nogle af de individuelle trin ovenfor.

### Tag backup af dit familietræ

I Gramps Web, gå til "Eksport" siden og vælg "Gramps XML (familietræ)" (standardmuligheden). Klik på "Generer" og vent, indtil download af .gramps-filen starter.

Du kan også oprette en backup i Gramps Desktop [på den sædvanlige måde](https://www.gramps-project.org/wiki/index.php/How_to_make_a_backup).

### Opdater træets databaseschema

Når du får adgang til Gramps Web for første gang efter at have opgraderet det, vil du blive præsenteret for en skærm, der viser beskeden:

> Det familietræ, du prøver at indlæse, er i en skemas version, der ikke understøttes af denne version af Gramps Web. Derfor kan du ikke indlæse dette familietræ uden at opgradere dets schema. Denne handling kan ikke fortrydes.

Du kan nu klikke på knappen "Opgrader Familietræ". Det vil vise en statuslinje og bør ikke tage mere end et minut at fuldføre. Bekymr dig ikke, hvis du ved et uheld lukker dit browservindue, vil konverteringen fortsætte i baggrunden.

!!! advarsel
    Hvis du bruger en ikke-standard opsætning *uden* baggrundsopgavekø, kan databaseopgraderingen fejle, og dit familietræ kan blive beskadiget. Du kan tjekke (før du opdaterer Gramps Web), om du bruger en baggrundsopgavekø ved at gå til sektionen "Systeminformation" i indstillingerne og sikre dig, at den har "opgavekø: true".

### Installer den opdaterede Sync-tilføjelse

Du kan bruge [den nye Plugin Manager](https://gramps-project.org/wiki/index.php/Addon:Plugin_Manager), der er inkluderet i Gramps 5.2 - søg blot efter Gramps Web Sync, klik på installer, og genstart Gramps.
