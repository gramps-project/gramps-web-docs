# Gramps 6.0 opgraderingsguide

Gramps 6.0 blev udgivet den 19. marts 2025. Da det bruger et nyt familie træ database skema sammenlignet med Gramps 5.2, er der nødvendige yderligere trin, når du opgraderer Gramps Web til at blive brugt med Gramps 6.0. Denne side beskriver de nødvendige trin.

!!! info
    Sidst opdateret: 4. maj 2025


## Oversigt over trinene


Dette er en oversigt over de trin, der kræves for at opgradere. 
Det viser også, hvilke trin der udføres af administratoren af Gramps Web-serveren, ejeren af familie træet, og om det kun er nødvendigt for brugere af [sync-tilføjelsen](../administration/sync.md)

Serveradministrator | Træ ejer  | Handling
----|----|----
&nbsp; | ✓ | Tag backup af dit familie træ
&nbsp; | ✓ <sup>1</sup> |  [Synkroniser Gramps Web med Gramps Desktop](../administration/sync.md)
✓ |  | [Opdater Gramps Web](update.md)
&nbsp; | ✓ | Opdater træets database skema
&nbsp; | ✓ <sup>1</sup> | Opdater Gramps Desktop
&nbsp; | ✓ <sup>1</sup> | Installer den opdaterede Sync-tilføjelse
&nbsp; | ✓ | Tag backup af dit familie træ
&nbsp; | ✓ <sup>1</sup> | [Synkroniser Gramps Web med Gramps Desktop](../administration/sync.md)


<sup>1</sup> Kun nødvendigt, hvis du synkroniserer Gramps Web med Gramps Desktop.

## Detaljerede trin

Dette afsnit giver flere detaljer om nogle af de individuelle trin ovenfor.

### Tag backup af dit familie træ

I Gramps Web, gå til "Eksport" siden og vælg "Gramps XML (familie træ)" (standardmuligheden). Klik på "Generer" og vent, indtil download af .gramps-filen starter.

Du kan også oprette en backup i Gramps Desktop [på den sædvanlige måde](https://www.gramps-project.org/wiki/index.php/How_to_make_a_backup).

### Opdater træets database skema

Når du tilgår Gramps Web for første gang efter at have opgraderet det, vil du blive præsenteret for en skærm, der viser beskeden: 

> Det familie træ, du forsøger at indlæse, er i en skemaversion, der ikke understøttes af denne version af Gramps Web. Derfor kan du ikke indlæse dette familie træ uden at opgradere dets skema. Denne handling kan ikke fortrydes.

Du kan nu klikke på knappen "Opgrader familie træ". Det vil vise en statuslinje og bør ikke tage mere end et minut at fuldføre. Bekymr dig ikke, hvis du ved et uheld lukker dit browservindue, vil konverteringen fortsætte i baggrunden.

!!! warning
    Hvis du bruger en ikke-standard opsætning *uden* baggrundsopgavekø, kan databaseopgraderingen mislykkes, og dit familie træ kan blive beskadiget. Du kan tjekke (før du opdaterer Gramps Web), om du bruger en baggrundsopgavekø ved at gå til sektionen "Systeminformation" i indstillingerne og sikre dig, at den har "opgavekø: true".

### Installer den opdaterede Sync-tilføjelse

Du kan bruge Plugin Manager i Gramps 6.0 til at installere Gramps Web Sync-tilføjelsen. Søg efter Gramps Web Sync, klik på installer, og genstart Gramps.
