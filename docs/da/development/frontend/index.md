# Gramps Web frontend udvikling

Den [Gramps Web frontend](https://github.com/gramps-project/gramps-web) er skrevet i Javascript og bygget ved hjælp af webkomponenter med [Lit](https://lit.dev/). Dette muliggør indkapsling af funktionalitet i genanvendelige komponenter.

Frontend'en er designet som en progressiv enkelt-sides webapp med en mobil-først oplevelse. Den gør intensiv brug af [Material Web Components](https://github.com/material-components/material-web) for et Material Design udseende og følelse.
