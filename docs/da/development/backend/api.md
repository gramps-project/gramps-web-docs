# API-specifikation

Den interaktive API-specifikation (Swagger UI) serveres direkte af din Gramps Web-instans på `/api/swagger-ui` – for eksempel `https://mygrampsweb.example.com/api/swagger-ui`.

Du kan også få adgang til den fra [Brugerindstillinger](../../user-guide/settings.md) via knappen **Start Swagger**, som åbner Swagger UI med din sessionstoken forudfyldt.
