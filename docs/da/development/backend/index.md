# Gramps Web backend udvikling

Backend'en af Gramps Web kaldes [Gramps Web API](https://github.com/gramps-project/gramps-web-api/). Faktisk er det mere end bare en app backend: det leverer et RESTful API til at læse og ændre en Gramps-database.

Gramps Web API er en Python-applikation og drives af `gramps`-biblioteket selv (som også driver Gramps Desktop-applikationen).

I denne sektion af dokumentationen kan du lære, hvordan du bidrager til udviklingen af Gramps Web API.
