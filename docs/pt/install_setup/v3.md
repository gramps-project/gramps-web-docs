# Guia de atualização do Gramps 6.0

O Gramps 6.0 foi lançado em 19 de março de 2025. Como ele utiliza um novo esquema de banco de dados de árvore genealógica em comparação com o Gramps 5.2, etapas adicionais são necessárias ao atualizar o Gramps Web para ser usado com o Gramps 6.0. Esta página descreve as etapas necessárias.

!!! info
    Última atualização: 4 de maio de 2025


## Visão geral das etapas


Esta é uma visão geral das etapas necessárias para a atualização. 
Ela também mostra quais etapas são executadas pelo administrador do servidor Gramps Web, o proprietário da árvore genealógica e se é necessário apenas para usuários do [addon de sincronização](../administration/sync.md)

Administrador do servidor | Proprietário da árvore  | Ação
----|----|----
&nbsp; | ✓ | Faça backup da sua árvore genealógica
&nbsp; | ✓ <sup>1</sup> |  [Sincronize o Gramps Web com o Gramps Desktop](../administration/sync.md)
✓ |  | [Atualize o Gramps Web](update.md)
&nbsp; | ✓ | Atualize o esquema do banco de dados da árvore
&nbsp; | ✓ <sup>1</sup> | Atualize o Gramps Desktop
&nbsp; | ✓ <sup>1</sup> | Instale o addon de Sincronização atualizado
&nbsp; | ✓ | Faça backup da sua árvore genealógica
&nbsp; | ✓ <sup>1</sup> | [Sincronize o Gramps Web com o Gramps Desktop](../administration/sync.md)


<sup>1</sup> Necessário apenas se você sincronizar o Gramps Web com o Gramps Desktop.

## Etapas detalhadas

Esta seção fornece mais detalhes sobre algumas das etapas individuais acima.

### Faça backup da sua árvore genealógica

No Gramps Web, vá para a página "Exportar" e selecione "Gramps XML (árvore genealógica)" (a opção padrão). Clique em "Gerar" e aguarde até que o download do arquivo .gramps comece.

Você também pode criar um backup no Gramps Desktop [da maneira usual](https://www.gramps-project.org/wiki/index.php/How_to_make_a_backup).

### Atualize o esquema do banco de dados da árvore

Quando você acessar o Gramps Web pela primeira vez após tê-lo atualizado, será apresentada uma tela mostrando a mensagem: 

> A Árvore Genealógica que você está tentando carregar está em uma versão de esquema não suportada por esta versão do Gramps Web. Portanto, você não pode carregar esta Árvore Genealógica sem atualizar seu esquema. Esta ação não pode ser desfeita.

Agora você pode clicar no botão "Atualizar Árvore Genealógica". Ele mostrará uma barra de progresso e não deve levar mais de um minuto para ser concluído. Não se preocupe, se você fechar acidentalmente a janela do seu navegador, a conversão continuará em segundo plano.

!!! warning
    Se você usar uma configuração não padrão *sem* fila de tarefas em segundo plano, a atualização do banco de dados pode falhar e sua árvore genealógica pode ficar corrompida. Você pode verificar (antes de atualizar o Gramps Web) se está usando uma fila de tarefas em segundo plano acessando a seção "Informações do Sistema" nas configurações e certificando-se de que tem "fila de tarefas: true".


### Atualize o Gramps Desktop

Siga as instruções de instalação do Gramps Desktop para o seu sistema operacional.

### Instale o addon de Sincronização atualizado

Você pode usar o Gerenciador de Plugins no Gramps 6.0 para instalar o addon de Sincronização do Gramps Web. Pesquise por Gramps Web Sync, clique em instalar e reinicie o Gramps.
