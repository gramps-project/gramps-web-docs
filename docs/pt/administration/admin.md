# Guia do Administrador do Gramps Web

Se você configurou sua própria instância do Gramps Web ou se inscreveu em uma pré-instalada, o próximo passo é preencher e gerenciar seu banco de dados de árvore genealógica.

O foco desta seção está nas tarefas administrativas realizadas pelos **proprietários da árvore**. Normalmente, há um proprietário da árvore para cada banco de dados de árvore genealógica (embora, como proprietário da árvore, você também possa criar outras contas de proprietário da árvore).

Esta seção trata apenas de ações que podem ser realizadas a partir da interface da web. Para ferramentas de linha de comando ou configuração do servidor, consulte a seção anterior [Configuração](../install_setup/setup.md).
