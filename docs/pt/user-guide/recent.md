# Objetos Recentemente Navegados

**Histórico** na barra lateral abre a página Objetos Recentemente Navegados, que lista os últimos objetos que você abriu no Gramps Web, com os mais recentemente visualizados no topo. Até 20 objetos são lembrados.

Isso é útil para navegar rapidamente de volta a uma pessoa, família ou outro objeto que você estava visualizando anteriormente em sua sessão, sem precisar procurá-lo novamente.

Clique em qualquer entrada para voltar a esse objeto. Use o botão **Limpar Tudo** para redefinir a lista.

O histórico é armazenado no armazenamento local do navegador – é específico para o dispositivo e navegador atuais, e não é compartilhado com outros usuários.
