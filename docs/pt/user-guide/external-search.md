# Pesquisa Externa

Gramps Web permite que você pesquise pessoas em sites de genealogia externos diretamente da página de detalhes de uma pessoa.


## Realizando uma pesquisa externa

Para pesquisar uma pessoa em sites externos:

1. Abra a visualização de detalhes de uma pessoa.
2. Clique no botão de pesquisa externa.
3. Um popup aparecerá com links para vários sites de genealogia, como CompGen, FamilySearch e Ancestry.
4. Clicar em um link abrirá a página de resultados de pesquisa nesse site, pré-preenchida com o nome da pessoa, locais de nascimento/morte e outras informações relevantes.

## Personalizando opções de pesquisa externa

Você pode personalizar quais sites de pesquisa externa são exibidos no popup.

1. No popup de pesquisa externa, clique no botão de engrenagem na parte superior para entrar no modo de edição.
2. Selecione ou desmarque os sites que você deseja mostrar ou ocultar.
3. Salve suas alterações.

## Adicionando um mecanismo de busca personalizado

Você pode adicionar novos sites de pesquisa externa com parâmetros personalizados.

1. No popup de pesquisa externa, clique no botão de engrenagem na parte superior para entrar no modo de edição.
2. Clique no botão "+ Adicionar mecanismo de busca personalizado".
3. Uma seção aparecerá com campos para "Nome" e "URL de Pesquisa". Insira o nome do site e a URL com parâmetros pesquisáveis, como `https://example.com/search?name={{name_given}}_{{name_surname}}&place={{birth_place}}`.
4. Clique no botão "Adicionar" para incluir o novo mecanismo de busca na lista.
5. O novo mecanismo de busca será salvo e poderá ser personalizado posteriormente.
