# Favoritos

Os favoritos permitem que você salve links para objetos frequentemente visitados (pessoas, famílias, lugares, fontes, etc.) para acesso rápido.

## Adicionando e removendo favoritos

Cada página de detalhes de objeto possui um ícone de favorito (contorno de favorito) na barra de ferramentas superior. Clique nele para adicionar o objeto aos favoritos – o ícone se torna um favorito preenchido. Clique novamente para remover o favorito.

## Visualizando favoritos

A página **Favoritos** é acessível a partir da barra lateral e lista todos os seus objetos salvos como favoritos. Clique em qualquer entrada para navegar até a página de detalhes desse objeto.

## Casos de uso

Um caso de uso típico é adicionar aos favoritos a pessoa ou família que você está pesquisando atualmente, para que você possa retornar rapidamente a ela sem precisar procurar. Você também pode adicionar aos favoritos fontes ou lugares-chave que você consulta com frequência.

## Armazenamento

Os favoritos são armazenados no armazenamento local do navegador, limitados à árvore atual. Eles são específicos para o dispositivo e navegador que você está usando e não são sincronizados entre dispositivos ou compartilhados com outros usuários.

!!! note
    Os favoritos do Gramps Web são independentes dos favoritos no Gramps Desktop. Eles não são armazenados no banco de dados do Gramps e não aparecerão no Gramps Desktop, e vice-versa.
