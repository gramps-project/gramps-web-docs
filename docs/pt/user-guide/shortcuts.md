# Usando Atalhos de Teclado

Atalhos de teclado permitem que você navegue rapidamente pelo Gramps Web sem usar o mouse, economizando cliques e tempo.

De qualquer visualização, você pode pular diretamente para ações e páginas comuns com apenas algumas teclas. Por exemplo, pressione **`s`** no seu teclado para abrir a busca, **`g`** + **`c`** para navegar até a Árvore Genealógica, ou **`n`** + **`p`** para criar uma nova pessoa.

Para ver uma lista completa de todos os atalhos de teclado disponíveis, basta pressionar **`?`** a qualquer momento.
