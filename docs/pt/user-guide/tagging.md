Gramps Web permite marcar pessoas em fotografias, semelhante ao Gramps Desktop, vinculando a pessoa e o objeto de mídia. Isso é simplificado pela detecção automática de rostos.

Para marcar uma pessoa em uma foto, siga estes passos:

- Abra a visualização de detalhes do objeto de mídia, por exemplo, clicando em "mostrar detalhes" na visualização em tela cheia.
- Clique no botão azul de editar no canto inferior direito. Se você não ver esse botão, seu usuário não tem permissões de edição.
- Sobreposto à imagem, você verá caixas com borda tracejada correspondendo às regiões faciais detectadas. Para marcar uma pessoa, clique na caixa para destacá-la e clique no botão "pessoa" para selecionar uma pessoa existente.
