# Especificação da API

A especificação interativa da API (Swagger UI) é servida diretamente pela sua instância do Gramps Web em `/api/swagger-ui` – por exemplo `https://mygrampsweb.example.com/api/swagger-ui`.

Você também pode acessá-la a partir das [Configurações do Usuário](../../user-guide/settings.md) através do botão **Iniciar Swagger**, que abre o Swagger UI com seu token de sessão preenchido automaticamente.
