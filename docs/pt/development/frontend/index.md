# Desenvolvimento do frontend do Gramps Web

O [frontend do Gramps Web](https://github.com/gramps-project/gramps-web) é escrito em Javascript e construído usando componentes da web com [Lit](https://lit.dev/). Isso permite encapsular funcionalidades em componentes reutilizáveis.

O frontend é projetado como um aplicativo da web de página única progressiva com uma experiência mobile-first. Ele faz uso intensivo de [Material Web Components](https://github.com/material-components/material-web) para uma aparência e sensação de Material Design.
