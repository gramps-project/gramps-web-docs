# Como contribuir para o Gramps Web

Obrigado por considerar contribuir para o Gramps Web! Dependendo de suas habilidades e interesses, há muitas maneiras de você ajudar.

## Ajude a traduzir o Gramps Web para o seu idioma

Gramps e Gramps Web suportam várias dezenas de idiomas diferentes. Ajudar a traduzir o aplicativo não requer habilidades de programação. Usamos a poderosa ferramenta online Weblate para traduções. Consulte [aqui](../development/frontend/translation.md) para instruções sobre como começar.


## Ajude a responder perguntas de outros usuários

Se você já está usando o Gramps Web, ajude novos usuários a encontrar respostas para suas perguntas contribuindo para discussões na [categoria Gramps Web](https://gramps.discourse.group/c/gramps-web/) do fórum oficial do Gramps.

## Ajude a melhorar a documentação

O site de documentação que você está visualizando está hospedado [no Github](https://github.com/gramps-project/gramps-web-docs). Contribuições para melhorá-lo são muito bem-vindas. Basta enviar um pull request para esse repositório.

## Contribua para o desenvolvimento do backend (Python)

Se você é um programador Python, contribua para o desenvolvimento da [API Gramps Web](https://github.com/gramps-project/gramps-web-api/), que é o backend que alimenta o Gramps Web. Consulte [Desenvolvimento do backend](../development/backend/index.md) para instruções sobre como começar.


## Contribua para o desenvolvimento do frontend (Javascript)

Se você é um programador web, contribua para o desenvolvimento do [frontend responsivo do Gramps Web](https://github.com/gramps-project/gramps-web/). Consulte [Desenvolvimento do frontend](../development/frontend/index.md) para instruções sobre como começar.

## Contribua para o addon Sync (Python)

Se você tem experiência em desenvolvimento com Python e GTK, contribua para a melhoria do Addon Sync do Gramps Web. Você pode usar o [repositório de desenvolvimento](https://github.com/DavidMStraub/gramps-web-sync/) para sugerir e enviar melhorias.
