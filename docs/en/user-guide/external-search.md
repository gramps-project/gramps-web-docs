# External Search

Gramps Web allows you to search for people on external genealogy websites directly from a person's detail page.

## Performing an external search

To search for a person on external websites:

1. Open the detail view of a person.
2. Click the external search button 
3. A popup will appear with links to various genealogy websites, such azs CompGen, FamilySearch, and Ancestry.
4. Clicking on a link will open the search results page on that website, pre-filled with the person's name, birth/death places, etc.

## Customizing external search options

You can customize which external search websites are displayed in the popup.

1. In the external search popup, click the gear button at the top to enter edit mode.
2. Select or deselect the websites you want to show or hide.
3. Save your changes.
