# External Search

Gramps Web allows you to search for people on external genealogy websites directly from a person's detail page.


## Performing an external search

To search for a person on external websites:

1. Open the detail view of a person.
2. Click the external search button.
3. A popup will appear with links to various genealogy websites, such as CompGen, FamilySearch, and Ancestry.
4. Clicking on a link will open the search results page on that website, pre-filled with the person's name, birth/death places, and other relevant information.

## Customizing external search options

You can customize which external search websites are displayed in the popup.

1. In the external search popup, click the gear button at the top to enter edit mode.
2. Select or deselect the websites you want to show or hide.
3. Save your changes.

## Adding a custom search engine

You can add new external search websites with custom parameters.

1. In the external search popup, click the gear button at the top to enter edit mode.
2. Click the "+ Add custom search engine" button.
3. A section will appear with fields for "Name" and "Search URL". Enter the website name and the URL with searchable parameters, such as `https://example.com/search?name={{name_given}}_{{name_surname}}&place={{birth_place}}`.
4. Click the "Add" button to include the new search engine in the list.
5. The new search engine will be saved and can be customized later.
