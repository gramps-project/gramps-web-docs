# Recently Browsed Objects

**History** in the sidebar opens the Recently Browsed Objects page, which lists the last objects you opened in Gramps Web, with the most recently viewed at the top. Up to 20 objects are remembered.

This is useful for quickly navigating back to a person, family, or other object you were looking at earlier in your session, without having to search for it again.

Click any entry to go back to that object. Use the **Clear All** button to reset the list.

The history is stored in the browser's local storage – it is specific to the current device and browser, and is not shared with other users.
