# Timeline

!!! info
    The Timeline view requires a sufficiently recent Gramps Web API version. If your server is older, the **Timeline** item is hidden from the sidebar.

**Timeline** is a sidebar item that shows all events in your family tree laid out chronologically on a horizontal, zoomable timeline.

## Browsing the timeline

- Scroll, drag, or pinch to move through time; use the zoom buttons (or your mouse wheel/pinch gesture) to zoom in and out.
- As you zoom in, event labels appear; click an event dot to see its details (and those of any other events on the same date).

## Filtering

Use the filter field to focus the timeline on a single **person** or **place**:

- **Person** – shows only events linked to that person. A toggle lets you switch between **Person** (events of that individual only), **Ancestors**, and **Descendants** (events of all their ancestors/descendants).
- **Place** – shows only events that took place there.

Clear the filter chip to return to the full timeline.

## Jumping to the timeline

On an event's or place's detail page, the **Show on timeline** button opens the Timeline view scrolled and zoomed to that event's date.
