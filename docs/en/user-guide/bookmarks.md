# Bookmarks

Bookmarks let you save links to frequently visited objects (people, families, places, sources, etc.) for quick access.

## Adding and removing bookmarks

Every object detail page has a bookmark icon (bookmark outline) in the top toolbar. Click it to bookmark the object – the icon becomes a filled bookmark. Click it again to remove the bookmark.

## Viewing bookmarks

The **Bookmarks** page is accessible from the sidebar and lists all your bookmarked objects. Click any entry to navigate to that object's detail page.

## Use cases

A typical use case is to bookmark the person or family you are currently researching so you can quickly return to them without searching. You can also bookmark key sources or places you refer to often.

## Storage

Bookmarks are stored in the browser's local storage, scoped to the current tree. They are specific to the device and browser you are using and are not synced across devices or shared with other users.

!!! note
    Gramps Web bookmarks are independent of the bookmarks in Gramps Desktop. They are not stored in the Gramps database and will not appear in Gramps Desktop, and vice versa.
