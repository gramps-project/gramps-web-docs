# Using Keyboard Shortcuts

Keyboard shortcuts allow you to navigate quickly through Gramps Web without using your mouse, saving you clicks and time. 

From any view, you can jump directly to common actions and pages with just a few keystrokes. For example, press **`s`** on your keyboard to open the search, **`g`** + **`c`** to navigate to the Family Tree, or **`n`** + **`p`** to create a new person. 

To see a complete list of all available keyboard shortcuts, simply press **`?`** at any time.
