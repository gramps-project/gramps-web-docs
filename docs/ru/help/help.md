Если у вас возникли проблемы или вам нужна помощь с Gramps Web, пожалуйста, выберите один из следующих вариантов.

[Форум :material-forum:](https://gramps.discourse.group/c/gramps-web/){ .md-button }
[Проблемы с бэкендом :material-github:](https://github.com/gramps-project/gramps-web-api/issues){ .md-button }
[Проблемы с фронтендом :material-github:](https://github.com/gramps-project/gramps-web/issues){ .md-button }

Смотрите ниже некоторые рекомендации о том, с чего начать.

## Задавание вопросов

Официальный форум Gramps Discourse имеет [отдельную категорию для Gramps Web](https://gramps.discourse.group/c/gramps-web/). Пожалуйста, используйте его, чтобы задать любые вопросы, которые у вас могут возникнуть о Gramps Web, например

- Вопросы по использованию Gramps Web
- Вопросы по настройке Gramps Web
- Устранение неполадок в развертывании Gramps Web
- Идеи по улучшению Gramps Web
- ...

## Сообщение о проблемах

Если вы столкнулись с проблемой, которую вы считаете ошибкой в Gramps Web, пожалуйста, сообщите об этом через GitHub.

Существуют два отдельных репозитория GitHub для кода, используемого в Gramps Web: один для пользовательского интерфейса (“frontend”) и один для серверного кода (“backend”):

- [Проблемы с фронтендом](https://github.com/gramps-project/gramps-web/issues)
- [Проблемы с бэкендом](https://github.com/gramps-project/gramps-web-api/issues)

Если вы не уверены, куда подать проблему, не беспокойтесь и просто выберите один из двух – кураторы смогут перенести проблему, если это будет необходимо.

В любом случае, пожалуйста, всегда включайте следующую информацию в ваш отчет:

- Подробности о вашей конфигурации (например, файл docker-compose с удаленными конфиденциальными значениями или используете ли вы хостинговую версию, такую как Grampshub, или предварительно настроенный образ, такой как DigitalOcean)
- Информация о версии. Чтобы получить ее, перейдите на вкладку "Информация о системе" на странице настроек в Gramps Web и скопируйте/вставьте значения в поле, которое должно выглядеть примерно так:

```
Gramps 5.1.6
Gramps Web API 1.5.1
Gramps.js 24.1.0
locale: en
multi-tree: false
task queue: true
```

## Предложение улучшений

Для общих идей и обсуждений о будущих улучшениях не стесняйтесь открывать обсуждение на [форуме](https://gramps.discourse.group/c/gramps-web/). Вы также можете проверить страницы проблем (см. ссылки выше), чтобы узнать, запланирована ли конкретная функция или над ней уже работают.

Для конкретных улучшений с ограниченным объемом не стесняйтесь напрямую открывать проблему с запросом функции в соответствующем репозитории GitHub для фронтенда или бэкенда.
