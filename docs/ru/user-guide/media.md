Есть два способа добавить новый медиафайл (изображение, аудиофайл, видеофайл или любой другой файл):

## Добавить новый отдельный медиафайл

Чтобы добавить отдельный медиафайл, нажмите на значок + в верхней панели приложения и выберите "Медиаобъект".

Нажмите "выбрать файл", чтобы выбрать файл с вашего компьютера. На мобильном устройстве нажатие этой кнопки даст вам возможность сразу сделать фотографию с камеры вашего устройства.

По желанию,

- введите описание медиафайла в поле "название"
- введите дату
- установите медиафайл как приватный (это сделает его видимым только для пользователей с достаточной авторизацией)

Нажмите "добавить", чтобы загрузить файл и создать медиаобъект.

## Добавить новый медиафайл и связать его с другим объектом

Следующие типы объектов в Gramps могут иметь прикрепленные медиаобъекты: люди, семьи, события, места, источники и цитаты.

В детальном представлении любого объекта нажмите на синюю кнопку редактирования в правом нижнем углу (если вы не видите ее, у вашего пользователя нет прав на редактирование). Нажмите на вкладку "галерея" и нажмите на синюю кнопку +.

Откроется диалоговое окно, которое предложит те же поля, что и в предыдущем разделе. Нажмите "сохранить", чтобы загрузить файл, добавить новый медиаобъект и связать его с просматриваемым объектом.
