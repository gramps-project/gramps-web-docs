# История изменений

Просмотр истории изменений показывает все правки, которые были внесены в семейное дерево.

Список правок отображает изменения, сгруппированные по "транзакциям". Транзакция — это группа из одного или нескольких добавлений, удалений или изменений объектов Gramps. Например, добавление новой семьи с двумя существующими лицами в роли отца и матери генерирует транзакцию с одним добавленным объектом семьи и двумя измененными объектами лиц (поскольку они содержат ссылку на новый объект семьи).

Щелчок по транзакции открывает просмотр деталей транзакции. Он содержит список отдельных добавлений, удалений и обновлений по объектам Gramps.

Выбор отдельного изменения открывает представление сырого JSON-формата объекта Gramps, где добавления и удаления выделены зеленым и красным цветами соответственно.
