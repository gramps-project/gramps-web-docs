# Хронология

!!! info
    Вид Хронологии требует достаточно новой версии Gramps Web API. Если ваш сервер устарел, элемент **Хронология** скрыт из боковой панели.

**Хронология** — это элемент боковой панели, который показывает все события в вашем семейном дереве, расположенные хронологически на горизонтальной, масштабируемой временной шкале.

## Просмотр хронологии

- Прокручивайте, перетаскивайте или сжимайте, чтобы перемещаться во времени; используйте кнопки масштабирования (или колесо мыши/жест сжатия), чтобы увеличивать и уменьшать масштаб.
- При увеличении масштаба появляются метки событий; щелкните на точке события, чтобы увидеть его детали (а также детали любых других событий в ту же дату).

## Фильтрация

Используйте поле фильтрации, чтобы сосредоточить хронологию на одном **человеке** или **месте**:

- **Человек** — показывает только события, связанные с этим человеком. Переключатель позволяет вам переключаться между **Человеком** (только события этого индивидуума), **Предками** и **Потомками** (события всех их предков/потомков).
- **Место** — показывает только события, которые произошли там.

Очистите фильтр, чтобы вернуться к полной хронологии.

## Переход к хронологии

На странице деталей события или места кнопка **Показать на хронологии** открывает вид Хронологии, прокрученный и увеличенный до даты этого события.
