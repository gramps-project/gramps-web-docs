# Недавно Просмотренные Объекты

**История** в боковой панели открывает страницу Недавно Просмотренные Объекты, которая перечисляет последние объекты, которые вы открыли в Gramps Web, с самыми недавно просмотренными вверху. Запоминается до 20 объектов.

Это полезно для быстрого возвращения к человеку, семье или другому объекту, который вы смотрели ранее в вашей сессии, без необходимости снова его искать.

Нажмите на любую запись, чтобы вернуться к этому объекту. Используйте кнопку **Очистить Все**, чтобы сбросить список.

История хранится в локальном хранилище браузера – она специфична для текущего устройства и браузера и не делится с другими пользователями.
