# Использование встроенного управления задачами

Gramps Web содержит встроенный инструмент управления генеалогическими задачами. Он предназначен для того, чтобы исследователи могли планировать и расставлять приоритеты, а также документировать свои задачи. Именно поэтому задачи представлены как источники в базе данных Gramps. После завершения задачи связанный контент может служить источником, документирующим процесс исследования.

## Основы задач

Задачи имеют следующие свойства:

- Статус. Это может быть "Открыто", "В процессе", "Заблокировано" или "Завершено"
- Приоритет. Это может быть "Низкий", "Средний" или "Высокий"
- Теги. Метки являются обычными тегами Gramps для основного источника. (Обратите внимание, что у всех задач дополнительно есть метка `ToDo`, чтобы идентифицировать их как задачи, но эта метка скрыта в списке задач, чтобы избежать загромождения.)
- Заголовок. Отображается в списке задач
- Описание. Поле с богатым текстом, которое можно использовать для описания проблемы, а также для документирования любого достигнутого прогресса
- Медиа. Любые медиафайлы, прикрепленные к задаче

## Создание задачи

Поскольку задачи являются обычными объектами Gramps, их могут редактировать или создавать те же группы пользователей, которые могут редактировать или создавать другие объекты (такие как люди или события).

Чтобы создать задачу, нажмите кнопку + на странице списка задач. Введите хотя бы заголовок. Статус всегда будет "Открыто" при создании.

## Редактирование задачи

Чтобы изменить любые детали задачи, нажмите на нее в списке задач.

Страница деталей задачи не имеет отдельного "режима редактирования", как другие объекты Gramps. Изменения заголовка, статуса и приоритета применяются немедленно. Изменения в описании с богатым текстом требуют нажатия кнопки "сохранить" под ним.

## Массовое изменение свойств задач

Приоритет и статус задач можно изменять массово, используя флажки в списке задач для выбора и соответствующие кнопки над списком задач.

## Задачи в Gramps Desktop

При добавлении задач через Gramps Web как источники, так и заметки будут иметь прикрепленную метку `ToDo`, поэтому задачи появятся в настольном [Грамплете Заметок To Do](https://gramps-project.org/wiki/index.php/Addon:ToDoNotesGramplet), а также в [Отчете To Do](https://gramps-project.org/wiki/index.php/Addon:ToDoReport).

Чтобы добавить или отредактировать задачу в Gramps Desktop, используйте следующие рекомендации:

- Добавьте источник с меткой `ToDo` и заголовком задачи в качестве заголовка
- При желании добавьте заметку к источнику с меткой `ToDo`, типом "To Do" и описанием в качестве текста
- Добавьте атрибут "Статус" и установите его на "Открыто", "В процессе", "Заблокировано" или "Завершено"
- Добавьте атрибут "Приоритет" и установите его на 9 для низкого, 5 для среднего или 1 для высокого (эти неинтуитивные значения взяты из спецификации iCalendar)
