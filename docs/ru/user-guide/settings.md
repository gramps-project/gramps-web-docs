# Настройки пользователя

Настройки пользователя доступны через значок пользователя в верхней панели приложения, затем **Настройки пользователя**. Страница организована в сворачиваемые секции. Изменения вступают в силу немедленно, если не указано иное.

!!! note
    Изменения в настройках пользователя касаются только вашей учетной записи. Настройки, которые влияют на всех пользователей дерева, управляются в [Настройках администрирования](../administration/settings.md).

## Учетная запись

Содержит информацию о вашем профиле, учетные данные и безопасность учетной записи.

### Информация о пользователе

Показывает ваше **имя пользователя** и текущую **роль пользователя** (например, Гость, Участник, Редактор). Эти данные доступны только для чтения.

### Изменить электронную почту

Введите новый адрес электронной почты и нажмите **Отправить**, чтобы обновить адрес, связанный с вашей учетной записью. Адрес электронной почты используется для сброса пароля и (если настроено) уведомлений.

### Изменить пароль

Введите ваш текущий пароль и новый пароль, затем нажмите **Отправить**. Если вы забыли свой текущий пароль, используйте ссылку **Забыли пароль** на странице входа.

## Внешний вид

Управляет предпочтениями отображения, сохраненными на вашем устройстве.

### Язык

Выберите язык для интерфейса Gramps Web. Настройка языка хранится в локальном хранилище браузера и применяется только к текущему устройству.

### Тема

Выберите между:

- **Системная** – следует предпочтениям светлой/темной темы операционной системы (по умолчанию)
- **Светлая** – всегда использовать светлую тему
- **Темная** – всегда использовать темную тему

Настройка темы хранится в локальном хранилище браузера.

### Предпочтения семейного дерева

#### Вид семейного дерева по умолчанию

Устанавливает, какой тип диаграммы открывается по умолчанию, когда вы переходите на страницу [Семейное дерево](tree.md). Доступные варианты: Древо предков, Древо потомков, Часовая диаграмма, Диаграмма отношений и Фан-диаграмма.

Это предпочтение хранится в локальном хранилище браузера.

## Инструменты разработчика

### API токен

Копирует ваш текущий токен сессии в буфер обмена. Токен можно использовать для аутентификации напрямую через REST API, например, в интерактивном Swagger UI, предоставляемом вашим экземпляром Gramps Web по адресу `/api/swagger-ui`.

Нажмите **Запустить Swagger**, чтобы открыть Swagger UI в новой вкладке с вашей сессией, уже доступной.

!!! note
    Токен сессии имеет короткий срок действия. Скопируйте его немедленно перед использованием в Swagger, так как он может истечь.
