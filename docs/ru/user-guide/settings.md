# Настройки пользователя

Настройки пользователя доступны через значок пользователя в верхней панели приложения, затем **Настройки пользователя**. Изменения вступают в силу немедленно, если не указано иное.

## Информация о пользователе

Показывает ваше **имя пользователя** и текущую **роль пользователя** (например, Гость, Участник, Редактор). Эти данные доступны только для чтения.

## Язык

Выберите язык для интерфейса Gramps Web. Настройка языка сохраняется в локальном хранилище браузера и применяется только к текущему устройству.

## Тема

Выберите между:

- **Системная** – следует предпочтению светлой/темной темы операционной системы (по умолчанию)
- **Светлая** – всегда использовать светлую тему
- **Темная** – всегда использовать темную тему

Настройка темы сохраняется в локальном хранилище браузера.

## Изменить адрес электронной почты

Введите новый адрес электронной почты и нажмите **Отправить**, чтобы обновить адрес, связанный с вашей учетной записью. Адрес электронной почты используется для сброса пароля и (если настроено) уведомлений.

## Изменить пароль

Введите ваш текущий пароль и новый пароль, затем нажмите **Отправить**. Если вы забыли свой текущий пароль, используйте ссылку **Забыли пароль** на странице входа.

## Предпочтения семейного дерева

### Вид семейного дерева по умолчанию

Устанавливает, какой тип диаграммы открывается по умолчанию, когда вы переходите на страницу [Семейное дерево](tree.md). Опции: Древо предков, Древо потомков, Часовая диаграмма, Диаграмма отношений и Фан-диаграмма.

Это предпочтение сохраняется в локальном хранилище браузера.

## Инструменты разработчика

### Токен API

Копирует ваш текущий токен сессии в буфер обмена. Токен может быть использован для аутентификации напрямую через REST API, например, в интерактивном интерфейсе Swagger UI, предоставляемом вашим экземпляром Gramps Web по адресу `/api/swagger-ui`.

Нажмите **Запустить Swagger**, чтобы открыть интерфейс Swagger UI в новой вкладке с вашей сессией, уже доступной.

!!! note
    Токен сессии имеет короткий срок действия. Скопируйте его немедленно перед использованием в Swagger, так как он может истечь.
