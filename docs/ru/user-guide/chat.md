# Использование AI Ассистента

!!! info
    AI Ассистент требует Gramps Web API версии 2.5.0 или выше и Gramps Web версии 24.10.0 или выше. Версия Gramps Web API 3.6.0 ввела возможности вызова инструментов для более интеллектуальных взаимодействий.

**Ассистент** в Gramps Web (если доступен в вашей установке, обозначен как "Чат" в более старых версиях) предоставляет доступ к AI ассистенту, который может отвечать на вопросы о вашем семейном дереве.

!!! warning
    Поскольку это все еще новая и развивающаяся функция, некоторые типы вопросов работают хорошо, в то время как другие — нет. Также, как и с любым AI ассистентом, он может давать фактически неверные ответы, поэтому всегда проверяйте информацию.

## Как это работает

Чтобы понять, на какие типы вопросов ассистент может ответить, полезно понять, как он работает под капотом:

1. Пользователь задает вопрос.
2. AI ассистент может использовать несколько подходов для поиска ответов:
   - **Семантический поиск**: Gramps Web идентифицирует объекты в вашем семейном дереве, которые с наибольшей вероятностью содержат релевантную информацию. Например, если вы спросите "Как зовут детей Джона Доу?", семьи, в которых Джон Доу является отцом, будут среди лучших результатов.
   - **Вызов инструментов (Gramps Web API v3.6.0+)**: Ассистент может напрямую запрашивать вашу базу данных, используя специализированные инструменты для поиска, фильтрации людей/событий/семей/мест по конкретным критериям, вычисления отношений между людьми и получения подробной информации.
3. Gramps Web передает вопрос вместе с полученной информацией большой языковой модели для формулирования ответа.
4. Ответ отображается вам.

Пока ассистент работает, индикаторы показывают, какие инструменты он в данный момент использует (например, поиск людей, поиск отношений), так что вы можете следить за тем, как он формирует свой ответ. Долгосрочные вопросы обрабатываются как фоновые задачи — вы можете перейти на другую страницу и вернуться, а прогресс также отображается в [Уведомлениях](notifications.md). Ответы форматируются с помощью Markdown (списки, выделение, ссылки и т.д.) для удобства чтения.

## Что вы можете спросить

С возможностями вызова инструментов, введенными в Gramps Web API версии 3.6.0, AI ассистент теперь может обрабатывать более сложные вопросы:

- **Семейные отношения**: "Кто бабушки и дедушки Джейн Смит?" или "Как Джон Доу связан с Мэри Джонсон?"
- **Фильтрованные поиски**: "Покажите мне всех людей, родившихся в Лондоне после 1850 года" или "Какие события произошли в Париже?"
- **Запросы по датам**: "Кто умер до 1900 года?" или "Список браков, которые состоялись между 1920 и 1950 годами"
- **Информация о местах**: "Какие места находятся во Франции?" или "Расскажите мне о церкви Святой Марии"
- **Общие вопросы**: "Как зовут детей Джона Доу?" или "Когда родилась Мэри Смит?"

## Советы по задаванию вопросов

Чтобы получить наилучшие результаты от AI ассистента:

- **Будьте конкретными**: Формулируйте ваш вопрос с максимальной детализацией, чтобы избежать неоднозначностей. Например, "Когда Джон Смит, родившийся в 1850 году в Бостоне, женился?" лучше, чем "Когда Джон Смит женился?"
- **Используйте правильные имена**: Упоминайте конкретные имена, места и даты, когда это уместно.
- **Спрашивайте об одном**: Разбивайте сложные вопросы на более простые части для получения лучших результатов.
- **Используйте свой язык**: Большие языковые модели многоязычны, поэтому вы можете задавать вопросы на своем языке и получать ответы на том же языке.

!!! tip
    Пожалуйста, поделитесь своим опытом о том, что работает, а что нет, с сообществом.
