# Использование AI чата

!!! info
    AI чат требует Gramps Web API версии 2.5.0 или выше и Gramps Web версии 24.10.0 или выше. В Gramps Web API версии 3.6.0 были введены возможности вызова инструментов для более интеллектуальных взаимодействий.

Чат в Gramps Web (если доступен в вашей установке) предоставляет доступ к AI помощнику, который может отвечать на вопросы о вашем семейном дереве.

!!! warning
    Поскольку это все еще новая и развивающаяся функция, некоторые типы вопросов работают хорошо, в то время как другие — нет. Также, как и любой AI помощник, он может давать фактически неверные ответы, поэтому всегда проверяйте информацию.

## Как это работает

Чтобы понять, на какие типы вопросов помощник может ответить, полезно понять, как он работает под капотом:

1. Пользователь задает вопрос.
2. AI помощник может использовать несколько подходов для поиска ответов:
   - **Семантический поиск**: Gramps Web определяет объекты в вашем семейном дереве, которые, вероятно, содержат соответствующую информацию. Например, если вы спросите "Как зовут детей Джона Доу?", семьи, в которых Джон Доу является отцом, будут среди первых результатов.
   - **Вызов инструментов (Gramps Web API v3.6.0+)**: Помощник может напрямую запрашивать вашу базу данных, используя специализированные инструменты для поиска, фильтрации людей/событий/семей/мест по конкретным критериям, вычисления отношений между людьми и получения подробной информации.
3. Gramps Web передает вопрос вместе с полученной информацией большой языковой модели для формулирования ответа.
4. Ответ отображается вам.

## Что вы можете спросить

С возможностями вызова инструментов, введенными в Gramps Web API версии 3.6.0, AI помощник теперь может обрабатывать более сложные вопросы:

- **Семейные отношения**: "Кто бабушки и дедушки Джейн Смит?" или "Как Джон Доу связан с Мэри Джонсон?"
- **Фильтрованные поиски**: "Покажите мне всех людей, родившихся в Лондоне после 1850 года" или "Какие события произошли в Париже?"
- **Запросы по датам**: "Кто умер до 1900 года?" или "Перечислите браки, которые состоялись между 1920 и 1950 годами"
- **Информация о местах**: "Какие места находятся во Франции?" или "Расскажите мне о церкви Святой Марии"
- **Общие вопросы**: "Как зовут детей Джона Доу?" или "Когда родилась Мэри Смит?"

## Советы по задаванию вопросов

Чтобы получить наилучшие результаты от AI помощника:

- **Будьте конкретными**: Формулируйте ваш вопрос с максимальной детализацией, чтобы избежать неоднозначностей. Например, "Когда Джон Смит, родившийся в 1850 году в Бостоне, женился?" лучше, чем "Когда Джон Смит женился?"
- **Используйте собственные имена**: Упоминайте конкретные имена, места и даты, когда это уместно.
- **Спрашивайте по одному**: Разделяйте сложные вопросы на более простые части для получения лучших результатов.
- **Используйте свой язык**: Большие языковые модели многоязычны, поэтому вы можете задавать вопросы на своем языке и получать ответы на том же языке.

!!! tip
    Пожалуйста, поделитесь своим опытом о том, что работает, а что нет, с сообществом.
