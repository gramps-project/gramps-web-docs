# Хромосомный браузер

Хромосомный браузер — это инструмент для визуализации ДНК-совпадений человека. Он показывает сегменты ДНК, общие с другими людьми в дереве, в виде цветных полос на хромосомах.

Вы можете найти его на вкладке &ldquo;Хромосомный браузер&rdquo; в представлении ДНК.

Для каждой хромосомы верхняя (светло-голубая) полоса представляет собой копию, унаследованную от отца, нижняя (светло-красная) полоса — ту, что унаследована от матери. Общие сегменты отображаются в виде цветных полос поверх этих. Если сторона (папина или мамина) неизвестна, сегмент охватывает обе полосы и отображается с штриховкой.

Щелкнув по имени совпадения в легенде, отображаются только общие сегменты этого человека.
