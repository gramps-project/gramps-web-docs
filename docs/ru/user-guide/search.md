# Поиск

Страница поиска доступна по нажатию на значок увеличительного стекла в верхней панели приложения или по нажатию клавиши `s` [горячей клавиши](shortcuts.md).

## Полнотекстовый поиск

Введите любой запрос в поле поиска и нажмите Enter (или нажмите кнопку поиска). Gramps Web ищет по всем типам объектов – людям, семьям, событиям, местам, источникам, цитатам, репозиториям, заметкам и медиа – и возвращает результаты, отсортированные по релевантности.

Результаты показывают тип объекта, имя и краткое описание. Нажмите на любой результат, чтобы открыть соответствующую страницу с деталями.

Знак `*` может использоваться как подстановочный символ, например, `Mey*` соответствует "Meyer", "Meyers", "Meyerhofer" и т.д.

## Фильтрация по типу объекта

Под полем поиска кнопки-переключатели для каждого типа объекта (Люди, Семьи, События, Места и т.д.) позволяют вам сузить результаты до одного или нескольких конкретных типов. По умолчанию ищутся все типы. Активируя один или несколько переключателей, вы ограничиваете результаты только этими типами.

## Семантический поиск

Если администратор сервера включил [семантический (на базе ИИ) поиск](../install_setup/configuration.md), в верхнем правом углу страницы поиска появляется переключатель режима с двумя опциями:

- **Поиск** – стандартный полнотекстовый поиск (по умолчанию)
- **Семантический** – поиск на основе ИИ, который понимает значение вашего запроса, а не просто совпадение точных слов

Семантический поиск полезен для запросов на естественном языке, таких как "фермер в Баварии в 19 веке". Он требует заполнения семантического индекса поиска; смотрите [Настройки администрирования](../administration/settings.md) для получения информации о том, как создать или обновить индекс.
