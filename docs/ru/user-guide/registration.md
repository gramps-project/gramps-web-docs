В установке с одной деревом любой может зарегистрировать новую учетную запись в экземпляре Gramps Web, но она должна быть вручную активирована владельцем сайта, прежде чем ее можно будет использовать для входа.

В установке с несколькими деревьями пользователям дополнительно требуется ID дерева для регистрации; на практике это означает, что они могут зарегистрироваться, используя ссылку на регистрацию, предоставленную владельцем дерева.

## 1. Найдите форму регистрации

На форме входа на главной странице Gramps Web обычно есть ссылка "Зарегистрировать новую учетную запись".

Если этой ссылки нет, вероятно, это установка Gramps Web с несколькими деревьями. В этом случае свяжитесь с владельцем дерева для получения ссылки на регистрацию.

Владелец дерева может найти и скопировать ссылку на регистрацию в разделе Настройки/Администрирование.

## 2. Отправьте форму регистрации

Форма регистрации имеет четыре обязательных поля: имя пользователя, пароль, адрес электронной почты и полное имя. Когда все четыре поля будут заполнены, нажмите "отправить", чтобы начать процесс регистрации.

## 3. Подтвердите адрес электронной почты

После отправки формы на указанный адрес электронной почты будет отправлено автоматическое письмо с ссылкой для подтверждения электронной почты. Нажатие на нее откроет страницу подтверждения.

Если подтверждающее письмо не приходит (и вы проверили папку "Спам"), возможно, что электронная почта не была настроена должным образом администратором сервера Gramps Web.

## 4. Активируйте учетную запись

После подтверждения адреса электронной почты владелец дерева получит уведомление по электронной почте и сможет активировать новую учетную запись пользователя, перейдя в раздел "Администрирование пользователей" на странице настроек (доступно через значок пользователя в верхней панели приложения) и изменив роль пользователя с "отключен" на любую из [других ролей](../install_setup/users.md).

Обратите внимание, что новый пользователь *не* получит автоматическое уведомление о том, что его учетная запись была активирована, поэтому администратору рекомендуется уведомить их лично.
