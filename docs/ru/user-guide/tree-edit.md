# Редактирование семейного дерева

Вид дерева предков предлагает быстрый способ построить и расширить структуру вашего семейного дерева прямо из диаграммы, не покидая вид дерева.

!!! note
    Редактирование дерева доступно в вкладках Дерево предков, Дерево потомков, Часовая диаграмма и Диаграмма отношений. Оно недоступно в Фан-диаграмме. Только пользователи с ролью Редактора или выше могут использовать эту функцию.

## Вход и выход из режима редактирования

Пользователи с правами редактирования видят плавающую кнопку с изображением карандаша (внизу справа) на поддерживаемых страницах диаграмм. Нажмите на нее, чтобы войти в режим редактирования – или нажмите `e` [горячую клавишу](shortcuts.md). Панель приложений покажет индикатор режима редактирования.

Каждое изменение (добавление или связывание человека) сохраняется в базе данных немедленно после подтверждения в диалоговом окне. Чтобы выйти из режима редактирования, нажмите кнопку закрытия на панели приложений или переключитесь на другую вкладку диаграммы.

!!! note
    Переключение вкладок диаграммы автоматически выходит из режима редактирования.

## Добавление члена семьи

В режиме редактирования каждая карточка человека в диаграмме показывает небольшую кнопку **+**. Нажатие на нее открывает диалоговое окно **Добавить члена семьи** для этого человека, с одной или двумя иконками для каждого доступного типа отношений:

- **Отец** – показывается только если у человека еще нет отца в его основной родительской семье
- **Мать** – показывается только если у человека еще нет матери в его основной родительской семье
- **Ребенок**
- **Супруг(а)**

Каждое отношение предлагает две опции рядом:

- **Иконка связи** – соединить с существующим человеком, уже находящимся в базе данных
- **Иконка +** – создать совершенно нового человека

## Связывание с существующим человеком

Выберите иконку связи и найдите человека по имени. Выберите его из результатов, чтобы создать семейную связь.

Для **ребенка** вы также можете установить тип отношения на отец и мать (Рождение, Усыновленный, Пасынок и т.д.).

Для **супруга(и)** вы можете установить тип семейных отношений (Женат, Гражданский брак, Неизвестно и т.д.).

## Создание нового человека

Выберите иконку +, чтобы открыть минимальную форму создания. Доступные поля:

- **Имя** (имя и фамилия)
- **Пол** (Неизвестно, Мужской, Женский, Другой)
- **Дата и место рождения**
- **Дата и место смерти**

Новый человек сохраняется и немедленно связывается с выбранным человеком в диаграмме.

## Ограничения

Вид редактирования дерева предназначен для быстрого построения скелета семейного дерева. Используйте страницу деталей человека для всего, что выходит за рамки основ: цитаты, заметки, медиа, атрибуты, альтернативные имена, детали существующих событий, многопарентские семьи (только основная семья доступна здесь) и управление несколькими супругами.

Используйте **Детали человека** (иконка карточки аккаунта на панели инструментов), чтобы перейти напрямую к полному виду редактирования текущего централизованного человека.
