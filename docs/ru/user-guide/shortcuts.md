# Использование сочетаний клавиш

Сочетания клавиш позволяют вам быстро перемещаться по Gramps Web без использования мыши, экономя клики и время.

Из любого представления вы можете перейти напрямую к общим действиям и страницам всего за несколько нажатий клавиш. Например, нажмите **`s`** на клавиатуре, чтобы открыть поиск, **`g`** + **`c`**, чтобы перейти к Древу семьи, или **`n`** + **`p`**, чтобы создать нового человека.

Чтобы увидеть полный список всех доступных сочетаний клавиш, просто нажмите **`?`** в любое время.
