# Теги

Теги — это метки, которые можно применять к любому объекту в базе данных Gramps — людям, семьям, событиям, местам, источникам, цитатам, репозиториям, заметкам и медиа. Они полезны для группировки и фильтрации объектов. Теги хранятся в базе данных семейного дерева Gramps и доступны всем пользователям; они также полностью совместимы с тегами, созданными в Gramps Desktop.

## Управление тегами

Теги управляются из раздела **Теги** в [Настройках администрирования](../administration/settings.md#tags), который доступен только пользователям с ролью Владельца или Администратора. Он показывает все существующие теги и позволяет вам:

- **Создать** новый тег с помощью кнопки **Новый тег**
- **Переименовать** тег с помощью значка редактирования (карандаш)
- **Изменить цвет** тега с помощью выбора цвета
- **Удалить** тег с помощью значка удаления

!!! note
    Удаление тега удаляет его из всех объектов, к которым он был применен.

## Применение тегов к объектам

Теги можно применять к объекту или удалять их с него на странице деталей объекта в режиме редактирования.

## Фильтрация по тегам

Все страницы списка объектов (Люди, Семьи, События, Места, Источники, Цитаты, Репозитории, Заметки, Медиа) включают фильтр по тегам. Используйте его, чтобы показать только объекты, к которым применен конкретный тег.

## Специальные теги

Два тега имеют особое значение в Gramps Web:

- **`Blog`** — любой источник, помеченный тегом `Blog`, рассматривается как запись в блоге и отображается в представлении [Блог](blog.md)
- **`ToDo`** — любая заметка, помеченная тегом `ToDo`, рассматривается как исследовательская задача и отображается в представлении [Задачи](tasks.md)

Эти теги создаются автоматически при первом использовании функций Блога или Задач. Переименование или удаление их нарушит соответствующую функцию.
