# Руководство пользователя

Этот раздел документирует функции, доступные пользователям Gramps Web.

!!! note "Не видите все функции?"
    Gramps Web использует систему разрешений на основе ролей. Некоторые функции – такие как редактирование данных, управление тегами или просмотр частных записей – доступны только пользователям с достаточными правами. Вы можете проверить свою текущую роль в [Настройках пользователя](settings.md). Если вам нужен больший доступ, свяжитесь с владельцем вашего дерева или администратором. См. [Система пользователей](../install_setup/users.md) для описания всех ролей.

## Навигация по интерфейсу

### Основная навигация

Боковая панель (или меню гамбургера на мобильных устройствах) является основным способом перемещения между разделами:

- **Главная** – панель управления (см. ниже)
- **Блог** – истории семейной истории, написанные в виде блог-постов
- **Семейное дерево** – интерактивные графики деревьев
- **Хронология** – хронологический обзор событий в дереве (требуется достаточно новая версия API Gramps Web)
- **Карта** – географический обзор мест в дереве
- **ДНК** – инструменты анализа совпадений ДНК
- **Списки** – просмотр всех объектов каждого типа: Люди, Семьи, События, Места, Источники, Цитаты, Репозитории, Заметки
- **Медиа** – просмотр всех медиафайлов (фото, документы и т. д.)
- **Ассистент** – AI-чат-ассистент (если включен администратором)
- **История** – недавно измененные объекты
- **Закладки** – ваши сохраненные закладки
- **Задачи** – исследовательские задачи
- **Отчеты** – создание отчетов
- **Экспорт** – экспорт семейного дерева
- **Версии** – полная история транзакций (видна для участников и выше)
- **Уведомления** – прошлые уведомления

!!! note
    Теги больше не управляются из боковой панели – управление тегами переместилось в [Настройки администрирования](../administration/settings.md#tags) (только для владельцев/администраторов). См. [Теги](tags.md) для того, как используются теги.

### Верхняя панель приложения

Панель в верхней части каждой страницы содержит:

- **Добавить** (значок плюса, виден для участников и выше) – открывает меню для создания нового объекта: Человек, Семья, Событие, Место, Источник, Цитата, Репозиторий, Заметка, Медиа-объект или Задача
- **Поиск** (лупа) – открывает страницу поиска
- **Иконка пользователя** – открывает меню настроек: Настройки пользователя, Администрирование (только для владельцев), Управление пользователями (только для владельцев), Информация о системе

## Главная страница (панель управления)

Панель управления отображается при первом входе в систему. Она имеет два столбца:

**Левый столбец:**

- **Карточка домашнего человека** – показывает имя, фото (если доступно) и ключевые факты о вашем выбранном домашнем человеке, с ссылкой на их полный профиль и быстрой навигацией к семейному дереву. Нажмите кнопку **Установить домашнего человека** на карточке, чтобы найти и выбрать другого человека.
- **Юбилеи** – предстоящие дни рождения и юбилеи из дерева, основанные на сегодняшней дате.
- **Недавно измененные** – короткий список недавно измененных объектов, полезный для отслеживания совместных правок.

**Правый столбец:**

- **Недавние блог-посты** – последние записи из [блога](blog.md), если таковые имеются.
- **Статистика** – сводка по количеству объектов в дереве (количество людей, семей, событий и т. д.).

Если администратор дерева настроил **заметку на главной странице** и/или **изображение на главной странице**, они отображаются prominently над основными столбцами. Изображение появляется рядом с текстом заметки, когда оба установлены. См. [Настройки администрирования](../administration/settings.md#customization) для того, как настроить это.

!!! tip
    Если дерево пустое и у вас есть права на редактирование, панель управления показывает подсказку "Начать" с кнопками для добавления вашего первого человека или импорта файла семейного дерева.
