# Руководство пользователя

Этот раздел документирует функции, доступные пользователям Gramps Web.

## Навигация по интерфейсу

### Основная навигация

Боковая панель (или меню гамбургера на мобильных устройствах) является основным способом перемещения между разделами:

- **Главная** – панель управления (см. ниже)
- **Блог** – истории семейной истории, написанные в виде блог-постов
- **Люди, Семьи, События, Места, Источники, Цитаты, Репозитории, Заметки** – просмотр всех объектов каждого типа
- **Медиа** – просмотр всех медиафайлов (фото, документы и т. д.)
- **Карта** – географический вид мест в дереве
- **Семейное древо** – интерактивные графики дерева
- **ДНК** – инструменты анализа совпадений ДНК
- **Чат** – AI чат-ассистент (если включен администратором)
- **История** – недавно измененные объекты
- **Закладки** – ваши сохраненные закладки
- **Задачи** – исследовательские задачи
- **Экспорт** – экспорт семейного древа
- **Отчеты** – генерация отчетов
- **Версии** – полная история транзакций (видна участникам и выше)
- **Теги** – управление тегами (видно редакторам и выше)
- **Уведомления** – прошлые уведомления

### Верхняя панель приложения

Панель в верхней части каждой страницы содержит:

- **Добавить** (иконка плюса, видна участникам и выше) – открывает меню для создания нового объекта: Человек, Семья, Событие, Место, Источник, Цитата, Репозиторий, Заметка, Медиа-объект или Задача
- **Поиск** (лупа) – открывает страницу поиска
- **Иконка пользователя** – открывает меню настроек: Настройки пользователя, Администрирование (только для владельцев), Управление пользователями (только для владельцев), Информация о системе

## Главная страница (панель управления)

Панель управления отображается при первом входе в систему. Она имеет два столбца:

**Левый столбец:**

- **Карточка домашнего человека** – показывает имя, фото (если доступно) и ключевые факты о вашем выбранном домашнем человеке, с ссылкой на их полный профиль и быстрой навигацией к семейному дереву. Нажмите кнопку **Установить домашнего человека** на карточке, чтобы найти и выбрать другого человека.
- **Юбилеи** – предстоящие дни рождения и юбилеи из дерева, основанные на сегодняшней дате.
- **Недавно измененные** – короткий список недавно измененных объектов, полезный для отслеживания совместных правок.

**Правый столбец:**

- **Недавние блог-посты** – последние записи из [блога](blog.md), если таковые имеются.
- **Статистика** – сводка по количеству объектов в дереве (количество людей, семей, событий и т. д.).

!!! tip
    Если дерево пустое и у вас есть права на редактирование, панель управления показывает подсказку "Начните" с кнопками для добавления вашего первого человека или импорта файла семейного древа.
