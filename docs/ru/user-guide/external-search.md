# Внешний поиск

Gramps Web позволяет вам искать людей на внешних генеалогических сайтах прямо со страницы деталей человека.


## Выполнение внешнего поиска

Чтобы найти человека на внешних сайтах:

1. Откройте детальный просмотр человека.
2. Нажмите кнопку внешнего поиска.
3. Появится всплывающее окно с ссылками на различные генеалогические сайты, такие как CompGen, FamilySearch и Ancestry.
4. Нажатие на ссылку откроет страницу результатов поиска на этом сайте, предварительно заполненную именем человека, местами рождения/смерти и другой соответствующей информацией.

## Настройка параметров внешнего поиска

Вы можете настроить, какие внешние сайты поиска отображаются во всплывающем окне.

1. В всплывающем окне внешнего поиска нажмите кнопку шестеренки вверху, чтобы войти в режим редактирования.
2. Выберите или снимите выбор с сайтов, которые вы хотите показать или скрыть.
3. Сохраните изменения.

## Добавление пользовательской поисковой системы

Вы можете добавить новые внешние сайты поиска с пользовательскими параметрами.

1. В всплывающем окне внешнего поиска нажмите кнопку шестеренки вверху, чтобы войти в режим редактирования.
2. Нажмите кнопку "+ Добавить пользовательскую поисковую систему".
3. Появится раздел с полями для "Имя" и "URL поиска". Введите название сайта и URL с параметрами для поиска, такими как `https://example.com/search?name={{name_given}}_{{name_surname}}&place={{birth_place}}`.
4. Нажмите кнопку "Добавить", чтобы включить новую поисковую систему в список.
5. Новая поисковая система будет сохранена и может быть настроена позже.
