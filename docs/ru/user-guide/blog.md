# Использование встроенного блога

Блог предназначен для представления историй о вашем исследовании семейной истории.

В базе данных Gramps записи блога представлены как источники с прикрепленной заметкой, содержащей текст блога, и, при желании, медиафайлы для изображений записи блога. Gramps Web рассматривает каждый источник с тегом `Blog` как статью блога.

## Добавить запись в блог

Чтобы добавить запись в блог, вы можете использовать Gramps Web или Gramps Desktop ([синхронизированный](../administration/sync.md) с Gramps Web), шаги одинаковы в обоих случаях:

- Добавьте новый источник. Заголовок источника будет заголовком вашей записи в блоге, автором источника будет автор записи.
- При желании, свяжите источник с репозиторием, соответствующим вашему блогу в Gramps Web.
- Добавьте новую заметку к источнику. Напишите вашу запись в блоге и скопируйте текст в заметку.
- При желании, добавьте один или несколько медиафайлов к вашему источнику. Первый медиафайл будет использоваться в качестве превью записи, отображаемого над текстом. Все медиафайлы будут показаны ниже текста в виде галереи.
- Добавьте метку `Blog` к источнику (создайте её, если она не существует).

## Связь между блогом и источниками

Поскольку записи блога являются просто источниками, все статьи блога также появляются в списке источников и отображаются как источники в поисковых запросах. В представлении источника есть кнопка "показать в блоге", которая перенаправит вас на представление блога для этой записи. URL записи блога также содержит ID Gramps соответствующего источника, поэтому статья по адресу `yourdomain.com/blog/S0123` соответствует источнику по адресу `yourdomain.com/source/S0123`.

Внизу каждой записи блога есть кнопка "подробности", которая перенаправит вас на представление источника.
