# Уведомления

**Уведомления** — это элемент боковой панели с иконкой колокольчика. Когда возникают ошибки, значок показывает количество непрочитанных уведомлений. Нажмите на него, чтобы открыть журнал уведомлений.

Журнал уведомлений — это запись об ошибках, которые произошли во время вашей сессии — неудачные запросы к API, ошибки фоновых задач, сбои сохранения или ошибки на уровне браузера. Каждая запись показывает короткое сообщение, источник (Сеть, Задача, Сохранение или Браузер) и временную метку.

Некоторые уведомления содержат структурированные данные. Нажатие на такую запись открывает диалоговое окно с разбивкой данных об ошибке и кнопкой **Копировать JSON**. Это полезно при сообщении об ошибке, так как JSON содержит точную информацию об ошибке с сервера.

Используйте **Очистить все**, чтобы удалить все уведомления.

!!! note
    Уведомления хранятся только в памяти и очищаются при перезагрузке страницы.
