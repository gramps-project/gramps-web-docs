# Уведомления

**Уведомления** — это элемент боковой панели с иконкой колокольчика. Когда происходят ошибки или выполняются фоновые задачи, значок показывает количество непрочитанных уведомлений. Нажмите на него, чтобы открыть журнал уведомлений.

Журнал уведомлений выполняет две функции:

- Это запись ошибок, которые произошли во время вашей сессии — неудачные API-запросы, ошибки фоновых задач, сбои сохранения или ошибки на уровне браузера.
- Он отслеживает прогресс длительных фоновых задач — таких как импорт и экспорт, генерация отчетов, распознавание текста OCR, обновления базы данных и перестройка поисковых/семантических индексов — показывая их состояние (например, ожидает, начато, в процессе) и уведомляя вас, когда они завершены или завершились неудачей.

Каждая запись показывает короткое сообщение, источник (Сеть, Задача, Сохранение или Браузер) и временную метку.

Некоторые уведомления содержат структурированные детали. Нажатие на такую запись открывает диалоговое окно с разбивкой данных об ошибке и кнопкой **Копировать JSON**. Это полезно при сообщении об ошибке, так как JSON содержит точную информацию об ошибке с сервера.

Используйте **Очистить все**, чтобы удалить все уведомления.

!!! note
    Уведомления хранятся только в памяти и очищаются при перезагрузке страницы.
