# Разработка бэкенда Gramps Web

Бэкенд Gramps Web называется [Gramps Web API](https://github.com/gramps-project/gramps-web-api/). На самом деле это больше, чем просто бэкенд приложения: он предоставляет RESTful API для чтения и изменения базы данных Gramps.

Gramps Web API — это приложение на Python, которое работает на библиотеке `gramps` (которая также поддерживает приложение Gramps Desktop).

В этом разделе документации вы можете узнать, как внести свой вклад в разработку Gramps Web API.
