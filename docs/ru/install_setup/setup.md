# Установка / Настройка Gramps Web

Этот раздел посвящен установке и настройке Gramps Web, а также другим вариантам начала работы.

## Начало работы с Gramps Web

Gramps Web — это веб-приложение, которое работает на сервере и доступно через веб-браузер. Оно предназначено для доступа аутентифицированным пользователям через интернет.

Если вы хотите использовать Gramps Web для своих генеалогических данных, вам нужно выбрать один из следующих вариантов:

1. Самостоятельный хостинг на собственном оборудовании
2. Самостоятельный хостинг в облаке
3. Подписка на хостинг-экземпляр

Хотя первый вариант предоставляет максимальную гибкость и контроль, он также может быть технически сложным.

!!! tip
    Один из основных принципов Gramps Web — предоставить пользователям контроль над своими данными в любое время, поэтому миграция данных с одного экземпляра на другой проста. Не беспокойтесь о том, что вы окажетесь в ловушке после выбора одного из вариантов!

## Самостоятельный хостинг на собственном оборудовании

Самый удобный способ самостоятельно хостить Gramps Web — через Docker Compose. Мы также предоставляем образы Docker для архитектуры ARM, так что вы можете запустить Gramps Web на Raspberry Pi в вашем подвале.

Смотрите [Развертывание с помощью Docker](deployment.md) для инструкций по настройке.

## Самостоятельный хостинг в облаке

Установка Gramps Web может быть более сложной, чем для других простых веб-приложений, и не совместима с обычными провайдерами "общего хостинга". Вы можете подписаться на виртуальный сервер и установить Gramps Web [вручную](deployment.md).

Более простой вариант — использовать предустановленный облачный образ. Одним из примеров является наше [приложение DigitalOcean с одним кликом](digital_ocean.md).

## Подписка на хостинг-экземпляр

Хостинг Gramps Web — это самый простой способ начать работу с Gramps Web, так как установка или настройка не требуется.

Вот список специализированных провайдеров хостинга для Gramps Web (сообщество Gramps с открытым исходным кодом не несет ответственности за их услуги):

- Grampshub ([www.grampshub.com](https://www.grampshub.com)), предлагаемый одним из основных участников Gramps Web

Если вы используете хостинг-опцию для Gramps Web, вы можете пропустить остальную часть этого раздела и перейти сразу к разделу [Администрирование](../administration/admin.md).
