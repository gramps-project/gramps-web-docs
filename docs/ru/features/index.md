# Особенности


![Image title](screenshots/sync.png){ align=left width="300"}

## Полная интеграция с Gramps Desktop

Gramps Web использует ту же **Модель / Структуру базы данных**, что и [Gramps Desktop](https://gramps-project.org/) для хранения генеалогических данных. Вы можете просматривать все те же [Типы записей](https://gramps-project.org/wiki/index.php/Gramps_Data_Model), что и в Gramps Desktop: ***люди, семьи, события, места, репозитории, источники, цитаты, медиаобъекты и заметки.***

С помощью [Дополнения Gramps Web Sync](../administration/sync.md) для Gramps Desktop данные могут синхронизироваться в обоих направлениях между Gramps Web и Gramps Desktop! Не стесняйтесь редактировать ваши данные с помощью Gramps Web или приложения Gramps Desktop, которое вам нравится - они работают вместе без проблем!

## Уровни конфиденциальности / Доступ пользователей

Многие люди хотят сохранить некоторые детали в тайне, и мы это уважаем! Вы можете пометить записи как Частные и контролировать, какие пользователи могут просматривать частные записи. Частные записи фильтруются на уровне базы данных для максимальной безопасности. Кроме того, вы можете контролировать, что пользователи могут добавлять и редактировать.

## Без привязки &ndash; импорт и экспорт данных

Помимо возможности импортировать данные в различных форматах, включая Gramps XML и GEDCOM, Gramps Web упрощает пользователям загрузку всех своих данных (данные семейного дерева, медиафайлы, учетные записи пользователей) в любое время, для резервного копирования или для переноса на другой сервер. Ваши данные принадлежат только вам!

## Интегрированное приложение для управления задачами

Gramps Web поставляется с интегрированным приложением для управления задачами, чтобы организовать и планировать ваше генеалогическое исследование. Задачи хранятся как источники в базе данных Gramps, поэтому они являются частью ваших генеалогических данных и могут быть доступны и редактированы в Gramps Desktop.

## Интеграция OpenHistoricalMap

Исторические векторные карты, созданные проектом [OpenHistoricalMap](https://www.openhistoricalmap.org/), идеально дополняют генеалогическое картографирование. Используйте ползунок времени, чтобы прокрутить эволюцию мест в вашей семейной истории и отобразить места, где жили предки или происходили события.

## Демонстрация

Чтобы войти в демонстрацию, используйте любые из следующих ***ПОЛЬЗОВАТЕЛЬ / ПАРОЛЬ*** учетных данных для входа. Каждая из них представляет тип пользователя, к которому может быть назначен пользователь Gramps Web.

`owner / owner` <br>
`editor / editor` <br>
`contributor / contributor` <br>
`member / member`
