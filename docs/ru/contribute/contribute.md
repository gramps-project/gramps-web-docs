# Как внести вклад в Gramps Web

Спасибо, что рассматриваете возможность внести вклад в Gramps Web! В зависимости от ваших навыков и интересов, есть множество способов, как вы можете помочь.

## Помощь в переводе Gramps Web на ваш язык

Gramps и Gramps Web поддерживают несколько десятков различных языков. Помощь в переводе приложения не требует никаких навыков программирования. Мы используем мощный онлайн-инструмент Weblate для переводов. Пожалуйста, смотрите [здесь](../development/frontend/translation.md) инструкции о том, как начать.

## Помощь в ответах на вопросы других пользователей

Если вы уже используете Gramps Web, помогите новым пользователям найти ответы на их вопросы, участвуя в обсуждениях в [категории Gramps Web](https://gramps.discourse.group/c/gramps-web/) официального форума Gramps Discourse.

## Помощь в улучшении документации

Сайт документации, который вы просматриваете, размещен [на Github](https://github.com/gramps-project/gramps-web-docs). Вклады для его улучшения очень приветствуются. Просто отправьте запрос на извлечение в этот репозиторий.

## Вклад в разработку бэкенда (Python)

Если вы программист на Python, внесите вклад в разработку [Gramps Web API](https://github.com/gramps-project/gramps-web-api/), который является бэкендом для Gramps Web. Смотрите [Разработка бэкенда](../development/backend/index.md) для получения инструкций о том, как начать.

## Вклад в разработку фронтенда (Javascript)

Если вы веб-программист, внесите вклад в разработку [отзывчивого фронтенда Gramps Web](https://github.com/gramps-project/gramps-web/). Смотрите [Разработка фронтенда](../development/frontend/index.md) для получения инструкций о том, как начать.

## Вклад в дополнение Sync (Python)

Если у вас есть опыт разработки на Python и GTK, внесите вклад в улучшение дополнения Gramps Web Sync. Вы можете использовать [репозиторий для разработки](https://github.com/DavidMStraub/gramps-web-sync/) для предложений и отправки улучшений.
