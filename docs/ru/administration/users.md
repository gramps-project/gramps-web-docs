# Управление пользователями

Интерфейс управления пользователями доступен через **Настройки > Управление пользователями** (значок пользователя в верхней панели приложения). Он доступен только пользователям с ролью Владельца или Администратора.

## Роли пользователей

Смотрите [Система пользователей](../install_setup/users.md) для полного описания доступных ролей пользователей и их прав.

## Просмотр и фильтрация пользователей

Страница управления пользователями показывает таблицу всех зарегистрированных учетных записей пользователей с следующими столбцами:

- **Имя пользователя** — имя для входа
- **Полное имя** — отображаемое имя
- **E-mail** — адрес электронной почты пользователя
- **Роль** — назначенная роль (Гость, Участник, Участник, Редактор, Владелец или Администратор)
- **Источник учетной записи** — либо "Пароль" (локальная учетная запись), либо название внешнего поставщика удостоверений (например, при использовании OIDC)

Используйте поле поиска и выпадающий список ролей в верхней части таблицы для фильтрации списка. Нажмите кнопку очистки фильтра, чтобы сбросить все фильтры.

## Редактирование пользователя

Нажмите на значок редактирования (карандаш) в любой строке, чтобы открыть диалог редактирования. Вы можете изменить:

- Полное имя
- Адрес электронной почты
- Роль

Это основной способ **активировать нового саморегистрированного пользователя**: измените его роль с *отключенной* на любую активную роль (например, Участник или Редактор).

## Добавление пользователя вручную

Нажмите на значок **добавить пользователя** (добавить человека) над таблицей, чтобы создать новую учетную запись пользователя напрямую без необходимости саморегистрации. Заполните имя пользователя, полное имя, адрес электронной почты, пароль и роль в диалоговом окне и нажмите **Сохранить**.

## Удаление пользователя

Нажмите на значок удаления (корзина) в любой строке и подтвердите действие в диалоговом окне. Это действие нельзя отменить.

## Экспорт и импорт учетных записей пользователей

Эти кнопки полезны при [миграции на другую инстанцию Gramps Web](export.md).

- **Экспорт деталей пользователей** (значок загрузки) — загружает файл JSON, содержащий все учетные записи пользователей (без паролей, так как пароли хранятся в зашифрованном виде).
- **Импорт учетных записей пользователей** (значок добавления группы) — загружает ранее экспортированный файл JSON для массового создания учетных записей пользователей. Все импортированные пользователи должны будут установить новый пароль через ссылку "Забыли пароль", так как пароли не могут быть переданы.

## Ссылка для регистрации (только для многодеревьев)

В многодеревьевом режиме ссылка для регистрации новых пользователей отображается в верхней части страницы управления пользователями. Вы можете скопировать эту ссылку и поделиться ею с людьми, которых хотите пригласить зарегистрировать учетную запись в вашем дереве.

!!! note
    В однодеревьевом режиме на странице входа есть общая ссылка "Зарегистрироваться"; ссылка для регистрации по деревьям нужна только в многодеревьевых установках.

## Разрешения на чат с ИИ

Если чат с ИИ был включен на сервере, выпадающий список в верхней части страницы позволяет вам контролировать, какие роли пользователей могут использовать функцию чата:

- Все (включая гостей)
- Участник и выше
- Участник и выше
- Редактор и выше
- Только владельцы и администраторы
- Никто (отключить чат для всех пользователей)
