## Создание резервной копии вашего семейного дерева

Чтобы создать резервную копию вашего семейного дерева, откройте страницу Экспорт в Gramps Web и выберите формат Gramps XML.

Нажатие на "экспорт" сгенерирует файл и начнет загрузку, как только он будет готов.

Обратите внимание, что если у вашего пользователя Gramps Web нет разрешения на просмотр частных записей, экспорт не будет полной резервной копией, так как он не будет содержать никаких частных записей.

## Поделитесь своим семейным деревом с пользователями других генеалогических программ

Когда обмен генеалогическими данными в формате Gramps XML невозможен, вы также можете экспортировать файл GEDCOM. Обратите внимание, что это не подходит в качестве резервной копии вашего дерева Gramps Web.

## Создание резервной копии ваших медиафайлов

Чтобы создать резервную копию ваших медиафайлов, вы можете создать и скачать ZIP-архив всех медиафайлов на странице Экспорт.

Обратите внимание, что, особенно для больших деревьев, это может быть затратной операцией для сервера и должно выполняться только в случае крайней необходимости.

Лучший вариант для регулярного создания резервных копий ваших медиафайлов — использовать [дополнение Gramps Web Sync](sync.md) (которое само по себе не является решением для резервного копирования) и создавать инкрементальные резервные копии на вашем локальном компьютере.

В обоих случаях, если у вашего пользователя Gramps Web нет разрешения на просмотр частных записей, экспорт не будет содержать файлы частных медиаобъектов.

## Перемещение на другой экземпляр Gramps Web

Gramps Web не привязывает вас к конкретному провайдеру, и вы всегда можете перейти на другой экземпляр Gramps Web, не теряя никаких данных и не имея прямого доступа к серверам.

Чтобы осуществить полную миграцию, выполните следующие шаги (при условии, что у вас есть права владельца дерева):

1. Перейдите на страницу Экспорт и экспортируйте ваше дерево в файл Gramps XML (`.gramps`). Если вы используете [дополнение Sync](sync.md), вы также можете сгенерировать экспорт в настольной версии Gramps.
2. На странице Экспорт сгенерируйте и скачайте архив медиафайлов. Если вы используете [дополнение Sync](sync.md), вы также можете просто ZIP-овать вашу локальную папку медиафайлов Gramps.
3. Перейдите в Настройки > Администрирование > Управление пользователями и нажмите кнопку "Экспортировать данные пользователя". Это загрузит файл JSON.
4. В новом экземпляре Gramps Web откройте страницу Импорт. Импортируйте файл `.gramps`, экспортированный на шаге 1.
5. На странице Импорт нового экземпляра Gramps Web загрузите архив медиафайлов (ZIP).
6. Перейдите в Настройки > Администрирование > Управление пользователями нового экземпляра Gramps Web. Нажмите кнопку "Импортировать учетные записи пользователей" и загрузите файл JSON, загруженный на шаге 3.

Обратите внимание, что, хотя ваши учетные записи пользователей будут мигрированы, всем вашим пользователям необходимо будет установить новые пароли, используя ссылку "забыли пароль", так как пароли хранятся в зашифрованном виде и не могут быть экспортированы.
