# Создание учетной записи для владельца дерева

Прежде чем вы сможете начать использовать Gramps Web, вам необходимо создать учетную запись для владельца дерева. Если для данного дерева не существует учетной записи пользователя, будет показана форма для создания учетной записи. Форма зависит от настройки сервера, которая может быть для одного дерева или для нескольких деревьев.

## Настройка для одного дерева: создание учетной записи администратора

На сервере с настройкой для одного дерева, когда учетная запись пользователя еще не существует, открытие Gramps Web показывает форму для создания учетной записи администратора. Администратор будет как владельцем (единственного) дерева, так и администратором установки. Форма также позволяет настроить конфигурацию электронной почты, необходимую для уведомлений по электронной почте (например, для сброса пароля пользователя). Если конфигурация электронной почты уже была добавлена через файл конфигурации или переменные окружения на сервере, эту часть формы можно оставить пустой.

## Настройка для нескольких деревьев: создание учетной записи администратора

В настройке для нескольких деревьев та же форма для создания учетной записи администратора будет показана, если пользователей не существует *в любом дереве*, т.е. когда сервер только что был создан.

## Настройка для нескольких деревьев: создание учетной записи владельца дерева

В настройке для нескольких деревьев каждый пользователь привязан к одному дереву. Даже если пользователи уже существуют в других деревьях, учетная запись владельца дерева может быть создана в веб-интерфейсе, если владельца *для этого дерева* еще нет.

Однако форма создания владельца не будет отображаться автоматически на главной странице Gramps Web, которая одинакова для всех деревьев. Вместо этого к ней можно получить доступ по адресу `https://my-gramps-instance/firstrun/my-tree-id`, где `https://my-gramps-instance` — это базовый адрес вашей установки Gramps Web, а `my-tree-id` — это ID вашего дерева.

Возможный рабочий процесс для администратора сайта по созданию нового дерева:

- Создать дерево через REST API, получив ID нового дерева
- Поделиться ссылкой на форму создания владельца с соответствующим ID дерева с предполагаемым владельцем дерева

Форма создания владельца дерева аналогична форме создания администратора, описанной выше, за исключением того, что она не позволяет изменять конфигурацию электронной почты (что разрешено только для администраторов).
