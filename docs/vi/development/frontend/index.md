# Phát triển giao diện web Gramps

Giao diện [Gramps Web](https://github.com/gramps-project/gramps-web) được viết bằng Javascript và được xây dựng bằng cách sử dụng các thành phần web với [Lit](https://lit.dev/). Điều này cho phép đóng gói chức năng thành các thành phần có thể tái sử dụng.

Giao diện được thiết kế như một ứng dụng web một trang tiến bộ với trải nghiệm ưu tiên di động. Nó sử dụng nhiều [Material Web Components](https://github.com/material-components/material-web) để có giao diện và cảm giác theo thiết kế Material.
