# Đặc tả API

Đặc tả API tương tác (Swagger UI) được phục vụ trực tiếp bởi phiên bản Gramps Web của bạn tại `/api/swagger-ui` – ví dụ `https://mygrampsweb.example.com/api/swagger-ui`.

Bạn cũng có thể truy cập nó từ [Cài đặt Người dùng](../../user-guide/settings.md) thông qua nút **Khởi động Swagger**, nút này mở Swagger UI với mã thông báo phiên của bạn được điền sẵn.
