# Harici Arama

Gramps Web, bir kişinin detay sayfasından doğrudan harici soybilim web sitelerinde insanları aramanıza olanak tanır.


## Harici arama yapma

Harici web sitelerinde bir kişiyi aramak için:

1. Bir kişinin detay görünümünü açın.
2. Harici arama butonuna tıklayın.
3. CompGen, FamilySearch ve Ancestry gibi çeşitli soybilim web sitelerine bağlantılar içeren bir açılır pencere görünecektir.
4. Bir bağlantıya tıkladığınızda, o web sitesinde kişinin adı, doğum/ölüm yerleri ve diğer ilgili bilgilerle önceden doldurulmuş arama sonuçları sayfası açılacaktır.

## Harici arama seçeneklerini özelleştirme

Açılır pencerede hangi harici arama web sitelerinin görüntüleneceğini özelleştirebilirsiniz.

1. Harici arama açılır penceresinde, üstteki dişli simgesine tıklayarak düzenleme moduna geçin.
2. Gösterilmesini veya gizlenmesini istediğiniz web sitelerini seçin veya seçimini kaldırın.
3. Değişikliklerinizi kaydedin.

## Özel bir arama motoru ekleme

Özel parametrelerle yeni harici arama web siteleri ekleyebilirsiniz.

1. Harici arama açılır penceresinde, üstteki dişli simgesine tıklayarak düzenleme moduna geçin.
2. "+ Özel arama motoru ekle" butonuna tıklayın.
3. "Ad" ve "Arama URL'si" için alanlar içeren bir bölüm görünecektir. Web sitesi adını ve `https://example.com/search?name={{name_given}}_{{name_surname}}&place={{birth_place}}` gibi arama yapılabilir parametrelerle URL'yi girin.
4. Yeni arama motorunu listeye eklemek için "Ekle" butonuna tıklayın.
5. Yeni arama motoru kaydedilecek ve daha sonra özelleştirilebilir.
