# Gramps Web arka uç geliştirme

Gramps Web'in arka ucu [Gramps Web API](https://github.com/gramps-project/gramps-web-api/) olarak adlandırılmaktadır. Aslında bu, sadece bir uygulama arka ucu olmaktan daha fazlasıdır: Gramps veritabanını okumak ve değiştirmek için bir RESTful API sağlar.

Gramps Web API, bir Python uygulamasıdır ve `gramps` kütüphanesi tarafından desteklenmektedir (bu kütüphane aynı zamanda Gramps Masaüstü uygulamasını da destekler).

Bu belgelerin bu bölümünde, Gramps Web API geliştirmesine nasıl katkıda bulunabileceğinizi öğrenebilirsiniz.
