# API spesifikasyonu

Etkileşimli API spesifikasyonu (Swagger UI), Gramps Web örneğiniz tarafından doğrudan `/api/swagger-ui` adresinde sunulmaktadır – örneğin `https://mygrampsweb.example.com/api/swagger-ui`.

Ayrıca, **Swagger'ı Başlat** butonunu kullanarak [Kullanıcı Ayarları](../../user-guide/settings.md) üzerinden de erişebilirsiniz; bu buton, oturum token'ınız önceden doldurulmuş olarak Swagger UI'yi açar.
