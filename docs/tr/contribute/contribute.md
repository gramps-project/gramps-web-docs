# Gramps Web'e nasıl katkıda bulunabilirsiniz

Gramps Web'e katkıda bulunmayı düşündüğünüz için teşekkürler! Becerilerinize ve ilgi alanlarınıza bağlı olarak, yardımcı olabileceğiniz birçok yol var.

## Gramps Web'i kendi dilinize çevirmeye yardımcı olun

Gramps ve Gramps Web, birkaç düzine farklı dili desteklemektedir. Uygulamayı çevirmek için herhangi bir programlama becerisi gerekmemektedir. Çeviriler için güçlü Weblate çevrimiçi aracını kullanıyoruz. Başlamak için talimatlar için lütfen [buraya](../development/frontend/translation.md) bakın.


## Diğer kullanıcıların sorularını yanıtlamaya yardımcı olun

Eğer zaten Gramps Web kullanıyorsanız, yeni kullanıcıların sorularına yanıt bulmalarına yardımcı olmak için resmi Gramps Discourse forumunun [Gramps Web kategorisi](https://gramps.discourse.group/c/gramps-web/) üzerindeki tartışmalara katkıda bulunun.

## Belgeleri geliştirmeye yardımcı olun

Göz attığınız belge sitesi [Github'ta](https://github.com/gramps-project/gramps-web-docs) barındırılmaktadır. Onu geliştirmek için katkılarınız çok değerlidir. Sadece o depoya bir pull request gönderin.

## Arka uç geliştirmeye katkıda bulunun (Python)

Eğer bir Python programcısıysanız, Gramps Web'i destekleyen [Gramps Web API](https://github.com/gramps-project/gramps-web-api/) geliştirmesine katkıda bulunun. Başlamak için talimatlar için [Arka uç geliştirme](../development/backend/index.md) sayfasına bakın.


## Ön uç geliştirmeye katkıda bulunun (Javascript)

Eğer bir web programcısıysanız, [Gramps Web'in duyarlı ön yüzü](https://github.com/gramps-project/gramps-web/) geliştirmesine katkıda bulunun. Başlamak için talimatlar için [Ön uç geliştirme](../development/frontend/index.md) sayfasına bakın.

## Sync eklentisine katkıda bulunun (Python)

Eğer Python ve GTK ile geliştirme deneyiminiz varsa, Gramps Web Sync Eklentisi'nin geliştirilmesine katkıda bulunun. İyileştirme önerileri ve gönderimleri için [geliştirme deposunu](https://github.com/DavidMStraub/gramps-web-sync/) kullanabilirsiniz.
