# Gramps 6.0 päivitysohje

Gramps 6.0 julkaistiin 19. maaliskuuta 2025. Koska se käyttää uutta sukupuuyhteystietokannan rakennetta verrattuna Gramps 5.2:een, tarvitaan lisäaskelia Gramps Webin päivittämiseksi käytettäväksi Gramps 6.0:n kanssa. Tämä sivu kuvaa tarvittavat vaiheet.

!!! info
    Viimeksi päivitetty: 4. toukokuuta 2025


## Yleiskatsaus vaiheista


Tässä on yleiskatsaus tarvittavista vaiheista päivitystä varten. 
Se näyttää myös, mitkä vaiheet suorittaa Gramps Web -palvelimen ylläpitäjä, sukupuun omistaja, ja onko se tarpeen vain [synkronointilisäosan](../administration/sync.md) käyttäjille.

## Yksityiskohtaiset vaiheet

Tässä osiossa annetaan tarkempia tietoja joistakin yllä olevista yksittäisistä vaiheista.

### Varmuuskopioi sukupuusi

Gramps Webissä siirry "Vienti" -sivulle ja valitse "Gramps XML (sukupuu)" (oletusvaihtoehto). Napsauta "Generoi" ja odota, kunnes .gramps-tiedoston lataaminen alkaa.

Voit myös luoda varmuuskopion Gramps Desktopissa [tavallisella tavalla](https://www.gramps-project.org/wiki/index.php/How_to_make_a_backup).

### Päivitä sukupuun tietokannan rakenne

Kun käyt Gramps Webissä ensimmäistä kertaa sen päivittämisen jälkeen, näet näytöllä viestin:

> Sukupuu, jota yrität ladata, on rakenteeltaan versiossa, jota tämä Gramps Web -versio ei tue. Siksi et voi ladata tätä sukupuuta ilman sen rakenteen päivittämistä. Tätä toimintoa ei voi peruuttaa.

Voit nyt napsauttaa "Päivitä sukupuu" -painiketta. Se näyttää edistymispalkin, eikä sen pitäisi kestää yli minuutti. Älä huoli, jos suljet vahingossa selainikkunan, muunnos jatkuu taustalla.

!!! warning
    Jos käytät ei-standardi asetusta *ilman* taustatehtäväjonoa, tietokannan päivitys voi epäonnistua ja sukupuusi voi vaurioitua. Voit tarkistaa (ennen Gramps Webin päivittämistä), käytätkö taustatehtäväjonoa siirtymällä asetuksissa "Järjestelmätiedot" -osioon ja varmistamalla, että siellä on "tehtäväjono: true".


### Päivitä Gramps Desktop

Noudata Gramps Desktopin asennusohjeita käyttöjärjestelmällesi.

### Asenna päivitetty Synkronointilisäosa

Voit käyttää Gramps 6.0:n Lisäosahallintoa asentaaksesi Gramps Web Sync -lisäosan. Etsi Gramps Web Sync, napsauta asenna ja käynnistä Gramps uudelleen.
