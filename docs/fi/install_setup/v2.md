# Gramps 5.2 päivitysohje

**Gramps 5.2** julkaistiin 23. helmikuuta 2024. Koska se käyttää uutta sukupuun tietokantarakennetta verrattuna Gramps 5.1:een, lisätoimia tarvitaan, kun päivität Gramps Webin käytettäväksi Gramps 5.2:n kanssa. Tämä sivu kuvaa tarvittavat vaiheet.

## Yksityiskohtaiset vaiheet

Tässä osiossa annetaan lisätietoja joistakin edellä mainituista yksittäisistä vaiheista.

### Varmuuskopioi sukupuusi

Gramps Webissä siirry "Vienti" -sivulle ja valitse "Gramps XML (sukupuu)" (oletusvaihtoehto). Napsauta "Generoi" ja odota, kunnes .gramps-tiedoston lataaminen alkaa.

Voit myös luoda varmuuskopion Gramps Desktopissa [tavallisella tavalla](https://www.gramps-project.org/wiki/index.php/How_to_make_a_backup).

### Päivitä puun tietokantarakenne

Kun käyt Gramps Webissä ensimmäistä kertaa sen päivittämisen jälkeen, näet näytöllä viestin:

> Lataamasi sukupuu on tietokantarakenteeltaan, jota tämä Gramps Webin versio ei tue. Siksi et voi ladata tätä sukupuuta ilman sen rakenteen päivittämistä. Tätä toimintoa ei voi peruuttaa.

Voit nyt napsauttaa "Päivitä sukupuu" -painiketta. Se näyttää edistymispalkin, eikä sen pitäisi kestää yli minuutin. Älä huoli, jos suljet vahingossa selainikkunan, muunnos jatkuu taustalla.

!!! warning
    Jos käytät ei-standardi asetusta *ilman* taustatehtäväjonoa, tietokannan päivitys saattaa epäonnistua ja sukupuusi voi vaurioitua. Voit tarkistaa (ennen Gramps Webin päivittämistä), käytätkö taustatehtäväjonoa siirtymällä asetuksissa "Järjestelmän tiedot" -osioon ja varmistaen, että siinä lukee "tehtäväjono: true".

### Päivitä Gramps Desktop

Seuraa Gramps Desktopin asennusohjeita käyttöjärjestelmällesi.

### Asenna päivitetty Synkronointilisäosa

Voit käyttää [uutta Lisäosahallintaa](https://gramps-project.org/wiki/index.php/Addon:Plugin_Manager), joka on mukana Gramps 5.2:ssa - etsi vain Gramps Web Synkronointia, napsauta asenna ja käynnistä Gramps uudelleen.
