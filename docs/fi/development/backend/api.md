# API-määrittely

Interaktiivinen API-määrittely (Swagger UI) tarjotaan suoraan Gramps Web -instanssistasi osoitteessa `/api/swagger-ui` – esimerkiksi `https://mygrampsweb.example.com/api/swagger-ui`.

Voit myös päästä siihen [Käyttäjäasetusten](../../user-guide/settings.md) kautta **Käynnistä Swagger** -painikkeella, joka avaa Swagger UI:n esitäytetyllä istuntotunnuksellasi.
