# Gramps Web -taustakehitys

Gramps Webin taustaosa kutsutaan [Gramps Web API](https://github.com/gramps-project/gramps-web-api/). Itse asiassa se on enemmän kuin vain sovelluksen tausta: se tarjoaa RESTful-rajapinnan Gramps-tietokannan lukemiseen ja muokkaamiseen.

Gramps Web API on Python-sovellus, ja sen taustalla on itse `gramps`-kirjasto (joka myös tukee Gramps Desktop -sovellusta).

Tässä asiakirjan osassa voit oppia, miten voit osallistua Gramps Web API:n kehittämiseen.
