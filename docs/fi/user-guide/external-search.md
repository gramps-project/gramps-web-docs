# Ulkoinen Haku

Gramps Web mahdollistaa henkilöiden etsimisen ulkoisilta sukututkimussivustoilta suoraan henkilön tietosivulta.


## Ulkoisen haun suorittaminen

Etsiäksesi henkilöä ulkoisilta sivustoilta:

1. Avaa henkilön tietonäkymä.
2. Napsauta ulkoisen haun painiketta.
3. Ponnahdusikkuna avautuu, jossa on linkkejä erilaisille sukututkimussivustoille, kuten CompGen, FamilySearch ja Ancestry.
4. Linkkiä napsauttamalla avautuu hakutulossivu kyseisellä sivustolla, joka on esitäytetty henkilön nimellä, syntymä-/kuolinpaikoilla ja muilla olennaisilla tiedoilla.

## Ulkoisten hakuvaihtoehtojen mukauttaminen

Voit mukauttaa, mitkä ulkoiset hakusivustot näkyvät ponnahdusikkunassa.

1. Ulkoisessa hakuponnahdusikkunassa napsauta ylhäällä olevaa rataskuvaketta siirtyäksesi muokkaustilaan.
2. Valitse tai poista valinta niistä verkkosivustoista, joita haluat näyttää tai piilottaa.
3. Tallenna muutoksesi.

## Mukautetun hakukoneen lisääminen

Voit lisätä uusia ulkoisia hakusivustoja mukautetuilla parametreilla.

1. Ulkoisessa hakuponnahdusikkunassa napsauta ylhäällä olevaa rataskuvaketta siirtyäksesi muokkaustilaan.
2. Napsauta "+ Lisää mukautettu hakukone" -painiketta.
3. Ilmoittautuu osio, jossa on kentät "Nimi" ja "Hakulinkki". Syötä verkkosivuston nimi ja URL, jossa on haettavat parametrit, kuten `https://example.com/search?name={{name_given}}_{{name_surname}}&place={{birth_place}}`.
4. Napsauta "Lisää" -painiketta lisätäksesi uusi hakukone luetteloon.
5. Uusi hakukone tallennetaan ja sitä voidaan mukauttaa myöhemmin.
