# Kirjanmerkit

Kirjanmerkit antavat sinun tallentaa linkkejä usein vierailluille kohteille (ihmiset, perheet, paikat, lähteet jne.) nopeaa pääsyä varten.

## Kirjanmerkkien lisääminen ja poistaminen

Jokaisella kohteen tietosivulla on kirjanmerkkikuvake (kirjanmerkin ääriviiva) ylätyökalupalkissa. Napsauta sitä tallentaaksesi kohteen kirjanmerkiksi – kuvake muuttuu täytetyksi kirjanmerkiksi. Napsauta sitä uudelleen poistaaksesi kirjanmerkin.

## Kirjanmerkkien tarkastelu

**Kirjanmerkit**-sivulle pääsee sivupalkista, ja se listaa kaikki tallennetut kohteesi. Napsauta mitä tahansa merkintää siirtyäksesi kyseisen kohteen tietosivulle.

## Käyttötapaukset

Tyypillinen käyttötapaus on tallentaa kirjanmerkki henkilölle tai perheelle, jota parhaillaan tutkit, jotta voit nopeasti palata heihin ilman hakua. Voit myös tallentaa kirjanmerkkejä tärkeille lähteille tai paikoille, joihin viittaat usein.

## Tallennus

Kirjanmerkit tallennetaan selaimen paikalliseen tallennustilaan, joka on rajattu nykyiseen puuhun. Ne ovat laite- ja selainkohtaisia, eivätkä ne synkronoidu laitteiden välillä tai jaa muiden käyttäjien kanssa.

!!! note
    Gramps Web -kirjanmerkit ovat itsenäisiä Gramps Desktopin kirjanmerkeistä. Ne eivät tallennu Gramps-tietokantaan eivätkä näy Gramps Desktopissa, ja päinvastoin.
