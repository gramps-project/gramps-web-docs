# Äskettäin Selatut Objektit

**Historia** sivupaneelissa avaa Äskettäin Selatut Objektit -sivun, joka listaa viimeisimmät objektit, joita olet avannut Gramps Webissä, uusimmat ensin. Muistissa on enintään 20 objektia.

Tämä on hyödyllistä, kun haluat nopeasti palata henkilöön, perheeseen tai muuhun objektiin, jota tarkastelit aiemmin istunnossasi, ilman että sinun tarvitsee etsiä sitä uudelleen.

Napsauta mitä tahansa merkintää palataksesi kyseiseen objektiin. Käytä **Tyhjennä kaikki** -painiketta nollataksesi luettelon.

Historia tallennetaan selaimen paikalliseen tallennustilaan – se on spesifinen nykyiselle laitteelle ja selaimelle, eikä sitä jaeta muiden käyttäjien kanssa.
