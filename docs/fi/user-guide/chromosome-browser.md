# Kromosomiselain

Kromosomiselain on työkalu, jolla voidaan visualisoida henkilön DNA-yhteensopivuuksia. Se näyttää DNA-segmentit, jotka on jaettu muiden ihmisten kanssa puussa värillisinä palkkeina kromosomeilla.

Voit löytää sen &ldquo;Kromosomiselain&rdquo; -välilehdeltä DNA-näkymässä.

Jokaiselle kromosomille ylempi (vaaleansininen) palkki edustaa isältä perittyä kopiota, alempi (vaaleanpunainen) palkki äidiltä perittyä kopiota. Jaetut segmentit näkyvät värillisinä palkkeina näiden päällä. Jos puoli (isä tai äiti) ei ole tiedossa, segmentti kattaa molemmat palkit ja se näytetään viivamerkeillä.

Napsauttamalla yhteensopivuuden nimeä selitteessä näytetään vain tämän henkilön jaetut segmentit.
