# Aikajana

!!! info
    Aikajanakatselu vaatii riittävän uuden Gramps Web API -version. Jos palvelimesi on vanhempi, **Aikajana**-kohta on piilotettu sivupalkista.

**Aikajana** on sivupalkin kohta, joka näyttää kaikki tapahtumat perhepuussasi aikajärjestyksessä vaakasuoralla, zoomattavalla aikajanalla.

## Aikajanan selaaminen

- Vieritä, vedä tai nipistä liikkuaksesi ajassa; käytä zoomauspainikkeita (tai hiiren rullaa/nipistysliikettä) zoomataksesi sisään ja ulos.
- Kun zoomaat sisään, tapahtumien nimilaput ilmestyvät; napsauta tapahtumapistettä nähdäksesi sen tiedot (ja muiden tapahtumien tiedot samana päivänä).

## Suodatus

Käytä suodatin kenttää keskittyäksesi aikajanaan, joka koskee yhtä **henkilöä** tai **paikkaa**:

- **Henkilö** – näyttää vain tapahtumat, jotka liittyvät kyseiseen henkilöön. Vaihtokytkin antaa sinun vaihtaa **Henkilö** (vain kyseisen yksilön tapahtumat), **Esivanhemmat** ja **Jälkeläiset** (kaikkien heidän esivanhempiensa/jälkeläistensä tapahtumat) välillä.
- **Paikka** – näyttää vain tapahtumat, jotka tapahtuivat siellä.

Tyhjennä suodatin, jotta voit palata koko aikajanalle.

## Hyppääminen aikajanalle

Tapahtuman tai paikan yksityiskohtasivulla **Näytä aikajanalla** -painike avaa Aikajanakatselun, joka on vieritetty ja zoomattu kyseisen tapahtuman päivämäärälle.
