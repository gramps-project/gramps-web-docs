# Ilmoitukset

**Ilmoitukset** on sivupalkin kohde, jossa on kellokuvake. Kun virheitä on tapahtunut, merkki näyttää lukemattomien ilmoitusten määrän. Napsauta sitä avataksesi ilmoituslokin.

Ilmoitusloki on rekisteri virheistä, jotka tapahtuivat istuntosi aikana – epäonnistuneet API-pyynnöt, taustatehtävien virheet, tallennusvirheet tai selainkohtaiset virheet. Jokaisessa merkinnässä näkyy lyhyt viesti, lähde (Verkko, Tehtävä, Tallennus tai Selain) ja aikaleima.

Jotkut ilmoitukset sisältävät rakenteellista tietoa. Napsauttamalla tällaista merkintää avautuu dialogi, jossa on erittely virhetiedoista ja **Kopioi JSON** -painike. Tämä on hyödyllistä, kun raportoidaan virheestä, sillä JSON sisältää tarkat virhetiedot palvelimelta.

Käytä **Tyhjennä kaikki** poistaaksesi kaikki ilmoitukset.

!!! note
    Ilmoitukset tallennetaan vain muistiin ja ne tyhjennetään, kun lataat sivun.
