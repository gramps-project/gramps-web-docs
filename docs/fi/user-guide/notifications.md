# Ilmoitukset

**Ilmoitukset** on sivupalkin kohde, jossa on kellokuvake. Kun virheitä on tapahtunut tai taustatehtäviä on käynnissä, merkki näyttää lukemattomien ilmoitusten määrän. Napsauta sitä avataksesi ilmoituslokin.

Ilmoituslokilla on kaksi tarkoitusta:

- Se on tallenne virheistä, jotka tapahtuivat istuntosi aikana – epäonnistuneet API-pyynnöt, taustatehtävien virheet, tallennusvirheet tai selainkohtaiset virheet.
- Se seuraa pitkään kestäviä taustatehtäviä – kuten tuonti ja vienti, raporttien luominen, OCR-tekstintunnistus, tietokannan päivitykset ja haku-/semanttisen indeksin uudelleenrakentaminen – näyttäen niiden tilan (esim. odottaa, aloitettu, käynnissä) ja ilmoittaen sinulle, kun ne valmistuvat tai epäonnistuvat.

Jokainen merkintä näyttää lyhyen viestin, lähteen (Verkko, Tehtävä, Tallennus tai Selain) ja aikaleiman.

Jotkut ilmoitukset sisältävät jäsenneltyä tietoa. Napsauttamalla tällaista merkintää avautuu dialogi, jossa on erittely virhetiedoista ja **Kopioi JSON** -painike. Tämä on hyödyllistä, kun ilmoitat virheestä, sillä JSON sisältää tarkat virhetiedot palvelimelta.

Käytä **Tyhjennä kaikki** poistaaksesi kaikki ilmoitukset.

!!! note
    Ilmoitukset tallennetaan vain muistiin ja ne tyhjennetään, kun lataat sivun uudelleen.
