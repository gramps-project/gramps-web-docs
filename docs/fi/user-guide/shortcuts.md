# Näppäinoikoteiden käyttäminen

Näppäinoikoteiden avulla voit navigoida nopeasti Gramps Webissä ilman hiiren käyttöä, mikä säästää klikkauksia ja aikaa.

Mistä tahansa näkymästä voit hypätä suoraan yleisiin toimintoihin ja sivuille vain muutamalla näppäinpainalluksella. Esimerkiksi, paina **`s`** näppäimistölläsi avataksesi haun, **`g`** + **`c`** siirtyäksesi Sukupuuhun, tai **`n`** + **`p`** luodaksesi uuden henkilön.

Näyttääksesi täydellisen luettelon kaikista käytettävissä olevista näppäinoikoteista, paina yksinkertaisesti **`?`** milloin tahansa.
