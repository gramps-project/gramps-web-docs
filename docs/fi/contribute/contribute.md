# Kuinka osallistua Gramps Webiin

Kiitos, että harkitset osallistumista Gramps Webiin! Taidoistasi ja kiinnostuksistasi riippuen on monia tapoja, joilla voit auttaa.

## Auta kääntämään Gramps Web omaan kieleesi

Gramps ja Gramps Web tukevat useita kymmeniä eri kieliä. Sovelluksen kääntäminen ei vaadi ohjelmointitaitoja. Käytämme voimakasta Weblate-verkkotyökalua käännöksiin. Katso ohjeet [täältä](../development/frontend/translation.md) aloittaaksesi.

## Auta vastaamaan muiden käyttäjien kysymyksiin

Jos käytät jo Gramps Webiä, auta uusia käyttäjiä löytämään vastauksia kysymyksiinsä osallistumalla keskusteluihin [Gramps Web -kategoriassa](https://gramps.discourse.group/c/gramps-web/) virallisella Gramps Discourse -foorumilla.

## Auta parantamaan dokumentaatiota

Dokumentaatiosivusto, jota tarkastelet, on isännöity [Githubissa](https://github.com/gramps-project/gramps-web-docs). Parannusehdotukset ovat erittäin tervetulleita. Lähetä vain pull request kyseiseen repositorioon.

## Osallistu taustakehitykseen (Python)

Jos olet Python-ohjelmoija, osallistu [Gramps Web API:n](https://github.com/gramps-project/gramps-web-api/) kehittämiseen, joka on Gramps Webin taustajärjestelmä. Katso [Taustakehitys](../development/backend/index.md) ohjeita varten aloittamiseen.

## Osallistu käyttöliittymäkehitykseen (Javascript)

Jos olet verkkosovellusohjelmoija, osallistu [Gramps Webin responsiivisen käyttöliittymän](https://github.com/gramps-project/gramps-web/) kehittämiseen. Katso [Käyttöliittymäkehitys](../development/frontend/index.md) ohjeita varten aloittamiseen.

## Osallistu Sync-lisäosan kehittämiseen (Python)

Jos sinulla on kokemusta Pythonin ja GTK:n kehittämisestä, osallistu Gramps Web Sync -lisäosan parantamiseen. Voit käyttää [kehitysrepositoriota](https://github.com/DavidMStraub/gramps-web-sync/) ehdotusten ja parannusten lähettämiseen.
