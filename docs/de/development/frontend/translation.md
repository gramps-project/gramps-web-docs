Gramps Web nutzt die Übersetzungen von Gramps Desktop, die von der Community in über 40 Sprachen bereitgestellt werden. Die zusätzlichen Strings, die nur für die Web-App benötigt werden, werden über Weblate übersetzt.

Um beizutragen, folgen Sie [den Anweisungen im Gramps-Wiki](https://gramps-project.org/wiki/index.php/Translating_Gramps_using_Weblate), um mit Weblate für Gramps zu beginnen. Nach der Authentifizierung können Sie mit der Übersetzung von Strings sowohl für Gramps Desktop als auch für Gramps Web beginnen.
