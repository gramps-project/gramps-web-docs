# Gramps 5.2 Upgrade-Anleitung

**Gramps 5.2** wurde am 23. Februar 2024 veröffentlicht. Da es ein neues Datenbankschema für Familienstammbäume im Vergleich zu Gramps 5.1 verwendet, sind zusätzliche Schritte erforderlich, um Gramps Web für die Verwendung mit Gramps 5.2 zu aktualisieren. Diese Seite beschreibt die notwendigen Schritte.

## Übersicht der Schritte

Dies ist eine Übersicht der erforderlichen Schritte für das Upgrade.  
Es zeigt auch, welche Schritte vom Administrator des Gramps Web-Servers, dem Eigentümer des Familienstammes ausgeführt werden und ob sie nur für Benutzer des [Sync-Addons](../administration/sync.md) erforderlich sind.

Serveradministrator | Baum-Eigentümer  | Aktion
----|----|----
&nbsp; | ✓ | Sichern Sie Ihren Familienstamm
&nbsp; | ✓ <sup>1</sup> |  [Synchronisieren Sie Gramps Web mit Gramps Desktop](../administration/sync.md)
✓ |  | [Gramps Web aktualisieren](update.md)
&nbsp; | ✓ | Aktualisieren Sie das Datenbankschema des Baums
&nbsp; | ✓ <sup>1</sup> | Gramps Desktop aktualisieren
&nbsp; | ✓ <sup>1</sup> | Installieren Sie das aktualisierte Sync-Addon
&nbsp; | ✓ | Sichern Sie Ihren Familienstamm
&nbsp; | ✓ <sup>1</sup> | [Synchronisieren Sie Gramps Web mit Gramps Desktop](../administration/sync.md)

<sup>1</sup> Nur erforderlich, wenn Sie Gramps Web mit Gramps Desktop synchronisieren.

## Detaillierte Schritte

Dieser Abschnitt bietet weitere Details zu einigen der oben genannten Schritte.

### Sichern Sie Ihren Familienstamm

Gehen Sie in Gramps Web zur Seite "Export" und wählen Sie "Gramps XML (Familienstamm)" (die Standardoption). Klicken Sie auf "Generieren" und warten Sie, bis der Download der .gramps-Datei beginnt.

Sie können auch ein Backup in Gramps Desktop [auf die übliche Weise](https://www.gramps-project.org/wiki/index.php/How_to_make_a_backup) erstellen.

### Aktualisieren Sie das Datenbankschema des Baums

Wenn Sie Gramps Web zum ersten Mal nach der Aktualisierung aufrufen, wird Ihnen ein Bildschirm mit der Nachricht angezeigt:

> Der Familienstamm, den Sie zu laden versuchen, befindet sich in einer Schema-Version, die von dieser Version von Gramps Web nicht unterstützt wird. Daher können Sie diesen Familienstamm nicht laden, ohne sein Schema zu aktualisieren. Diese Aktion kann nicht rückgängig gemacht werden.

Sie können jetzt auf die Schaltfläche "Familienstamm aktualisieren" klicken. Es wird ein Fortschrittsbalken angezeigt und sollte nicht länger als eine Minute dauern. Machen Sie sich keine Sorgen, wenn Sie versehentlich Ihr Browserfenster schließen, die Konvertierung wird im Hintergrund fortgesetzt.

!!! warning
    Wenn Sie eine nicht standardmäßige Einrichtung *ohne* Hintergrundaufgabenwarteschlange verwenden, könnte das Datenbank-Upgrade fehlschlagen und Ihr Familienstamm könnte beschädigt werden. Sie können (bevor Sie Gramps Web aktualisieren) überprüfen, ob Sie eine Hintergrundaufgabenwarteschlange verwenden, indem Sie zum Abschnitt "Systeminformationen" in den Einstellungen gehen und sicherstellen, dass dort "task queue: true" steht.

### Aktualisieren Sie Gramps Desktop

Befolgen Sie die Installationsanweisungen für Gramps Desktop für Ihr Betriebssystem.

### Installieren Sie das aktualisierte Sync-Addon

Sie können [den neuen Plugin-Manager](https://gramps-project.org/wiki/index.php/Addon:Plugin_Manager) verwenden, der in Gramps 5.2 enthalten ist - suchen Sie einfach nach Gramps Web Sync, klicken Sie auf Installieren und starten Sie Gramps neu.
