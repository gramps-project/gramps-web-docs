# Benachrichtigungen

**Benachrichtigungen** ist ein Sidebar-Element mit einem Glockensymbol. Wenn Fehler aufgetreten sind, zeigt ein Badge die Anzahl der ungelesenen Benachrichtigungen an. Klicken Sie darauf, um das Benachrichtigungsprotokoll zu öffnen.

Das Benachrichtigungsprotokoll ist ein Protokoll der Fehler, die während Ihrer Sitzung aufgetreten sind – fehlgeschlagene API-Anfragen, Fehler bei Hintergrundaufgaben, Speicherfehler oder Fehler auf Browser-Ebene. Jeder Eintrag zeigt eine kurze Nachricht, die Quelle (Netzwerk, Aufgabe, Speicherung oder Browser) und einen Zeitstempel an.

Einige Benachrichtigungen enthalten strukturierte Details. Das Klicken auf einen solchen Eintrag öffnet einen Dialog mit einer Aufschlüsselung der Fehlermeldungsdaten und einer **JSON kopieren**-Schaltfläche. Dies ist nützlich, wenn Sie einen Fehler melden, da die JSON die genauen Fehlerinformationen vom Server enthält.

Verwenden Sie **Alle löschen**, um alle Benachrichtigungen zu entfernen.

!!! Hinweis
    Benachrichtigungen werden nur im Speicher gespeichert und werden gelöscht, wenn Sie die Seite neu laden.
