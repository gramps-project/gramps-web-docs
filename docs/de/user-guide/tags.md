# Tags

Tags sind Labels, die auf jedes Objekt in der Gramps-Datenbank angewendet werden können – Personen, Familien, Ereignisse, Orte, Quellen, Zitationen, Repositories, Notizen und Medien. Sie sind nützlich, um Objekte zu gruppieren und zu filtern. Tags werden in der Gramps-Familienstammbaum-Datenbank gespeichert und sind für alle Benutzer zugänglich; sie sind auch vollständig kompatibel mit Tags, die in Gramps Desktop erstellt wurden.

## Verwaltung von Tags

Die **Tags**-Seite ist über die Seitenleiste zugänglich und nur für Benutzer mit der Rolle Editor oder höher sichtbar. Sie zeigt alle vorhandenen Tags an und ermöglicht es Ihnen:

- **Einen** neuen Tag mit der +-Schaltfläche (unten rechts) zu erstellen
- **Einen** Tag mit dem Bearbeiten (Bleistift)-Symbol umzubenennen
- **Die Farbe** eines Tags mit dem Farbauswähler zu ändern
- **Einen** Tag mit dem Löschen-Symbol zu entfernen

!!! note
    Das Löschen eines Tags entfernt ihn von allen Objekten, auf die er angewendet wurde.

## Tags auf Objekte anwenden

Tags können auf ein Objekt auf seiner Detailseite im Bearbeitungsmodus angewendet oder von ihm entfernt werden.

## Nach Tag filtern

Alle Objektlisten-Seiten (Personen, Familien, Ereignisse, Orte, Quellen, Zitationen, Repositories, Notizen, Medien) enthalten einen Tag-Filter. Verwenden Sie ihn, um nur Objekte anzuzeigen, die ein bestimmtes Tag haben.

## Besondere Tags

Zwei Tags haben eine besondere Bedeutung in Gramps Web:

- **`Blog`** – jede Quelle, die mit `Blog` gekennzeichnet ist, wird als Blogbeitrag behandelt und erscheint in der [Blog](blog.md)-Ansicht
- **`ToDo`** – jede Notiz, die mit `ToDo` gekennzeichnet ist, wird als Forschungsaufgabe behandelt und erscheint in der [Tasks](tasks.md)-Ansicht

Diese Tags werden automatisch erstellt, wenn Sie die Blog- oder Aufgabenfunktionen zum ersten Mal verwenden. Das Umbenennen oder Löschen dieser Tags wird die entsprechende Funktion beeinträchtigen.
