# Externe Suche

Gramps Web ermöglicht es Ihnen, direkt von der Detailseite einer Person aus nach Personen auf externen Genealogie-Websites zu suchen.


## Durchführung einer externen Suche

Um nach einer Person auf externen Websites zu suchen:

1. Öffnen Sie die Detailansicht einer Person.
2. Klicken Sie auf die Schaltfläche für die externe Suche.
3. Ein Popup-Fenster erscheint mit Links zu verschiedenen Genealogie-Websites, wie CompGen, FamilySearch und Ancestry.
4. Ein Klick auf einen Link öffnet die Suchergebnisseite auf dieser Website, die bereits mit dem Namen der Person, Geburts-/Sterbeorten und anderen relevanten Informationen vorausgefüllt ist.

## Anpassen der Optionen für die externe Suche

Sie können anpassen, welche externen Such-Websites im Popup angezeigt werden.

1. Klicken Sie im Popup für die externe Suche auf die Zahnradtaste oben, um den Bearbeitungsmodus zu aktivieren.
2. Wählen Sie die Websites aus, die Sie anzeigen oder ausblenden möchten.
3. Speichern Sie Ihre Änderungen.

## Hinzufügen einer benutzerdefinierten Suchmaschine

Sie können neue externe Such-Websites mit benutzerdefinierten Parametern hinzufügen.

1. Klicken Sie im Popup für die externe Suche auf die Zahnradtaste oben, um den Bearbeitungsmodus zu aktivieren.
2. Klicken Sie auf die Schaltfläche "+ Benutzerdefinierte Suchmaschine hinzufügen".
3. Ein Abschnitt erscheint mit Feldern für "Name" und "Such-URL". Geben Sie den Namen der Website und die URL mit durchsuchbaren Parametern ein, wie `https://example.com/search?name={{name_given}}_{{name_surname}}&place={{birth_place}}`.
4. Klicken Sie auf die Schaltfläche "Hinzufügen", um die neue Suchmaschine in die Liste aufzunehmen.
5. Die neue Suchmaschine wird gespeichert und kann später angepasst werden.
