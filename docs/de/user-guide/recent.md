# Kürzlich Durchsuchte Objekte

**Verlauf** in der Seitenleiste öffnet die Seite Kürzlich Durchsuchte Objekte, die die letzten Objekte auflistet, die Sie in Gramps Web geöffnet haben, wobei die zuletzt angesehenen Objekte oben stehen. Es werden bis zu 20 Objekte gespeichert.

Dies ist nützlich, um schnell zu einer Person, Familie oder einem anderen Objekt zurückzukehren, das Sie zuvor in Ihrer Sitzung angesehen haben, ohne es erneut suchen zu müssen.

Klicken Sie auf einen Eintrag, um zu diesem Objekt zurückzukehren. Verwenden Sie die Schaltfläche **Alle Löschen**, um die Liste zurückzusetzen.

Der Verlauf wird im lokalen Speicher des Browsers gespeichert – er ist spezifisch für das aktuelle Gerät und den aktuellen Browser und wird nicht mit anderen Benutzern geteilt.
