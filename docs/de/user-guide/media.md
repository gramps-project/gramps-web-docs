Es gibt zwei Möglichkeiten, eine neue Mediendatei (ein Bild, eine Audiodatei, eine Videodatei oder eine andere Datei) hinzuzufügen:

## Fügen Sie eine neue eigenständige Mediendatei hinzu

Um eine eigenständige Mediendatei hinzuzufügen, klicken Sie auf das +-Symbol in der oberen App-Leiste und wählen Sie "Medienobjekt".

Klicken Sie auf "eine Datei auswählen", um eine Datei von Ihrem Computer auszuwählen. Auf einem mobilen Gerät haben Sie die Möglichkeit, direkt ein Foto mit der Kamera Ihres Geräts aufzunehmen, wenn Sie diese Schaltfläche drücken.

Optional können Sie

- eine Beschreibung der Mediendatei unter "Titel" eingeben
- ein Datum eingeben
- die Mediendatei als privat festlegen (was bedeutet, dass sie nur für Benutzer mit ausreichenden Berechtigungen sichtbar ist)

Klicken Sie auf "hinzufügen", um die Datei hochzuladen und das Medienobjekt zu erstellen.

## Fügen Sie eine neue Mediendatei hinzu und verlinken Sie sie mit einem anderen Objekt

Die folgenden Objekttypen in Gramps können Medienobjekte angehängt haben: Personen, Familien, Ereignisse, Orte, Quellen und Zitationen.

Klicken Sie in der Detailansicht eines beliebigen Objekts auf die blaue Schaltfläche "Bearbeiten" in der unteren rechten Ecke (wenn Sie sie nicht sehen, hat Ihr Benutzer keine Bearbeitungsberechtigungen). Klicken Sie auf die Registerkarte "Galerie" und klicken Sie auf die blaue +-Schaltfläche.

Ein Dialogfeld öffnet sich, das die gleichen Felder wie im vorherigen Abschnitt beschrieben anbietet. Klicken Sie auf "speichern", um die Datei hochzuladen, ein neues Medienobjekt hinzuzufügen und es mit dem angezeigten Objekt zu verlinken.
