# Développement du frontend Gramps Web

Le [frontend Gramps Web](https://github.com/gramps-project/gramps-web) est écrit en Javascript et construit à l'aide de composants web avec [Lit](https://lit.dev/). Cela permet d'encapsuler des fonctionnalités dans des composants réutilisables.

Le frontend est conçu comme une application web monopage progressive avec une expérience mobile-first. Il utilise largement les [Material Web Components](https://github.com/material-components/material-web) pour un aspect et une sensation de Material Design.
