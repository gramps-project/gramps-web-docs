# Développement du backend de Gramps Web

Le backend de Gramps Web s'appelle [Gramps Web API](https://github.com/gramps-project/gramps-web-api/). En fait, c'est plus qu'un simple backend d'application : il fournit une API RESTful pour lire et modifier une base de données Gramps.

Gramps Web API est une application Python et est alimentée par la bibliothèque `gramps` elle-même (qui alimente également l'application Gramps Desktop).

Dans cette section de la documentation, vous pouvez apprendre comment contribuer au développement de Gramps Web API.
