# Spécification de l'API

La spécification API interactive (Swagger UI) est servie directement par votre instance Gramps Web à `/api/swagger-ui` – par exemple `https://mygrampsweb.example.com/api/swagger-ui`.

Vous pouvez également y accéder depuis [Paramètres de l'utilisateur](../../user-guide/settings.md) via le bouton **Lancer Swagger**, qui ouvre l'interface Swagger avec votre jeton de session pré-rempli.
