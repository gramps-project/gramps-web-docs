# Recherche externe

Gramps Web vous permet de rechercher des personnes sur des sites de généalogie externes directement depuis la page de détails d'une personne.


## Effectuer une recherche externe

Pour rechercher une personne sur des sites externes :

1. Ouvrez la vue détaillée d'une personne.
2. Cliquez sur le bouton de recherche externe.
3. Une fenêtre contextuelle apparaîtra avec des liens vers divers sites de généalogie, tels que CompGen, FamilySearch et Ancestry.
4. En cliquant sur un lien, la page des résultats de recherche s'ouvrira sur ce site, pré-remplie avec le nom de la personne, les lieux de naissance/décès et d'autres informations pertinentes.

## Personnaliser les options de recherche externe

Vous pouvez personnaliser les sites de recherche externes affichés dans la fenêtre contextuelle.

1. Dans la fenêtre contextuelle de recherche externe, cliquez sur le bouton d'engrenage en haut pour entrer en mode édition.
2. Sélectionnez ou désélectionnez les sites que vous souhaitez afficher ou masquer.
3. Enregistrez vos modifications.

## Ajouter un moteur de recherche personnalisé

Vous pouvez ajouter de nouveaux sites de recherche externes avec des paramètres personnalisés.

1. Dans la fenêtre contextuelle de recherche externe, cliquez sur le bouton d'engrenage en haut pour entrer en mode édition.
2. Cliquez sur le bouton "+ Ajouter un moteur de recherche personnalisé".
3. Une section apparaîtra avec des champs pour "Nom" et "URL de recherche". Entrez le nom du site et l'URL avec des paramètres de recherche, tels que `https://example.com/search?name={{name_given}}_{{name_surname}}&place={{birth_place}}`.
4. Cliquez sur le bouton "Ajouter" pour inclure le nouveau moteur de recherche dans la liste.
5. Le nouveau moteur de recherche sera enregistré et pourra être personnalisé ultérieurement.
