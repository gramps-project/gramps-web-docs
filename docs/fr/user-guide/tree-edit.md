# Édition de l'Arbre Généalogique

La vue de l'Arbre Ancêtre offre un moyen rapide de construire et d'étendre la structure de votre arbre généalogique directement à partir du graphique, sans quitter la vue de l'arbre.

!!! note
    L'édition de l'arbre est disponible dans les onglets Arbre Ancêtre, Arbre Descendant, Graphique Sablier et Graphique de Relation. Elle n'est pas disponible dans le Graphique en éventail. Seuls les utilisateurs ayant le rôle d'Éditeur ou supérieur peuvent l'utiliser.

## Entrer et sortir du mode édition

Les utilisateurs ayant des permissions d'édition voient un bouton crayon flottant (en bas à droite) sur les pages de graphique prises en charge. Cliquez dessus pour entrer en mode édition – ou appuyez sur le raccourci clavier `e` [raccourci clavier](shortcuts.md). La barre d'application affichera un indicateur de mode édition.

Chaque changement (ajout ou liaison d'une personne) est enregistré dans la base de données immédiatement lorsqu'il est confirmé dans la boîte de dialogue. Pour sortir du mode édition, cliquez sur le bouton de fermeture dans la barre d'application, ou passez à un autre onglet de graphique.

!!! note
    Changer d'onglet de graphique quitte automatiquement le mode édition.

## Ajouter un membre de la famille

En mode édition, chaque carte de personne dans le graphique affiche un petit bouton **+**. En cliquant dessus, vous ouvrez la boîte de dialogue **Ajouter un Membre de la Famille** pour cette personne, avec une ou deux icônes pour chaque relation disponible :

- **Père** – affiché uniquement si la personne n'a pas encore de père dans sa famille parentale principale
- **Mère** – affiché uniquement si la personne n'a pas encore de mère dans sa famille parentale principale
- **Enfant**
- **Conjoint**

Chaque relation offre deux options côte à côte :

- **Icône de lien** – connecter une personne existante déjà dans la base de données
- **Icône +** – créer une toute nouvelle personne

## Lier une personne existante

Choisissez l'icône de lien et recherchez la personne par son nom. Sélectionnez-la dans les résultats pour créer le lien familial.

Pour un **enfant**, vous pouvez également définir le type de relation comme père et mère (Naissance, Adopté, Enfant de cœur, etc.).

Pour un **conjoint**, vous pouvez définir le type de relation familiale (Marié, Union Civile, Inconnu, etc.).

## Créer une nouvelle personne

Choisissez l'icône + pour ouvrir un formulaire de création minimal. Les champs disponibles sont :

- **Nom** (prénom et nom de famille)
- **Genre** (Inconnu, Masculin, Féminin, Autre)
- **Date et lieu de naissance**
- **Date et lieu de décès**

La nouvelle personne est enregistrée et immédiatement liée à la personne sélectionnée dans le graphique.

## Limitations

La vue d'édition de l'arbre est destinée à construire rapidement le squelette d'un arbre généalogique. Utilisez la page de détails de la personne pour tout ce qui va au-delà des bases : citations, notes, médias, attributs, noms alternatifs, détails d'événements existants, familles avec plusieurs parents (seule la famille principale est accessible ici), et gestion de plusieurs conjoints.

Utilisez **Détails de la Personne** (l'icône de carte de compte dans la barre d'outils) pour passer directement à la vue d'édition complète de la personne actuellement centrée.
