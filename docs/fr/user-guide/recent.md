# Objets Consultés Récemment

**Historique** dans la barre latérale ouvre la page Objets Consultés Récemment, qui liste les derniers objets que vous avez ouverts dans Gramps Web, avec les objets les plus récemment consultés en haut. Jusqu'à 20 objets sont mémorisés.

Ceci est utile pour naviguer rapidement vers une personne, une famille ou un autre objet que vous regardiez plus tôt dans votre session, sans avoir à le rechercher à nouveau.

Cliquez sur n'importe quelle entrée pour revenir à cet objet. Utilisez le bouton **Tout Effacer** pour réinitialiser la liste.

L'historique est stocké dans le stockage local du navigateur – il est spécifique à l'appareil et au navigateur actuels, et n'est pas partagé avec d'autres utilisateurs.
