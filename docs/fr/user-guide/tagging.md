Gramps Web permet de taguer des personnes sur des photographies, similaire à Gramps Desktop, en liant la personne et l'objet média. Cela est simplifié par la détection automatique des visages.

Pour taguer une personne sur une photo, suivez ces étapes :

- Ouvrez la vue détaillée de l'objet média, par exemple en cliquant sur "afficher les détails" dans l'aperçu plein écran.
- Cliquez sur le bouton bleu d'édition dans le coin inférieur droit. Si vous ne voyez pas ce bouton, votre utilisateur n'a pas les autorisations d'édition.
- Superposées à l'image, vous verrez des cases avec une bordure en pointillés correspondant aux régions faciales détectées. Pour taguer une personne, cliquez sur la case pour la mettre en surbrillance et cliquez sur le bouton "personne" pour sélectionner une personne existante.
