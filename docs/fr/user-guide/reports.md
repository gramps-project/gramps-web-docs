# Rapports

Un rapport est un document formaté et téléchargeable généré à partir des données de votre arbre généalogique. Les rapports sont les mêmes plugins disponibles dans Gramps Desktop – consultez la [page Wiki de Gramps sur les Rapports](https://gramps-project.org/wiki/index.php/Gramps_5.1_Wiki_Manual_-_Reports) pour un aperçu complet. L'interface web exécute le même moteur de rapport sur le serveur et livre le fichier résultant à votre navigateur.

Les rapports sont accessibles depuis la barre latérale.

## Catégories de rapports

La page des Rapports liste tous les rapports disponibles regroupés par catégorie :

- **Rapports Textuels** – documents narratifs ou tabulaires tels que des listes d'ancêtres, des rapports de descendants et des calendriers d'anniversaires. Les formats de sortie incluent PDF, ODT/OpenDocument, RTF, HTML, et d'autres.
- **Rapports Graphiques** – graphiques et diagrammes tels que des arbres d'ancêtres et des graphiques en éventail. Les formats de sortie incluent PDF, SVG, PNG, et d'autres.
- **Arbres** – diagrammes d'arbres généalogiques.
- **Livres** – livres de rapports multi-chapitres qui combinent plusieurs rapports textuels et graphiques en un seul document.
- **Pages Web** – générateurs de sites web statiques qui produisent un ensemble complet de pages HTML à partir de votre arbre.

Les rapports affichés dépendent des plugins de rapport Gramps installés sur le serveur.

## Exécution d'un rapport

Cliquez sur un rapport pour ouvrir sa page d'options, qui affiche :

- **Description** – ce que contient le rapport
- **Auteur** et **Version** du plugin
- **Options** – paramètres spécifiques au rapport tels que la personne de départ, le nombre de générations, le format de sortie, la taille de page, et d'autres paramètres

Ajustez les options selon vos besoins, puis cliquez sur **Générer**. Le rapport est généré sur le serveur et le fichier résultant est téléchargé automatiquement dans votre navigateur.

!!! astuce
    La plupart des rapports sont par défaut basés sur la **personne principale de la base de données** – la personne principale définie dans la base de données Gramps, qui peut différer de votre réglage personnel de la personne principale dans le navigateur. Si le rapport concerne la mauvaise personne, changez l'option de personne dans les paramètres du rapport.
