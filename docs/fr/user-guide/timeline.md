# Chronologie

!!! info
    La vue Chronologie nécessite une version suffisamment récente de l'API Gramps Web. Si votre serveur est ancien, l'élément **Chronologie** est masqué dans la barre latérale.

**Chronologie** est un élément de la barre latérale qui montre tous les événements de votre arbre généalogique disposés chronologiquement sur une chronologie horizontale et zoomable.

## Parcourir la chronologie

- Faites défiler, faites glisser ou pincez pour avancer dans le temps ; utilisez les boutons de zoom (ou la molette de votre souris/geste de pincement) pour zoomer avant et arrière.
- Lorsque vous zoomez, les étiquettes des événements apparaissent ; cliquez sur un point d'événement pour voir ses détails (et ceux de tout autre événement à la même date).

## Filtrage

Utilisez le champ de filtre pour concentrer la chronologie sur une **personne** ou un **lieu** unique :

- **Personne** – affiche uniquement les événements liés à cette personne. Un bouton bascule vous permet de passer entre **Personne** (événements de cet individu uniquement), **Ancêtres** et **Descendants** (événements de tous leurs ancêtres/descendants).
- **Lieu** – affiche uniquement les événements qui s'y sont déroulés.

Effacez la puce de filtre pour revenir à la chronologie complète.

## Accéder à la chronologie

Sur la page de détails d'un événement ou d'un lieu, le bouton **Afficher sur la chronologie** ouvre la vue Chronologie défilée et zoomée à la date de cet événement.
