# Utilisation des Raccourcis Clavier

Les raccourcis clavier vous permettent de naviguer rapidement dans Gramps Web sans utiliser votre souris, vous faisant économiser des clics et du temps.

Depuis n'importe quelle vue, vous pouvez accéder directement à des actions et des pages courantes en quelques frappes de clavier. Par exemple, appuyez sur **`s`** sur votre clavier pour ouvrir la recherche, **`g`** + **`c`** pour naviguer vers l'Arbre Généalogique, ou **`n`** + **`p`** pour créer une nouvelle personne.

Pour voir une liste complète de tous les raccourcis clavier disponibles, appuyez simplement sur **`?`** à tout moment.
