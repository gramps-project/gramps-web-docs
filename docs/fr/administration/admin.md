# Guide de l'administrateur Gramps Web

Que vous ayez configuré votre propre instance de Gramps Web ou que vous vous soyez inscrit à une instance pré-installée, l'étape suivante consiste à peupler et gérer votre base de données d'arbre généalogique.

L'accent de cette section est mis sur les tâches administratives effectuées par les **propriétaires d'arbre**. En général, il y a un propriétaire d'arbre pour chaque base de données d'arbre généalogique (bien qu'en tant que propriétaire d'arbre, vous puissiez également créer d'autres comptes de propriétaires d'arbre).

Cette section ne traite que des actions qui peuvent être effectuées depuis l'interface web. Pour les outils en ligne de commande ou la configuration du serveur, voir la section précédente [Configuration](../install_setup/setup.md).
