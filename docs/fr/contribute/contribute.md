# Comment contribuer à Gramps Web

Merci de considérer de contribuer à Gramps Web ! En fonction de vos compétences et de vos intérêts, il existe de nombreuses façons de vous aider.

## Aidez à traduire Gramps Web dans votre langue

Gramps et Gramps Web prennent en charge plusieurs dizaines de langues différentes. Aider à traduire l'application ne nécessite aucune compétence en programmation. Nous utilisons le puissant outil en ligne Weblate pour les traductions. Veuillez consulter [ici](../development/frontend/translation.md) pour des instructions sur la façon de commencer.


## Aidez à répondre aux questions des autres utilisateurs

Si vous utilisez déjà Gramps Web, aidez les nouveaux utilisateurs à trouver les réponses à leurs questions en contribuant aux discussions dans la [catégorie Gramps Web](https://gramps.discourse.group/c/gramps-web/) du forum officiel de Gramps Discourse.

## Aidez à améliorer la documentation

Le site de documentation que vous consultez est hébergé [sur Github](https://github.com/gramps-project/gramps-web-docs). Les contributions pour l'améliorer sont les bienvenues. Il suffit de soumettre une demande de tirage à ce dépôt.

## Contribuez au développement backend (Python)

Si vous êtes un programmeur Python, contribuez au développement de l'[API Gramps Web](https://github.com/gramps-project/gramps-web-api/), qui est le backend alimentant Gramps Web. Consultez [Développement backend](../development/backend/index.md) pour des instructions sur la façon de commencer.


## Contribuez au développement frontend (Javascript)

Si vous êtes un programmeur web, contribuez au développement du [frontend réactif de Gramps Web](https://github.com/gramps-project/gramps-web/). Consultez [Développement frontend](../development/frontend/index.md) pour des instructions sur la façon de commencer.

## Contribuez à l'addon Sync (Python)

Si vous avez de l'expérience dans le développement avec Python et GTK, contribuez à l'amélioration de l'addon Sync de Gramps Web. Vous pouvez utiliser le [dépôt de développement](https://github.com/DavidMStraub/gramps-web-sync/) pour suggérer et soumettre des améliorations.
