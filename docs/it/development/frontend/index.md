# Sviluppo del frontend di Gramps Web

Il [frontend di Gramps Web](https://github.com/gramps-project/gramps-web) è scritto in Javascript e costruito utilizzando componenti web con [Lit](https://lit.dev/). Questo consente di incapsulare la funzionalità in componenti riutilizzabili.

Il frontend è progettato come un'app web a pagina singola progressiva con un'esperienza mobile-first. Fa ampio uso di [Material Web Components](https://github.com/material-components/material-web) per un aspetto e una sensazione di Material Design.
