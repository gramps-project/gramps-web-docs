# Specifica API

La specifica API interattiva (Swagger UI) è servita direttamente dalla tua istanza Gramps Web all'indirizzo `/api/swagger-ui` – ad esempio `https://mygrampsweb.example.com/api/swagger-ui`.

Puoi anche accedervi dalle [Impostazioni Utente](../../user-guide/settings.md) tramite il pulsante **Avvia Swagger**, che apre la Swagger UI con il tuo token di sessione già compilato.
