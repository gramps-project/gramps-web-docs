# Sviluppo del backend di Gramps Web

Il backend di Gramps Web è chiamato [Gramps Web API](https://github.com/gramps-project/gramps-web-api/). In effetti, è più di un semplice backend per un'app: fornisce un'API RESTful per leggere e modificare un database di Gramps.

Gramps Web API è un'applicazione Python ed è alimentata dalla libreria `gramps` stessa (che alimenta anche l'applicazione Gramps Desktop).

In questa sezione della documentazione, puoi imparare come contribuire allo sviluppo di Gramps Web API.
