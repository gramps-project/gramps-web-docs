# Guida all'aggiornamento di Gramps 5.2

**Gramps 5.2** è stato rilasciato il 23 febbraio 2024. Poiché utilizza un nuovo schema di database per l'albero genealogico rispetto a Gramps 5.1, sono necessari ulteriori passaggi quando si aggiorna Gramps Web per essere utilizzato con Gramps 5.2. Questa pagina descrive i passaggi necessari.

## Passaggi dettagliati

Questa sezione fornisce maggiori dettagli su alcuni dei passaggi individuali sopra.

### Esegui il backup del tuo albero genealogico

In Gramps Web, vai alla pagina "Esporta" e seleziona "Gramps XML (albero genealogico)" (l'opzione predefinita). Clicca su "Genera" e attendi che inizi il download del file .gramps.

Puoi anche creare un backup in Gramps Desktop [nel modo consueto](https://www.gramps-project.org/wiki/index.php/How_to_make_a_backup).

### Aggiorna lo schema del database dell'albero

Quando accedi a Gramps Web per la prima volta dopo averlo aggiornato, ti verrà presentata una schermata che mostra il messaggio:

> L'albero genealogico che stai cercando di caricare è in una versione di schema non supportata da questa versione di Gramps Web. Pertanto, non puoi caricare questo albero genealogico senza aggiornare il suo schema. Questa azione non può essere annullata.

Ora puoi cliccare sul pulsante "Aggiorna Albero Genealogico". Verrà mostrata una barra di avanzamento e non dovrebbe richiedere più di un minuto per completarsi. Non preoccuparti, se chiudi accidentalmente la finestra del browser, la conversione continuerà in background.

!!! warning
    Se utilizzi una configurazione non standard *senza* coda di attività in background, l'aggiornamento del database potrebbe non riuscire e il tuo albero genealogico potrebbe corrompersi. Puoi controllare (prima di aggiornare Gramps Web) se stai utilizzando una coda di attività in background andando nella sezione "Informazioni di sistema" nelle impostazioni e assicurandoti che abbia "coda di attività: true".

### Aggiorna Gramps Desktop

Segui le istruzioni di installazione di Gramps Desktop per il tuo sistema operativo.

### Installa l'addon Sync aggiornato

Puoi utilizzare [il nuovo Plugin Manager](https://gramps-project.org/wiki/index.php/Addon:Plugin_Manager) incluso in Gramps 5.2 - cerca semplicemente Gramps Web Sync, clicca su installa e riavvia Gramps.
