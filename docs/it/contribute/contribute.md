# Come contribuire a Gramps Web

Grazie per aver considerato di contribuire a Gramps Web! A seconda delle tue abilità e interessi, ci sono molti modi in cui puoi aiutare.

## Aiuta a tradurre Gramps Web nella tua lingua

Gramps e Gramps Web supportano diverse dozzine di lingue diverse. Aiutare a tradurre l'app non richiede alcuna competenza di programmazione. Utilizziamo il potente strumento online Weblate per le traduzioni. Si prega di vedere [qui](../development/frontend/translation.md) per istruzioni su come iniziare.


## Aiuta a rispondere alle domande di altri utenti

Se stai già utilizzando Gramps Web, aiuta i nuovi utenti a trovare le risposte alle loro domande contribuendo alle discussioni nella [categoria Gramps Web](https://gramps.discourse.group/c/gramps-web/) del forum ufficiale di Gramps.

## Aiuta a migliorare la documentazione

Il sito di documentazione che stai visualizzando è ospitato [su Github](https://github.com/gramps-project/gramps-web-docs). I contributi per migliorarlo sono molto benvenuti. Basta inviare una pull request a quel repository.

## Contribuisci allo sviluppo del backend (Python)

Se sei un programmatore Python, contribuisci allo sviluppo della [Gramps Web API](https://github.com/gramps-project/gramps-web-api/), che è il backend che alimenta Gramps Web. Vedi [Sviluppo del backend](../development/backend/index.md) per istruzioni su come iniziare.


## Contribuisci allo sviluppo del frontend (Javascript)

Se sei un programmatore web, contribuisci allo sviluppo del [frontend reattivo di Gramps Web](https://github.com/gramps-project/gramps-web/). Vedi [Sviluppo del frontend](../development/frontend/index.md) per istruzioni su come iniziare.

## Contribuisci all'addon Sync (Python)

Se hai esperienza nello sviluppo con Python e GTK, contribuisci al miglioramento dell'addon Sync di Gramps Web. Puoi utilizzare il [repository di sviluppo](https://github.com/DavidMStraub/gramps-web-sync/) per suggerire e inviare miglioramenti.
