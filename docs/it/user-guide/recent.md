# Oggetti Recentemente Navigati

**Cronologia** nella barra laterale apre la pagina Oggetti Recentemente Navigati, che elenca gli ultimi oggetti che hai aperto in Gramps Web, con i più recentemente visualizzati in cima. Fino a 20 oggetti vengono ricordati.

Questo è utile per navigare rapidamente di nuovo verso una persona, una famiglia o un altro oggetto che stavi guardando in precedenza nella tua sessione, senza doverlo cercare di nuovo.

Clicca su qualsiasi voce per tornare a quell'oggetto. Usa il pulsante **Cancella Tutto** per ripristinare l'elenco.

La cronologia è memorizzata nella memoria locale del browser – è specifica per il dispositivo e il browser attuali, e non è condivisa con altri utenti.
