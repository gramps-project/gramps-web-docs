# Ricerca Esterna

Gramps Web ti consente di cercare persone su siti web genealogici esterni direttamente dalla pagina dei dettagli di una persona.


## Eseguire una ricerca esterna

Per cercare una persona su siti web esterni:

1. Apri la vista dettagli di una persona.
2. Fai clic sul pulsante di ricerca esterna.
3. Apparirà un popup con collegamenti a vari siti web genealogici, come CompGen, FamilySearch e Ancestry.
4. Facendo clic su un collegamento si aprirà la pagina dei risultati di ricerca su quel sito, precompilata con il nome della persona, i luoghi di nascita/morte e altre informazioni rilevanti.

## Personalizzare le opzioni di ricerca esterna

Puoi personalizzare quali siti web di ricerca esterna vengono visualizzati nel popup.

1. Nel popup di ricerca esterna, fai clic sul pulsante a forma di ingranaggio in alto per entrare in modalità modifica.
2. Seleziona o deseleziona i siti web che desideri mostrare o nascondere.
3. Salva le tue modifiche.

## Aggiungere un motore di ricerca personalizzato

Puoi aggiungere nuovi siti web di ricerca esterna con parametri personalizzati.

1. Nel popup di ricerca esterna, fai clic sul pulsante a forma di ingranaggio in alto per entrare in modalità modifica.
2. Fai clic sul pulsante "+ Aggiungi motore di ricerca personalizzato".
3. Apparirà una sezione con campi per "Nome" e "URL di ricerca". Inserisci il nome del sito web e l'URL con i parametri ricercabili, come `https://example.com/search?name={{name_given}}_{{name_surname}}&place={{birth_place}}`.
4. Fai clic sul pulsante "Aggiungi" per includere il nuovo motore di ricerca nell'elenco.
5. Il nuovo motore di ricerca verrà salvato e potrà essere personalizzato in seguito.
