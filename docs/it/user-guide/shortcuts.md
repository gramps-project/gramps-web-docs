# Utilizzo delle Scorciatoie da Tastiera

Le scorciatoie da tastiera ti permettono di navigare rapidamente attraverso Gramps Web senza utilizzare il mouse, risparmiando clic e tempo.

Da qualsiasi vista, puoi saltare direttamente ad azioni e pagine comuni con solo pochi tasti. Ad esempio, premi **`s`** sulla tua tastiera per aprire la ricerca, **`g`** + **`c`** per navigare nell'Albero Genealogico, o **`n`** + **`p`** per creare una nuova persona.

Per vedere un elenco completo di tutte le scorciatoie da tastiera disponibili, basta premere **`?`** in qualsiasi momento.
