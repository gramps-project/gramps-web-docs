# Rapporti

Un rapporto è un documento formattato e scaricabile generato dai dati del tuo albero genealogico. I rapporti sono gli stessi plugin disponibili in Gramps Desktop – consulta la [pagina Wiki di Gramps sui Rapporti](https://gramps-project.org/wiki/index.php/Gramps_5.1_Wiki_Manual_-_Reports) per una panoramica completa. L'interfaccia web esegue lo stesso motore di rapporti sul server e consegna il file risultante al tuo browser.

I rapporti sono accessibili dalla barra laterale.

## Categorie di rapporti

La pagina Rapporti elenca tutti i rapporti disponibili raggruppati per categoria:

- **Rapporti Testuali** – documenti narrativi o tabulari come elenchi di antenati, rapporti sui discendenti e calendari delle nascite. I formati di output includono PDF, ODT/OpenDocument, RTF, HTML e altri.
- **Rapporti Grafici** – grafici e diagrammi come alberi genealogici e diagrammi a ventaglio. I formati di output includono PDF, SVG, PNG e altri.
- **Alberi** – diagrammi ad albero genealogico.
- **Libri** – libri di rapporti multi-capitolo che combinano diversi rapporti testuali e grafici in un unico documento.
- **Pagine Web** – generatori di siti web statici che producono un insieme completo di pagine HTML dal tuo albero.

I rapporti mostrati dipendono dai plugin di rapporti di Gramps installati sul server.

## Esecuzione di un rapporto

Fai clic su un rapporto per aprire la sua pagina delle opzioni, che mostra:

- **Descrizione** – cosa contiene il rapporto
- **Autore** e **Versione** del plugin
- **Opzioni** – impostazioni specifiche del rapporto come la persona iniziale, il numero di generazioni, il formato di output, le dimensioni della pagina e altri parametri

Regola le opzioni come necessario, quindi fai clic su **Genera**. Il rapporto viene generato sul server e il file risultante viene scaricato automaticamente nel tuo browser.

!!! tip
    La maggior parte dei rapporti predefiniti si basa sulla **persona principale del database** – la persona principale impostata nel database di Gramps, che potrebbe differire dalla tua impostazione personale della persona principale nel browser. Se il rapporto copre la persona sbagliata, modifica l'opzione della persona nelle impostazioni del rapporto.
