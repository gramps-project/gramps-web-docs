# Guida dell'Amministratore di Gramps Web

Che tu abbia configurato la tua istanza di Gramps Web o ti sia registrato per una già pre-installata, il passo successivo è popolare e gestire il tuo database dell'albero genealogico.

L'attenzione di questa sezione è sulle attività amministrative svolte dai **proprietari dell'albero**. Di solito, c'è un proprietario dell'albero per ogni database dell'albero genealogico (anche se come proprietario dell'albero, puoi anche creare altri account di proprietari dell'albero).

Questa sezione tratta solo le azioni che possono essere eseguite dall'interfaccia web. Per strumenti da riga di comando o configurazione del server, vedere la sezione precedente [Setup](../install_setup/setup.md).
