# Caratteristiche


![Image title](screenshots/sync.png){ align=left width="300"}

## Integrazione completa con Gramps Desktop

Gramps Web utilizza la stessa struttura **Modello / Database** che [Gramps Desktop](https://gramps-project.org/) utilizza per memorizzare i dati genealogici. Puoi navigare tra tutti gli stessi [Tipi di Record](https://gramps-project.org/wiki/index.php/Gramps_Data_Model) che trovi in Gramps Desktop: ***persone, famiglie, eventi, luoghi, repository, fonti, citazioni, oggetti multimediali e note.***

Utilizzando il [Gramps Web Sync Add-on](../administration/sync.md) per Gramps Desktop, i dati possono essere sincronizzati in entrambe le direzioni tra Gramps Web e Gramps Desktop! Procedi pure a modificare i tuoi dati con Gramps Web o l'app Gramps Desktop che conosci e ami - funzionano insieme senza problemi!

## Livelli di privacy / Accesso utente

Molte persone vogliono mantenere alcuni dettagli privati e noi rispettiamo questo! Puoi contrassegnare i Record come Privati e controllare quali utenti possono visualizzare i record privati. I record privati vengono filtrati a livello di database per la massima sicurezza. Inoltre, puoi controllare cosa gli utenti possono aggiungere e modificare.

## Nessun lock-in &ndash; importazione ed esportazione dei dati

Oltre a poter importare dati in vari formati, tra cui Gramps XML e GEDCOM, Gramps Web rende facile per gli utenti scaricare tutti i propri dati (dati dell'albero genealogico, file multimediali, account utente) in qualsiasi momento, per scopi di backup o per trasferirli su un altro server. I tuoi dati sono solo tuoi!

## App di gestione delle attività integrata

Gramps Web viene fornito con un'app di gestione delle attività integrata per organizzare e pianificare la tua ricerca genealogica. Le attività sono memorizzate come fonti nel database di Gramps, quindi fanno parte dei tuoi dati genealogici e possono essere accessibili e modificate anche in Gramps Desktop.

## Integrazione di OpenHistoricalMap

Le mappe vettoriali storiche create dal progetto [OpenHistoricalMap](https://www.openhistoricalmap.org/) sono il complemento perfetto per la mappatura genealogica. Usa il cursore temporale per scorrere l'evoluzione dei luoghi nella tua storia familiare e visualizza i luoghi in cui vivevano gli antenati o si sono verificati eventi.
