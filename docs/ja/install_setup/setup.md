# Gramps Web インストール / セットアップ

このセクションでは、Gramps Webのインストールとセットアップ、および開始するための他のオプションについて説明します。

## Gramps Web の始め方

Gramps Webはサーバー上で動作し、ウェブブラウザを介してアクセスされるウェブアプリです。インターネットを通じて認証されたユーザーにアクセス可能にすることを目的としています。

あなたが自分の系譜研究データのためにGramps Webを使用したい場合、次のオプションのいずれかを選択する必要があります：

1. 自分のハードウェアで自己ホスト
2. クラウドで自己ホスト
3. ホスティングインスタンスにサインアップ

最初のオプションは最大の柔軟性と制御を提供しますが、技術的に挑戦的であることもあります。

!!! tip
    Gramps Webの主な原則の1つは、ユーザーが常に自分のデータを制御できるようにすることですので、データを1つのインスタンスから別のインスタンスに移行するのは簡単です。オプションの1つを選んだ後にロックインされることを心配しないでください！

## 自分のハードウェアで自己ホスト

Gramps Webを自己ホストする最も便利な方法は、Docker Composeを使用することです。また、ARMアーキテクチャ用のDockerイメージも提供しているため、地下室のRaspberry PiでGramps Webを実行できます。

セットアップ手順については、[Dockerでのデプロイ](deployment.md)を参照してください。

## クラウドで自己ホスト

Gramps Webのインストールは、他の単純なウェブアプリケーションよりも難しい場合があり、通常の「共有ホスティング」プロバイダーとは互換性がありません。仮想サーバーにサインアップし、Gramps Webを[手動で](deployment.md)インストールできます。

より簡単なオプションは、プレインストールされたクラウドイメージを使用することです。1つの例は、私たちの[DigitalOcean 1クリックアプリ](digital_ocean.md)です。

## ホスティングインスタンスにサインアップ

ホスティングされたGramps Webは、インストールや設定が不要なため、Gramps Webを始める最も簡単な方法です。

以下は、Gramps Webのための専用ホスティングプロバイダーのリストです（Grampsオープンソースコミュニティは彼らのサービスに対して責任を負いません）：

- Grampshub ([www.grampshub.com](https://www.grampshub.com))、Gramps Webの主要な貢献者の1人が提供

Gramps Webのホスティングオプションを使用する場合は、このセクションの残りをスキップして、[管理](../administration/admin.md)セクションに直接進むことができます。
