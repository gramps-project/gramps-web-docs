# タイムライン

!!! info
    タイムラインビューには、十分に最近のGramps Web APIバージョンが必要です。サーバーが古い場合、**タイムライン**項目はサイドバーから隠されます。

**タイムライン**は、家系図内のすべてのイベントを水平方向のズーム可能なタイムラインに沿って時系列で表示するサイドバー項目です。

## タイムラインの閲覧

- スクロール、ドラッグ、またはピンチして時間を移動します。ズームボタン（またはマウスホイール/ピンチジェスチャー）を使用してズームインおよびズームアウトします。
- ズームインすると、イベントラベルが表示されます。イベントのドットをクリックすると、その詳細（および同じ日に発生した他のイベントの詳細）を見ることができます。

## フィルタリング

フィルターフィールドを使用して、タイムラインを単一の**人物**または**場所**に絞り込みます：

- **人物** – その人物にリンクされたイベントのみを表示します。トグルを使用して、**人物**（その個人のイベントのみ）、**先祖**、および**子孫**（その先祖/子孫のすべてのイベント）を切り替えることができます。
- **場所** – そこで発生したイベントのみを表示します。

フィルターチップをクリアして、完全なタイムラインに戻ります。

## タイムラインへのジャンプ

イベントまたは場所の詳細ページで、**タイムラインに表示**ボタンをクリックすると、そのイベントの日付にスクロールおよびズームされたタイムラインビューが開きます。
