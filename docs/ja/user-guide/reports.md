# レポート

レポートは、家系図データから生成されたフォーマットされたダウンロード可能な文書です。レポートはGramps Desktopで利用可能な同じプラグインであり、詳細については[Gramps Wikiのレポートページ](https://gramps-project.org/wiki/index.php/Gramps_5.1_Wiki_Manual_-_Reports)を参照してください。ウェブインターフェースはサーバー上で同じレポートエンジンを実行し、生成されたファイルをブラウザに配信します。

レポートはサイドバーからアクセスできます。

## レポートカテゴリ

レポートページには、カテゴリ別にグループ化されたすべての利用可能なレポートがリストされています：

- **テキストレポート** – 先祖リスト、子孫レポート、誕生日カレンダーなどの物語的または表形式の文書。出力形式にはPDF、ODT/OpenDocument、RTF、HTMLなどがあります。
- **グラフィカルレポート** – 先祖の木やファンチャートなどのチャートや図。出力形式にはPDF、SVG、PNGなどがあります。
- **ツリー** – 家系図の図。
- **書籍** – 複数のテキストおよびグラフィカルレポートを1つの文書にまとめたマルチチャプターレポートブック。
- **ウェブページ** – あなたの家系図から完全なHTMLページセットを生成する静的ウェブサイトジェネレーター。

表示されるレポートは、サーバーにインストールされているGrampsレポートプラグインによって異なります。

## レポートの実行

レポートをクリックすると、そのオプションページが開き、以下が表示されます：

- **説明** – レポートに含まれる内容
- プラグインの**著者**と**バージョン**
- **オプション** – 開始人物、世代数、出力形式、ページサイズ、その他のパラメータなど、レポート固有の設定

必要に応じてオプションを調整し、次に**生成**をクリックします。レポートはサーバー上で生成され、生成されたファイルが自動的にブラウザにダウンロードされます。

!!! tip
    ほとんどのレポートは**データベースのホーム人物**をデフォルトとして使用します – Grampsデータベースに設定されたホーム人物であり、ブラウザの個人のホーム人物設定とは異なる場合があります。レポートが間違った人物をカバーしている場合は、レポートの設定で人物オプションを変更してください。
