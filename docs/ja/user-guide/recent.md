# 最近閲覧したオブジェクト

**履歴** サイドバーで最近閲覧したオブジェクトページを開きます。このページでは、Gramps Webで最後に開いたオブジェクトがリストされており、最も最近閲覧したものが上に表示されます。最大20個のオブジェクトが記憶されます。

これは、セッション中に以前見ていた人物、家族、または他のオブジェクトに素早く戻るのに便利で、再度検索する必要がありません。

任意のエントリをクリックすると、そのオブジェクトに戻ります。**すべてクリア**ボタンを使用してリストをリセットします。

履歴はブラウザのローカルストレージに保存されます – これは現在のデバイスとブラウザに特有であり、他のユーザーと共有されることはありません。
