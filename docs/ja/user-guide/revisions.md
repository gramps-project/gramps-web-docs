# 改訂履歴

改訂履歴ビューは、ファミリーツリーに対して行われたすべての編集を表示します。

リストビューは、「トランザクション」によってグループ化された編集を表示します。トランザクションは、Grampsオブジェクトに対する1つ以上の追加、削除、または変更のグループです。たとえば、父親と母親として2人の既存の人物を持つ新しい家族を追加すると、1つの追加された家族オブジェクトと2つの変更された人物オブジェクトを含むトランザクションが生成されます（なぜなら、それらは新しい家族オブジェクトへのリンクを含んでいるからです）。

トランザクションをクリックすると、トランザクション詳細ビューが開きます。そこには、Grampsオブジェクトによる個々の追加、削除、および更新のリストが含まれています。

個々の変更を選択すると、追加と削除がそれぞれ緑と赤で強調表示されたGrampsオブジェクトの生のJSON表現のビューが開きます。
