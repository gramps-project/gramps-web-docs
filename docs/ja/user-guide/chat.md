# AIチャットの使用

!!! info
    AIチャットにはGramps Web APIバージョン2.5.0以上およびGramps Webバージョン24.10.0以上が必要です。



Gramps Webのチャットビュー（インストールに利用可能な場合）は、家系図に関する質問に答えるAIアシスタントにアクセスできます。

!!! warning
    これはまだ新しい進化中の機能であるため、質問の種類によってはうまく機能するものとそうでないものがあります。また、他のAIアシスタントと同様に、事実に基づかない回答をすることがあるため、必ず二重チェックしてください。

## 仕組み

アシスタントがどのような質問に答えられるかを理解するためには、以下のようにその仕組みを理解することが役立ちます：

1. ユーザーが質問をします。
2. Gramps Webは、質問に答える情報を含む可能性が最も高いGrampsオブジェクトをいくつか（例えば、10個）特定します。この目的のために、「セマンティックサーチ」と呼ばれる技術を使用します。たとえば、「ジョン・ドーの子供の名前は何ですか？」と尋ねると、ジョン・ドーを父とする家族が存在する場合、それは上位の結果の中に含まれる可能性が高いです。
3. Gramps Webは、ユーザーの質問と取得したコンテキスト情報を大規模言語モデル（「チャットボット」）に渡し、正しい回答を抽出するように依頼します。
4. 回答がユーザーに表示されます。

## 質問の仕方

チャットの仕組み上、AIアシスタントは親や子供以外の特定の関係に関する質問には答えられない（現在のところ）、この情報がノートのテキストとして含まれている場合を除きます。

各回答は限られた数の上位セマンティックサーチ結果に基づいているため、統計に関する質問（「私のデータベースには何人いますか...」）にも答えることができません。

あいまいさや誤解を避けるために、できるだけ詳細に質問を構成することが役立ちます。

大規模言語モデルは多言語対応であるため、自分の言語で話しかけることができ、同じ言語で回答します。

!!! tip
    うまくいったことやうまくいかなかったことについて、コミュニティと経験を共有してください。
