# AIアシスタントの使用

!!! info
    AIアシスタントは、Gramps Web APIバージョン2.5.0以上およびGramps Webバージョン24.10.0以上を必要とします。Gramps Web APIバージョン3.6.0では、よりインテリジェントな対話のためのツール呼び出し機能が導入されました。



Gramps Webの**アシスタント**ビュー（インストールに利用可能な場合、古いバージョンでは「チャット」と表示されます）では、家系図に関する質問に答えるAIアシスタントにアクセスできます。

!!! warning
    これはまだ新しい進化中の機能であるため、質問の種類によってはうまく機能するものもあれば、そうでないものもあります。また、すべてのAIアシスタントと同様に、事実に基づかない回答をすることがあるため、必ず二重チェックを行ってください。

## 仕組み

アシスタントがどのような質問に答えられるかを理解するためには、以下の仕組みを理解することが役立ちます：

1. ユーザーが質問をします。
2. AIアシスタントは、回答を見つけるために複数のアプローチを使用できます：
   - **セマンティック検索**: Gramps Webは、家系図の中で関連情報を含む可能性が最も高いオブジェクトを特定します。たとえば、「ジョン・ドーの子供の名前は何ですか？」と尋ねると、ジョン・ドーを父とする家族が上位の結果に表示されます。
   - **ツール呼び出し（Gramps Web API v3.6.0以上）**: アシスタントは、特定の基準で人々/イベント/家族/場所を検索・フィルタリングし、個人間の関係を計算し、詳細情報を取得するために専門のツールを使用してデータベースに直接クエリを実行できます。
3. Gramps Webは、質問と取得した情報を大規模な言語モデルに渡して回答を生成します。
4. 回答が表示されます。

アシスタントが作業している間、現在使用しているツール（例：人を検索中、関係を調べているなど）を示すインジケーターが表示されるため、回答を構築する様子を追うことができます。長時間かかる質問はバックグラウンドタスクとして処理されるため、他のページに移動して戻ってくることができ、進捗は[通知](notifications.md)にも反映されます。回答はMarkdown（リスト、強調、リンクなど）でフォーマットされており、読みやすくなっています。

## 何を尋ねることができるか

Gramps Web APIバージョン3.6.0で導入されたツール呼び出し機能により、AIアシスタントはより複雑な質問にも対応できるようになりました：

- **家族関係**: 「ジェーン・スミスの祖父母は誰ですか？」や「ジョン・ドーはメアリー・ジョンソンとどのように関係していますか？」
- **フィルタリングされた検索**: 「1850年以降にロンドンで生まれたすべての人を見せて」や「パリで起こったイベントは何ですか？」
- **日付に基づくクエリ**: 「1900年以前に亡くなった人は誰ですか？」や「1920年から1950年の間に行われた結婚をリストアップしてください」
- **場所情報**: 「フランスにはどんな場所がありますか？」や「セント・メアリー教会について教えてください」
- **一般的な質問**: 「ジョン・ドーの子供の名前は何ですか？」や「メアリー・スミスはいつ生まれましたか？」

## 質問の仕方のヒント

AIアシスタントから最良の結果を得るためには：

- **具体的に**: 曖昧さを避けるために、できるだけ詳細に質問を構成してください。たとえば、「1850年にボストンで生まれたジョン・スミスはいつ結婚しましたか？」は「ジョン・スミスはいつ結婚しましたか？」よりも良いです。
- **固有名詞を使用**: 関連する場合は、特定の名前、場所、日付を言及してください。
- **一度に一つのことを尋ねる**: 複雑な質問をよりシンプルな部分に分けて、より良い結果を得てください。
- **自分の言語を使用**: 大規模な言語モデルは多言語に対応しているため、自分の言語で質問をし、同じ言語で回答を受け取ることができます。

!!! tip
    うまくいったことやうまくいかなかったことについて、コミュニティと経験を共有してください。
