# ユーザー設定

ユーザー設定は、アプリの上部バーにあるユーザーアイコンからアクセスでき、その後**ユーザー設定**を選択します。ページは折りたたみ可能なセクションに整理されています。変更は特に記載がない限り、即座に反映されます。

!!! note
    ユーザー設定の変更は、自分のアカウントにのみ影響します。すべてのユーザーに影響を与える設定は、[管理設定](../administration/settings.md)で管理されます。

## アカウント

プロフィール情報、認証情報、およびアカウントのセキュリティをカバーします。

### ユーザー情報

あなたの**ユーザー名**と現在の**ユーザー役割**（例：ゲスト、メンバー、編集者）を表示します。これらは読み取り専用です。

### メールアドレスの変更

新しいメールアドレスを入力し、**送信**をクリックして、アカウントに関連付けられたアドレスを更新します。メールアドレスはパスワードリセットや（設定されている場合）通知に使用されます。

### パスワードの変更

現在のパスワードと新しいパスワードを入力し、次に**送信**をクリックします。現在のパスワードを忘れた場合は、ログインページの**パスワードを忘れた**リンクを使用してください。

## 外観

デバイスに保存された表示設定を制御します。

### 言語

Gramps Webインターフェースの言語を選択します。言語設定はブラウザのローカルストレージに保存され、現在のデバイスにのみ適用されます。

### テーマ

次のいずれかを選択します：

- **システム** – オペレーティングシステムの明るい/暗い設定に従います（デフォルト）
- **ライト** – 常にライトテーマを使用
- **ダーク** – 常にダークテーマを使用

テーマ設定はブラウザのローカルストレージに保存されます。

### 家系図の設定

#### デフォルトの家系図ビュー

[家系図](tree.md)ページに移動したときにデフォルトで開くチャートタイプを設定します。オプションは、祖先ツリー、子孫ツリー、砂時計グラフ、関係グラフ、ファンチャートです。

この設定はブラウザのローカルストレージに保存されます。

## 開発者ツール

### APIトークン

現在のセッショントークンをクリップボードにコピーします。このトークンは、たとえば、あなたのGramps Webインスタンスの`/api/swagger-ui`で提供されるインタラクティブなSwagger UIに対して直接認証するために使用できます。

**Swaggerを起動**をクリックして、新しいタブでSwagger UIを開き、すでにセッションが利用可能な状態にします。

!!! note
    セッショントークンは短命です。Swaggerで使用する前にすぐにコピーしてください。期限切れになる可能性があります。
