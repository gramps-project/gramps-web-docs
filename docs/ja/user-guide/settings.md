# ユーザー設定

ユーザー設定は、アプリの上部バーにあるユーザーアイコンをクリックし、次に**ユーザー設定**からアクセスできます。変更は、特に記載がない限り、即座に反映されます。

## ユーザー情報

あなたの**ユーザー名**と現在の**ユーザー役割**（例：ゲスト、メンバー、編集者）を表示します。これらは読み取り専用です。

## 言語

Gramps Webインターフェースの言語を選択します。言語設定はブラウザのローカルストレージに保存され、現在のデバイスのみに適用されます。

## テーマ

次の中から選択します：

- **システム** – オペレーティングシステムのライト/ダークの設定に従います（デフォルト）
- **ライト** – 常にライトテーマを使用
- **ダーク** – 常にダークテーマを使用

テーマ設定はブラウザのローカルストレージに保存されます。

## メールアドレスの変更

新しいメールアドレスを入力し、**送信**をクリックしてアカウントに関連付けられたアドレスを更新します。メールアドレスは、パスワードのリセットや（設定されている場合）通知に使用されます。

## パスワードの変更

現在のパスワードと新しいパスワードを入力し、次に**送信**をクリックします。現在のパスワードを忘れた場合は、ログインページの**パスワードを忘れた**リンクを使用してください。

## 家系図の設定

### デフォルトの家系図ビュー

[家系図](tree.md)ページに移動したときにデフォルトで開くチャートタイプを設定します。オプションは、祖先ツリー、子孫ツリー、砂時計グラフ、関係グラフ、ファンチャートです。

この設定はブラウザのローカルストレージに保存されます。

## 開発者ツール

### APIトークン

現在のセッショントークンをクリップボードにコピーします。このトークンは、例えば、あなたのGramps Webインスタンスで提供されるインタラクティブなSwagger UIに対してREST APIに直接認証するために使用できます。パスは`/api/swagger-ui`です。

**Swaggerを起動**をクリックすると、新しいタブでSwagger UIが開き、あなたのセッションがすでに利用可能になります。

!!! note
    セッショントークンは短命です。Swaggerで使用する前にすぐにコピーしてください。期限切れになる可能性があります。
