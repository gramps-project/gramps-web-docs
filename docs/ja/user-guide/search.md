# 検索

検索ページは、アプリバーの虫眼鏡アイコンをクリックするか、`s` [キーボードショートカット](shortcuts.md)を押すことでアクセスできます。

## フルテキスト検索

検索フィールドに任意のクエリを入力し、Enterを押す（または検索ボタンをクリック）します。Gramps Webは、すべてのオブジェクトタイプ（人物、家族、イベント、場所、ソース、引用、リポジトリ、ノート、メディア）を検索し、関連性に基づいて結果を返します。

結果にはオブジェクトタイプ、名前、および簡単な概要が表示されます。任意の結果をクリックすると、対応する詳細ページが開きます。

末尾に`*`を使用するとワイルドカードとして機能します。たとえば、`Mey*`は「Meyer」、「Meyers」、「Meyerhofer」などに一致します。

## オブジェクトタイプによるフィルタリング

検索フィールドの下には、各オブジェクトタイプ（人物、家族、イベント、場所など）のトグルボタンがあり、結果を特定のタイプに絞り込むことができます。デフォルトではすべてのタイプが検索されます。1つ以上のトグルを有効にすると、結果はそのタイプのみに制限されます。

## セマンティック検索

サーバー管理者が[セマンティック（AI駆動）検索](../install_setup/configuration.md)を有効にしている場合、検索ページの右上隅に2つのオプションを持つモードトグルが表示されます：

- **検索** – 標準のフルテキスト検索（デフォルト）
- **セマンティック** – 正確な単語に一致するのではなく、クエリの意味を理解するAI駆動の検索

セマンティック検索は、「19世紀のバイエルンの農夫」のような自然言語のクエリに役立ちます。セマンティック検索インデックスが構築されている必要があります。インデックスの構築または更新方法については、[管理設定](../administration/settings.md)を参照してください。
