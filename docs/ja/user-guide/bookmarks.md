# ブックマーク

ブックマークを使用すると、頻繁に訪れるオブジェクト（人物、家族、場所、ソースなど）へのリンクを保存し、迅速にアクセスできます。

## ブックマークの追加と削除

すべてのオブジェクト詳細ページには、上部ツールバーにブックマークアイコン（ブックマークのアウトライン）が表示されます。それをクリックするとオブジェクトがブックマークされ、アイコンが塗りつぶされたブックマークになります。再度クリックするとブックマークが削除されます。

## ブックマークの表示

**ブックマーク**ページはサイドバーからアクセスでき、すべてのブックマークされたオブジェクトが一覧表示されます。任意のエントリをクリックすると、そのオブジェクトの詳細ページに移動します。

## 使用例

一般的な使用例は、現在調査中の人物や家族をブックマークして、検索せずに迅速に戻れるようにすることです。また、頻繁に参照する重要なソースや場所をブックマークすることもできます。

## ストレージ

ブックマークはブラウザのローカルストレージに保存され、現在のツリーにスコープされています。使用しているデバイスとブラウザに特有であり、デバイス間で同期されず、他のユーザーと共有されることはありません。

!!! note
    Gramps Webのブックマークは、Gramps Desktopのブックマークとは独立しています。Grampsデータベースには保存されず、Gramps Desktopには表示されず、その逆も同様です。
