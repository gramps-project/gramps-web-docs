# 外部検索

Gramps Webを使用すると、人物の詳細ページから直接外部系譜ウェブサイトで人を検索できます。

## 外部検索の実行

外部ウェブサイトで人物を検索するには：

1. 人物の詳細ビューを開きます。
2. 外部検索ボタンをクリックします。
3. CompGen、FamilySearch、Ancestryなど、さまざまな系譜ウェブサイトへのリンクが表示されるポップアップが表示されます。
4. リンクをクリックすると、そのウェブサイトの検索結果ページが開き、人物の名前、生年月日/死亡地、その他の関連情報が事前に入力されます。

## 外部検索オプションのカスタマイズ

ポップアップに表示される外部検索ウェブサイトをカスタマイズできます。

1. 外部検索ポップアップで、上部のギアボタンをクリックして編集モードに入ります。
2. 表示または非表示にしたいウェブサイトを選択または選択解除します。
3. 変更を保存します。

## カスタム検索エンジンの追加

カスタムパラメータを使用して新しい外部検索ウェブサイトを追加できます。

1. 外部検索ポップアップで、上部のギアボタンをクリックして編集モードに入ります。
2. 「+ カスタム検索エンジンを追加」ボタンをクリックします。
3. 「名前」と「検索URL」のフィールドが表示されるセクションが現れます。ウェブサイト名と、検索可能なパラメータを含むURLを入力します。例：`https://example.com/search?name={{name_given}}_{{name_surname}}&place={{birth_place}}`。
4. 「追加」ボタンをクリックして、新しい検索エンジンをリストに含めます。
5. 新しい検索エンジンは保存され、後でカスタマイズできます。
