# タグ

タグは、Grampsデータベース内の任意のオブジェクト（人物、家族、イベント、場所、ソース、引用、リポジトリ、ノート、メディア）に適用できるラベルです。タグはオブジェクトをグループ化したりフィルタリングしたりするのに便利です。タグはGrampsファミリーツリーデータベースに保存され、すべてのユーザーで共有されます。また、Gramps Desktopで作成されたタグとも完全に互換性があります。

## タグの管理

タグは、**管理設定**の[タグ](../administration/settings.md#tags)セクションから管理されます。このセクションは、オーナーまたは管理者の役割を持つユーザーのみが利用できます。ここでは、すべての既存のタグが表示され、以下の操作が可能です：

- **新しいタグを作成**するには、**新しいタグ**ボタンを使用します
- **タグの名前を変更**するには、編集（鉛筆）アイコンを使用します
- **タグの色を変更**するには、カラーピッカーを使用します
- **タグを削除**するには、削除アイコンを使用します

!!! note
    タグを削除すると、そのタグが適用されていたすべてのオブジェクトから削除されます。

## オブジェクトへのタグの適用

タグは、編集モードの詳細ページでオブジェクトに適用したり、削除したりできます。

## タグによるフィルタリング

すべてのオブジェクトリストページ（人物、家族、イベント、場所、ソース、引用、リポジトリ、ノート、メディア）にはタグフィルターが含まれています。特定のタグが適用されたオブジェクトのみを表示するために使用します。

## 特殊なタグ

Gramps Webでは、2つのタグに特別な意味があります：

- **`Blog`** – `Blog`タグが付けられたソースはブログ投稿として扱われ、[ブログ](blog.md)ビューに表示されます
- **`ToDo`** – `ToDo`タグが付けられたノートは研究タスクとして扱われ、[タスク](tasks.md)ビューに表示されます

これらのタグは、ブログまたはタスク機能を初めて使用する際に自動的に作成されます。名前を変更したり削除したりすると、対応する機能が破損します。
