# 通知

**通知**はベルアイコンを持つサイドバー項目です。エラーが発生した場合、バッジが未読通知の数を表示します。クリックすると通知ログが開きます。

通知ログは、セッション中に発生したエラーの記録です – 失敗したAPIリクエスト、バックグラウンドタスクエラー、保存失敗、またはブラウザレベルのエラー。各エントリには短いメッセージ、ソース（ネットワーク、タスク、保存、またはブラウザ）、およびタイムスタンプが表示されます。

一部の通知には構造化された詳細が含まれています。そのようなエントリをクリックすると、エラーデータの内訳と**JSONをコピー**ボタンを含むダイアログが開きます。これはバグを報告する際に便利で、JSONにはサーバーからの正確なエラー情報が含まれています。

**すべてクリア**を使用して、すべての通知を消去します。

!!! note
    通知はメモリにのみ保存され、ページをリロードするとクリアされます。
