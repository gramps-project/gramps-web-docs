# クロモソームブラウザ

クロモソームブラウザは、個人のDNAマッチを視覚化するためのツールです。これは、染色体上の他の人と共有されているDNAセグメントを色付きのバーとして表示します。

DNAビューの「クロモソームブラウザ」タブの下にあります。

各染色体について、上部（薄青）のバーは父親から受け継いだコピーを表し、下部（薄赤）のバーは母親から受け継いだものを表します。共有されているセグメントは、これらの上に色付きのバーとして表示されます。どちらの側（父方または母方）が不明な場合、セグメントは両方のバーにまたがり、ハッチマークで表示されます。

凡例のマッチ名をクリックすると、その人の共有セグメントのみが表示されます。
