# Gramps Webへの貢献方法

Gramps Webへの貢献を検討していただきありがとうございます！あなたのスキルや興味に応じて、助ける方法はたくさんあります。

## Gramps Webをあなたの言語に翻訳する手助け

GrampsとGramps Webは数十の異なる言語をサポートしています。アプリの翻訳を手伝うためにプログラミングスキルは必要ありません。私たちは強力なWeblateオンラインツールを翻訳に使用しています。始めるための手順については[こちら](../development/frontend/translation.md)をご覧ください。


## 他のユーザーの質問に答える手助け

すでにGramps Webを使用している場合は、新しいユーザーが質問の答えを見つける手助けをするために、公式Gramps Discourseフォーラムの[Gramps Webカテゴリ](https://gramps.discourse.group/c/gramps-web/)での議論に貢献してください。

## ドキュメントの改善を手伝う

あなたが見ているドキュメントサイトは[Github](https://github.com/gramps-project/gramps-web-docs)にホストされています。これを改善するための貢献は大歓迎です。そのリポジトリにプルリクエストを提出してください。

## バックエンド開発に貢献する（Python）

Pythonプログラマーであれば、Gramps Webを支えるバックエンドである[Gramps Web API](https://github.com/gramps-project/gramps-web-api/)の開発に貢献してください。始めるための手順については[バックエンド開発](../development/backend/index.md)をご覧ください。


## フロントエンド開発に貢献する（Javascript）

ウェブプログラマーであれば、[Gramps Webのレスポンシブフロントエンド](https://github.com/gramps-project/gramps-web/)の開発に貢献してください。始めるための手順については[フロントエンド開発](../development/frontend/index.md)をご覧ください。

## Syncアドオンに貢献する（Python）

PythonとGTKでの開発経験がある場合は、Gramps Web Syncアドオンの改善に貢献してください。改善の提案や提出には[開発リポジトリ](https://github.com/DavidMStraub/gramps-web-sync/)を使用できます。
